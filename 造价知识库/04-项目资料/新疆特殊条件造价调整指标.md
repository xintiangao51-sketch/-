### 1.3 冬施各分项增加费率

| 费用项目 | 占冬施完成工程量直接费比例 | 说明 |
|----------|------------------------|------|
| 保温材料费 | 1.0 - 3.0% | 保温被、彩条布、棚布等 |
| 防冻剂/早强剂 | 0.5 - 1.5% | 混凝土外加剂 |
| 冬施人工降效 | 1.0 - 2.5% | 工效降低 |
| 冬施机械降效 | 0.5 - 1.5% | 燃料消耗增加、效率降低 |

| 冬施管理费 | 0.5 - 1.0% | 增加管理人员、测温等 |
| **合计** | **3 - 10%** | 视地区和冬施天数 |

> **实操案例：** 乌鲁木齐某住宅项目，冬施期间（11月-3月，已考虑冬歇暂停）实际冬施期按2个月（60天）计，冬施期间完成产值约500万元（混凝土结构），冬施增加费按6%计算：500万×6%=30万元。其中保温材料10万元，外加剂5万元，人工降效8万元，机械降效3万元，取暖4万元。

### 1.4 冬施取费实操注意

1. **费率按实际：** 定额给出的冬施费率是综合费率，实际费用往往高于定额费率。建议大型项目按实际措施费用签证，中小项目按定额费率包干。
2. **冬歇期确认：** 冬施不等于冬歇，冬季施工增加费适用于实际施工的冬季期间，冬歇期间不产生冬施费。
3. **冬施界面划分：** 建议在合同中明确冬施与冬歇的起止时间确认方式（以气象数据为准或双方确认）。
4. **定额调整期：** 新疆2024版定额修订了冬施费率，新费率较2014版有所上调，使用时应确认版本。

### 2.4 高原降效综合应用示例

**示例：塔什库尔干县某房屋建筑项目**
```
施工地点海拔：3,350m
合同清单人工费：680万元（按乌鲁木齐标准）
人工降效系数取：1.18（海拔3350m，查表）
人工降效增加费：680 × (1.18 - 1.00) = 122.4万元

合同清单机械费：320万元
机械降效系数取：1.12（海拔3350m）
机械降效增加费：320 × (1.12 - 1.00) = 38.4万元

高原医疗保障费：现场施工8个月 × 月均50工日/天 × 240天 × 5元/天 = 6万元
生活物资增加费：按产值0.5%计 = 约5万元

高原降效增加合计：122.4 + 38.4 + 6 + 5 = 171.8万元
占原合同造价比例：171.8 / (680+320) ≈ 17.2%
```

### 5.3 超运距增加费用计算公式

**方式1：按每公里运费加价（常见于合同约定）**
```
超运距增加费 = 超运距里程（km） × 约定运价（元/t·km）
```

**方式2：按实际运输发票（凭票报销）**
```
超运距增加费 = 实际运输发票金额 - 合同约定运距的等价运费
```

**方式3：按材料单价比例增加（估算阶段）**
```
偏远地区材料价 = 基准地区材料价 × (1 + 运距系数)
运距系数：
  500km以内：1.05 - 1.10
  500-1,000km：1.10 - 1.20
  1,000-1,500km：1.20 - 1.35
  1,500-2,000km：1.35 - 1.55
  >2,000km：1.55 - 2.00
```

> **实操提示：** 材料运距是影响新疆偏远地区工程造价的第一大因素。以和田为例，一吨钢筋从乌鲁木齐运至和田，运费约600-900元（按1,800km×0.35-0.5元/t·km），相当于钢筋本身价格的15-20%。因此偏远项目建议考虑地材自给或就近采购方案。

## 六、综合应用案例

### 案例1：阿勒泰某住宅项目冬季施工造价调整

**项目概况：** 阿勒泰市某6层住宅，建筑面积6,000m²，2025年11月至2026年4月冬季施工（含冬歇）

**冬施调整计算：**
```
正常建安造价（夏季，按乌鲁木齐标准）：6,000 × 2,200 = 1,320万元

调整因素：
1. 冬施增加费：按冬施期间完成量800万元的6% = 48万元
2. 地区人工调整（阿勒泰vs乌鲁木齐）：(正常造价中人工费占25%)
   人工调整系数1.12：1,320 × 25% × (1.12 - 1.00) = 39.6万元
3. 材料运距调整（材料费占55%）：
   材料调整系数1.10：1,320 × 55% × (1.10 - 1.00) = 72.6万元
4. 冻土施工增加：
   基础施工增加费用（换填/防冻）：200万元 × 15% = 30万元

调整后造价：1,320 + 48 + 39.6 + 72.6 + 30 = 1,510.2万元
调整后单方造价：1,510.2万/6,000 = 2,517元/m²
调整幅度：14.4%
```

### 案例2：塔什库尔干县某项目高原+运距综合调整

**项目概况：** 塔什库尔干县某三层办公楼，建筑面积2,000m²，海拔3,300m

**综合调整计算：**
```
正常建安造价（按乌鲁木齐标准，框架结构）：2,000 × 3,200 = 640万元

调整因素：
1. 高原人工降效（海拔3,300m，系数1.18）：人工费占27%
   640 × 27% × (1.18 - 1.00) = 31.1万元
2. 高原机械降效（海拔3,300m，系数1.12）：机械费占15%
   640 × 15% × (1.12 - 1.00) = 11.5万元
3. 超长运距（距乌鲁木齐1,800km）：
   材料运距系数1.45：材料费占48%
   640 × 48% × (1.45 - 1.00) = 138.2万元
4. 冬施增加（冬施天数190天，费率8%）：
   冬施期间完成量约480万元 × 8% = 38.4万元
5. 地震设防增加（8度设防，结构占55%，增加15%）：
   640 × 55% × 15% = 52.8万元
6. 风沙防护（南疆II级，增加1.5%）：
   640 × 1.5% = 9.6万元
7. 高原医疗保障：
   640 × 1.0%（参考费率） = 6.4万元

调整后造价：640 + 31.1 + 11.5 + 138.2 + 38.4 + 52.8 + 9.6 + 6.4 = 928万元
调整后单方造价：928万/2,000 = 4,640元/m²
调整幅度：45%
```

### 案例3：喀什某市政道路工程综合调整

**项目概况：** 喀什市某城市次干路，长度3km，宽度30m，沥青混凝土路面

**综合调整：**
```
正常建安造价（按乌鲁木齐标准，60cm厚结构层）：
30,000m² × 320元/m² = 960万元（含路基）

调整因素：
1. 人工费地区调整（喀什人工费低于乌鲁木齐）：
   人工费占12%，调整系数0.92（喀什比乌市低8%）
   960 × 12% × (1 - 0.92) = -9.2万元（减项）
2. 材料运距（沥青从克拉玛依运距约1,300km）：
   沥青占道路造价30%，运距系数1.25
   960 × 30% × (1.25 - 1.00) = 72.0万元
3. 风沙防护（南疆III级，增加0.8%）：
   960 × 0.8% = 7.7万元
4. 路基冻土处理（喀什冻土深度0.8-1.0m）：
   增加防冻层换填：3km × 30m × 0.3m（换填厚度）× 35元/m³ = 9.5万元
5. 地震设防增加（喀什8度设防）：
   道路上结构物（桥涵）占20%，增加10%
   960 × 20% × 10% = 19.2万元

调整后造价：960 - 9.2 + 72.0 + 7.7 + 9.5 + 19.2 = 1,059.2万元
调整后单方造价：1,059.2万/30,000 = 353元/m²
调整幅度：10.3%
```

### 各类调整因素对造价影响汇总

| 调整因素 | 低影响（%） | 中影响（%） | 高影响（%） | 影响最大的工程类型 |

| 高原机械降效 | 3 - 5 | 10 - 18 | 18 - 30 | 机械密集型工程 |

### 综合调整应用原则

1. **累加原则：** 多项调整同时适用时，一般情况下各项调整累加计算
2. **不重复原则：** 同一影响因素不得重复计算（如高原降效已包含部分冬季施工影响，但实际中应避免交叉计算）
3. **基准统一：** 所有调整以乌鲁木齐地区标准为基准，各地州分别计算
4. **侧重主因：** 影响最大的1-2个因素决定了调整幅度，其他因素影响相对较小
