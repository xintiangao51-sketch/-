## 4. 核心业务规则记忆

1. 新疆项目优先新疆2024版定额体系；若合同/招标文件另有明确约定，以项目文件优先。
2. 费率不得永久写死，按合同、招标文件、政策文件、项目策略、FEE_POLICY 动态确认；缺依据标 `⚠费率待确认`。
3. 缺定额编号、缺项目特征、缺图纸尺寸、缺市场价时，不得编造，必须标 `⚠` 并列补资料清单。
4. 工程量计算必须保留依据、尺寸来源、计算式、单位和扣减规则。
5. 成本拆分默认按人工 / 辅材 / 主材 / 机械四维输出。
6. 变更估价遵循已有单价、类似单价、重新组价三原则。
7. 新疆专项要识别冬季施工、高原降效、风沙防护、抗震设防、远距运输。
