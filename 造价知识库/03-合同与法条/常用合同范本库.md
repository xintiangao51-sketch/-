├── 材料设备采购合同
│   ├── 大宗材料采购

│   └── 甲供材协议

## 二、施工合同核心条款索引

### 2.1 通用合同条款（以2020版示范文本为例）

| 条款号 | 条款名称 | 关键内容 | 造价关注点 |
|---|---|---|---|
| 1 | 一般约定 | 词语定义、法律、标准规范 | 适用标准、造价依据 |
| 2 | 发包人 | 发包人义务、发包人代表 | 付款义务、签证权 |
| 3 | 承包人 | 承包人义务、项目经理 | 工期延误责任、质量责任 |
| 4 | 监理人 | 监理职责、监理指示 | 签证权限、审核期限 |
| 5 | 工程质量 | 质量要求、质量保证措施 | 质量违约责任 |
| 6 | 安全文明施工与环境保护 | 安全文明施工费、措施费 | 安全文明施工费调整 |
| 7 | 工期和进度 | 开工、工期延误、暂停施工 | 工期索赔依据 |
| 8 | 材料与设备 | 材料设备供应、检验 | 材料价格调整 |
| 9 | 试验与检验 | 试验检验费用、重新检验 | 检测费用承担 |
| 10 | 变更 | 变更范围、变更估价、暂列金额 | 变更计价原则 |
| 11 | 价格调整 | 市场价格波动引起的调整 | 材料价格调差 |
| 12 | 合同价格、计量与支付 | 合同价格形式、预付款、进度款 | 付款条件、计量规则 |
| 13 | 验收和工程试车 | 竣工验收、竣工日期 | 竣工时间节点 |
| 14 | 竣工结算 | 结算申请、审核期限、最终结清 | 结算审核时限 |
| 15 | 缺陷责任与保修 | 缺陷责任期、质量保证金 | 质保金比例、返还 |
| 16 | 违约 | 发包人违约、承包人违约 | 违约责任、违约金 |
| 17 | 不可抗力 | 不可抗力确认、后果承担 | 不可抗力损失分担 |
| 18 | 保险 | 工程保险、人员工伤 | 保险费用承担 |
| 19 | 索赔 | 索赔程序、索赔期限 | 索赔时效、程序 |
| 20 | 争议解决 | 协商、调解、仲裁、诉讼 | 争议解决方式 |

### 2.2 合同价格形式

| 价格形式 | 适用范围 | 风险承担 | 结算方式 |
|---|---|---|---|
| 固定总价合同 | 工期短（1年内）、技术简单、图纸完备 | 承包人承担主要风险 | 一次性包死，除设计变更外不调整 |
| 固定单价合同 | 工期较长、技术较复杂、图纸不够详细 | 价格风险承包人承担，工程量风险发包人承担 | 按实际完成工程量×单价结算 |
| 可调价格合同 | 工期长、市场价格波动大、技术复杂 | 双方按约定分担风险 | 按约定调差调整 |
| 成本加酬金合同 | 特别复杂、紧急、新技术项目 | 发包人承担主要风险 | 成本实报实销+酬金 |

## 三、专业分包合同要点

### 3.1 分包合同核心条款

| 条款 | 关注点 | 常见风险 |
|---|---|---|
| 分包工程范围 | 明确分包内容、界面划分 | 界面不清导致扯皮 |
| 合同价款 | 计价方式、付款方式 | 付款节点不合理 |
| 工期 | 开工、竣工、总工期、与总包工期衔接 | 工期延误责任不清 |
| 质量标准 | 质量等级、验收标准 | 质量要求过高，价格未相应提高 |
| 安全文明施工 | 安全责任、安全费用 | 安全事故责任划分 |
| 材料供应 | 甲供材/乙供材划分、材料价格 | 材料供应不及时 |
| 工程变更 | 变更程序、变更计价 | 变更签证难、价款难结算 |
| 竣工验收 | 验收程序、资料要求 | 资料要求不明确 |
| 结算与支付 | 结算方式、付款比例、质保金 | 付款拖延、结算难 |
| 违约责任 | 工期延误、质量缺陷、付款违约 | 违约责任不对等 |

### 3.2 分包合同关键比例

| 项目 | 常规比例 | 说明 |
|---|---|---|
| 预付款比例 | 10%~30% | 包工包料的分包 |
| 进度款支付比例 | 70%~85% | 按月进度支付 |
| 竣工付款比例 | 85%~90% | 竣工验收后 |
| 结算付款比例 | 95%~97% | 结算完成后 |
| 质量保证金 | 3%~5% | 缺陷责任期满返还 |
| 履约保证金 | 5%~10% | 合同额的比例 |

## 五、材料设备采购合同要点

### 5.1 采购合同核心条款

| 条款 | 关注点 | 造价影响 |
|---|---|---|
| 产品名称、规格、型号 | 明确技术参数、质量标准 | 规格不明导致价格差异 |
| 数量、计量单位 | 数量正负偏差、计量单位 | 数量计算错误 |
| 价款与支付 | 单价、总价、付款方式 | 付款条件影响资金成本 |
| 质量标准 | 国家标准、行业标准、样品 | 质量缺陷损失 |
| 交货方式、地点 | 送货/自提、交货地点 | 运输费用承担 |
| 交货期限 | 交货时间、延期交货责任 | 工期延误损失 |
| 验收标准 | 验收方法、验收期限 | 质量问题发现时机 |
| 违约责任 | 延期交货、质量不合格 | 违约金、损失赔偿 |
| 保修与服务 | 保修期、售后服务 | 维修成本 |
| 争议解决 | 管辖法院、仲裁机构 | 争议解决成本 |

### 5.2 采购付款节点

| 付款节点 | 常规比例 | 说明 |
|---|---|---|
| 预付款 | 10%~30% | 大宗材料、定制设备 |
| 到货款 | 50%~70% | 货到验收合格 |
| 安装调试款 | 10%~20% | 安装调试合格后 |
| 质保金 | 5%~10% | 质保期满后 |

## 六、新疆地区合同特殊条款

### 6.1 冬季施工相关条款

**建议增加的特殊条款：**
1. 冬施期的确定：明确当地冬季施工起止日期
2. 冬施措施费的承担：约定冬施增加费的计算方式和承担方
3. 冬施停工的处理：冬施期间停工的工期和费用处理
4. 冬休期：约定冬季休工期是否计入总工期

### 6.2 材料运输与供应条款

**新疆特殊考虑：**
1. 运距约定：明确材料运距计算方式
2. 冬季运输：冬季道路封路对材料供应的影响
3. 偏远地区：偏远地区材料运输的特殊费用
4. 材料价格波动：约定材料价格涨跌的调整机制

### 6.3 民族地区特殊条款

**注意事项：**
1. 用工政策：当地民族用工比例要求
2. 节假日：少数民族节假日对工期的影响
3. 语言文字：合同文件的语言要求
4. 风俗习惯：施工期间对当地风俗习惯的尊重

## 七、合同审查要点清单

### 7.1 价款条款审查（最高风险）

- [ ] 合同价格形式是否明确
- [ ] 计价依据和计价方法是否清楚
- [ ] 材料价格调整的范围、方法、幅度是否约定
- [ ] 人工单价调整是否有约定
- [ ] 工程量变更的计价原则是否明确
- [ ] 暂估价、暂列金额的使用程序是否约定
- [ ] 付款节点和付款比例是否合理
- [ ] 结算审核期限和逾期审核的后果是否约定
- [ ] 质量保证金的比例和返还方式是否明确
- [ ] 违约金的计算方式和上限是否合理

### 7.2 工期条款审查

- [ ] 开工日期和竣工日期是否明确
- [ ] 总工期日历天数是否合理
- [ ] 工期延误的责任划分是否清楚
- [ ] 工期索赔的程序和期限是否约定
- [ ] 工期延误违约金的计算方式和上限是否明确
- [ ] 提前竣工奖的约定是否合理
- [ ] 冬季施工期是否计入工期
- [ ] 不可抗力对工期的影响如何处理

### 7.3 变更与索赔条款审查

- [ ] 变更的范围和程序是否明确
- [ ] 变更估价的原则和方法是否约定
- [ ] 变更签证的签字权限是否明确
- [ ] 索赔的程序和期限是否约定
- [ ] 索赔费用的组成和计算方法是否明确
- [ ] 索赔逾期失权条款是否合理
- [ ] 发包方逾期答复的默认条款是否存在

### 7.4 风险分担条款审查

- [ ] 市场价格波动风险如何分担
- [ ] 工程量变更风险如何分担
- [ ] 地质条件变化风险如何承担
- [ ] 不可抗力的损失分担是否约定
- [ ] 政策法规变化的风险如何承担
- [ ] 物价上涨的风险如何分担
