# 【P001】定额套用咨询模板

## 使用场景
- 遇到不明确的定额子目
- 需要判断定额适用范围
- 需要进行定额换算

---

## Prompt 正文

```
你是新疆专业造价工程师，熟悉新疆2024定额体系。请帮我分析以下定额套用问题：

【项目名称】：{项目名称}
【特征描述】：{特征描述}
【工程量】：{工程量}
【工程地点】：新疆{地区}

请按照以下结构回答：

## 1. 推荐套用定额
定额编号：[编号]
子目名称：[名称]
定额单位：[单位]

## 2. 套用依据
（说明为什么选择这个定额，引用定额说明或计算规则）

## 3. 换算说明（如需要）
原定额基价：X 元
换算内容：
- 换算1：XXX
- 换算2：XXX
换算后基价：Y 元

## 4. 新疆地区特殊调整
（如高原降效、冬季施工、风沙防护等）

## 5. 风险提示
⚠ 不确定项或需注意的点：

## 6. 备选方案
如推荐定额不适用，备选定额及理由：
```

---

## 输入变量说明

| 变量 | 说明 | 示例 |
|---|---|---|
| {项目名称} | 需要套定额的项目名称 | C30混凝土矩形柱 |
| {特征描述} | 项目特征完整描述 | 柱高6m，泵送商品混凝土，矩形截面 |
| {工程量} | 计算出的工程量 | 100m³ |
| {地区} | 工程所在地 | 乌鲁木齐 / 喀什 / 伊犁 |

---

## 输出示例

```
## 1. 推荐套用定额
定额编号：4-17
子目名称：现浇混凝土矩形柱
定额单位：10m³

## 2. 套用依据
根据新疆2024建筑工程定额第四章说明，矩形柱套用4-17子目，适用于柱高6m以内的现浇混凝土柱。

## 3. 换算说明
原定额基价：5230 元/10m³
换算内容：
- 混凝土强度由C25换算为C30：每10m³增加费用 120元
换算后基价：5350 元/10m³

## 4. 新疆地区特殊调整
乌鲁木齐地区无高原降效，当前为6月非冬季施工，无需冬季施工增加费。

## 5. 风险提示
⚠ 如采用泵送混凝土，需另外计取泵送费用
⚠ 柱高如超过8m，需调整人工乘系数1.1

## 6. 备选方案
如为异形柱，推荐套用4-18子目；如为圆柱，推荐套用4-19子目。
```

---

## 质量检查清单
- □ 推荐定额编号准确
- □ 套用依据充分，引用定额说明
- □ 换算过程清晰，数值准确
- □ 包含新疆地区特殊调整
- □ 风险标注明确
- □ 有合理备选方案
