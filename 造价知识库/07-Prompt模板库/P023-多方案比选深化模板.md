## 一、技术方案经济性对比（全生命周期成本 LCC）

### 1.1 全生命周期成本计算

全生命周期成本公式：
```
LCC = I₀ + Σ (O&M)t / (1+i)^t - S / (1+i)^n

其中：
I₀ = 初始投资
(O&M)t = 第t年的运营维护成本
i = 折现率（基准收益率）
S = 残值
n = 计算期（年）
```
