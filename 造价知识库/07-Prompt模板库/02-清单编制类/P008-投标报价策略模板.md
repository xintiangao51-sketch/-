## 质量检查清单

- [ ] 不平衡报价在招标文件允许范围内（未触发废标条款）
- [ ] 调增和调减项目的总平衡控制价基本一致
- [ ] 降幅分配合理，无极端低价出现
- [ ] 未低于成本价（满足异常低价评审要求）
- [ ] 考虑了评标办法和报价评分规则
- [ ] 报价策略与施工组织设计协调一致
- [ ] 风险应对措施在实际操作中可行
- [ ] 总报价有竞争力和合理性
- [ ] 对各分部的调整幅度有合理解释
- [ ] 策略方案获得公司内部经营决策层确认
