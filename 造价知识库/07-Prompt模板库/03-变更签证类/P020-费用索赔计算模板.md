### 7.2 索赔谈判策略
1. **分层索赔策略**：先易后难，先确认双方无争议部分
2. **打包索赔策略**：将多个小金额索赔打包处理
3. **换量不换价**：可适当让渡单价，但确认工程量
4. **结算时处理**：争议不大的可留到结算时一并处理
5. **工期换费用**：如工期不紧，可考虑放弃部分费用换取工期顺延

## 八、索赔报告撰写建议

### 8.1 索赔意向书结构
```
致：{监理/业主单位名称}
    1. 索赔事件描述
    2. 索赔依据（合同条款、法规）
    3. 索赔意向（工期X天，费用X元）
    4. 后续将补充详细资料
    5. 附件：相关证据材料
承包单位：{单位名称}
日期：{年月日}
```

### 8.2 正式索赔报告结构
```
第一部分 索赔事件概述
第二部分 索赔依据（合同+法规+事实）
第三部分 工期索赔计算（如有）
第四部分 费用索赔详细计算
第五部分 证据材料清单
第六部分 索赔金额汇总
附件：全部证据材料
```

### 8.3 撰写注意事项
1. 事实陈述客观，避免情绪化语言
2. 每项索赔都要有合同或法规依据
3. 数据计算准确，过程清晰可追溯
4. 证据材料编号，便于查阅核对
5. 金额大写小写一致

## 附录：费用索赔计算原则速查

### 可索赔费用一览表

| 费用项目 | 工程变更 | 窝工停工 | 业主延误 | 不可抗力 |
|---|---|---|---|---|
| 人工费增加 | ✅ | ✅（窝工费） | ✅ | ⚠（只赔窝工） |
| 材料费增加 | ✅ | ✅（超期储存） | ✅ | ❌（各自承担） |
| 机械费增加 | ✅ | ✅（停滞费） | ✅ | ⚠（停滞费） |
| 现场管理费 | ✅ | ⚠（部分） | ✅ | ❌ |
| 企业管理费 | ✅ | ⚠（争议） | ✅ | ❌ |
| 利润 | ✅ | ❌ | ❌ | ❌ |
| 资金成本 | ⚠（延期付款时） | ⚠（需举证） | ✅ | ❌ |

### 计算要点
1. **窝工费**：按窝工单价，不是全工日单价，一般为定额人工的40%~60%
2. **机械停滞费**：自有机械按折旧费，租赁机械按租赁费
3. **管理费**：窝工一般不计总部管理费，工程量增加计取
4. **利润**：只有工程量增加/变更类索赔才计利润
5. **税金**：所有费用最终都要计取税金
