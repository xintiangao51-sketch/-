# 【知识卡片 K001】工程造价构成

## 核心知识点
| 知识点 | 掌握程度 | 说明 |
|---|---|---|
| 直接费构成 | 精通 | 人工费 + 材料费 + 机械费 |
| 管理费计算 | 精通 | (人工费 + 机械费) × 管理费费率（见FEE_POLICY） |
| 利润计算 | 精通 | (人工费 + 机械费) × 利润率（见FEE_POLICY） |
| 规费计算 | 精通 | 人工费 × 规费费率（见FEE_POLICY） |
| 税金计算 | 精通 | 税前总造价 × 税率（见FEE_POLICY） |
| 人工调增规则 | 精通 | 只计税金，不计管理费、利润、规费 |

## 计算公式
```
直接费 = 人工费 + 材料费 + 机械费
管理费 = (人工费 + 机械费) × 管理费费率
利润 = (人工费 + 机械费) × 利润率
规费 = 人工费 × 规费费率
税金 = (直接费 + 管理费 + 利润 + 规费) × 税率
总造价 = 直接费 + 管理费 + 利润 + 规费 + 税金
```
*所有费率数值统一从 memory/00_global/FEE_POLICY.md 读取，不写死*
