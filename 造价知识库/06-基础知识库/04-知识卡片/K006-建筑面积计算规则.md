# 【知识卡片 K006】建筑面积计算规则

## 核心知识点
规范依据：GB/T 50353-2013《建筑工程建筑面积计算规范》

### 计算全面积
| 情形 | 计算规则 |
|---|---|
| 建筑物内层高 ≥ 2.20m | 按外墙外围水平面积计算 |
| 门厅、大厅 | 按一层计算面积 |
| 走廊、檐廊 | 按围护结构外围水平面积计算 |
| 有顶盖的架空走廊 | 按围护结构外围水平面积计算 |
| 地下室、半地下室 | 按其外墙上口外围水平面积计算 |

### 计算1/2面积
| 情形 | 计算规则 |
|---|---|
| 层高 < 2.20m 的建筑物 | 计算 1/2 面积 |
| 有顶盖无围护结构的场馆看台 | 按顶盖水平投影面积 1/2 计算 |
| 挑出宽度 ≥ 2.10m 的无柱雨篷 | 按结构板水平投影面积 1/2 计算 |
| 室外楼梯 | 按所依附建筑物自然层水平投影面积 1/2 计算 |
| 阳台（不论封闭与否） | 按结构底板水平投影面积 1/2 计算 |

### 不计算面积
| 情形 |
|---|
| 与建筑物不连通的建筑部件 |
| 层高 < 2.20m 的设备层、管道层 |
| 台阶、散水、坡道、露天游泳池 |
| 操作平台、上料平台、安装箱 |
| 构筑物（烟囱、烟道、水塔、油罐等） |

## 应用场景
- ✅ 建筑面积指标计算
- ✅ 单方造价指标分析
- ✅ 脚手架、垂直运输等措施费计算
