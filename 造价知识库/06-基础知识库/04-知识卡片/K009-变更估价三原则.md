# 【知识卡片 K009】变更估价三原则

## 核心知识点
依据：《建设工程工程量清单计价规范》GB50500-2013 第9.3.1条

### 原则一：已有适用子目 → 直接套用
```
已标价工程量清单中有适用于变更工程项目的
→ 采用该项目的单价
```
**适用条件**：
- 变更工作与子目工作内容完全相同
- 施工工艺、方法不变
- 地理位置、工程性质相似
- 没有增加额外资源

### 原则二：只有类似子目 → 参照调整
```
已标价工程量清单中没有适用但有类似于变更工程项目的
→ 可在合理范围内参照类似项目的单价
```
**调整方法**：
- 人工、材料、机械用量调整
- 合同、招标文件、政策文件或项目策略已锁定的费率保持一致
- 缺少费率依据时，不得默认套用固定值，标注 `⚠费率待确认`
- 双方协商确认调整幅度

### 原则三：无适用无类似 → 重新组价
```
已标价工程量清单中没有适用也没有类似于变更工程项目的
→ 由承包人根据变更工程资料、计量规则和计价办法、
   工程造价管理机构发布的信息价格和承包人报价浮动率
   提出变更工程项目的单价
→ 报发包人确认后调整
```
**报价浮动率**：
- 招标工程：报价浮动率 L = (1 - 中标价/招标控制价) × 100%
- 非招标工程：报价浮动率 L = (1 - 报价值/施工图预算) × 100%

## 应用场景
- ✅ 设计变更费用计算
- ✅ 现场签证费用确认
- ✅ 索赔费用计算
- ✅ 工程价款调整
