# 【知识卡片 K005】工程量计算规则总则

## 核心知识点
### 计算依据
1. 施工图纸及设计说明
2. 《建设工程工程量清单计价规范》GB50500-2013
3. 《房屋建筑与装饰工程工程量计算规范》GB50854-2013
4. 招标文件及答疑纪要
5. 经批准的施工组织设计

### 计算原则
| 原则 | 说明 |
|---|---|
| 按图计算 | 严格按照图纸尺寸，不得随意加大或缩小 |
| 规则一致 | 同一工程采用相同计算规则，保持口径一致 |
| 计量单位统一 | 与定额或清单规范单位一致 |
| 精度要求 | 保留小数：立方米、平方米、米 2位，吨 3位，个取整 |
| 不重不漏 | 分项工程划分清晰，避免重复计算或漏算 |

### 常用计量单位
| 项目类型 | 单位 | 保留小数 |
|---|---|---:|
| 体积（混凝土、土方） | m³ | 2位 |
| 面积（抹灰、模板） | m² | 2位 |
| 长度（管线、栏杆） | m | 2位 |
| 重量（钢筋、钢材） | t | 3位 |
| 数量（门窗、设备） | 个/套 | 取整 |
