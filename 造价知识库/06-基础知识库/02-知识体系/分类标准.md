# 造价基础知识库分类标准

> 任务编号：HERMES-20260618-002  
> 版本：v1.0  
> 日期：2026-06-19  
> 适用范围：造价基础知识库、Prompt 模板库、规则库、案例库、Agent/RAG 检索  
> 维护口径：新疆项目优先新疆 2024 版定额；费率按合同、招标文件、政策文件、项目策略或 FEE_POLICY 动态确认，缺依据标 `⚠费率待确认`

---

## 一、分类原则

| 原则 | 规则 | 验收口径 |
|---|---|---|
| 业务优先 | 按造价业务场景分类，不按资料来源堆放 | 能直接服务清单、组价、合同、签证、结算等任务 |
| 新疆优先 | 涉及新疆项目时，定额和本地化规则优先归入新疆专项 | 不得把全国通用资料直接当新疆依据 |
| 原始与加工分离 | 原始资料、规则提炼、案例复盘、模板输出分层存放 | 原始资料不被改写，规则文档有来源字段 |
| 可检索 | 每篇文档必须有标题、标签、适用范围、来源、版本 | 适合 Obsidian、RAG、Agent 调用 |
| 可追溯 | 金额、费率、定额、法条、市场价必须能回到来源 | 不确定处必须标 `⚠` |

## 三、二级分类维度

### 3.1 按专业

| 专业 | 适用内容 | 示例 |
|---|---|---|
| 建筑 | 土方、砌筑、混凝土、钢筋、装饰 | 建筑面积、钢筋算量、混凝土清单 |
| 安装 | 电气、给排水、暖通、消防、设备 | 电缆桥架、管道安装、设备调试 |
| 市政 | 道路、管网、照明、绿化、市政附属 | 道路基层、检查井、路灯基础 |
| 园林 | 苗木、景观、小品、铺装 | 苗木养护、景观铺装 |
| 公路 | 路基、路面、桥涵、交安 | 路基填筑、沥青面层 |
| 水利 | 渠道、堤防、水工建筑物、机电 | ⚠具体规则待补权威资料 |

### 3.2 按阶段

| 阶段 | 资料类型 | 输出要求 |
|---|---|---|
| 估算 | 指标、类似项目、经验参数 | 标明项目类型、地区、时间 |
| 概算 | 初设工程量、概算定额、设备价 | 标明概算依据和价格时点 |
| 预算 | 施工图、清单、定额、信息价 | 四维拆解、定额依据、费率依据 |
| 招投标 | 招标清单、控制价、投标报价 | 标明招标口径和报价策略 |
| 施工过程 | 变更、签证、进度款、材料调差 | 证据链、时效、审批状态 |
| 结算审计 | 竣工图、结算书、审计意见 | 差异清单、争议点、处理建议 |

### 3.3 按成果类型

| 类型 | 用途 | 存放建议 |
|---|---|---|
| 原始资料 | PDF、规范、合同、图纸、网页截图 | `01-原始资料/` |
| 知识卡片 | 单一知识点复用 | `04-知识卡片/` |
| 规则汇总 | 可被 Agent/RAG 调用的结构化规则 | `03-规则库/` |
| Prompt 模板 | 可直接复制使用的工作提示词 | `D:\造价知识库\07-Prompt模板库\` |
| 案例复盘 | 项目实战经验沉淀 | 后续 `05-案例库/` |
| 台账表格 | 项目、价格、参数、任务追踪 | 后续 `06-台账/` |

## 六、归档规则

| 资料状态 | 处理方式 | 示例 |
|---|---|---|
| 原始资料 | 保留原样，新增索引说明 | PDF、网页截图、合同扫描件 |
| 已提炼规则 | 写入规则库，注明来源 | 定额套用原则、取费依据优先级 |
| 待核资料 | 标 `⚠待核验`，不得进入最终结论 | 未确认市场价、缺页合同 |
| 冲突资料 | 建冲突记录，交 Hermes 裁决 | 合同费率与招标文件不一致 |
| 过期资料 | 标明失效原因，移入历史归档 | 旧版定额、过期信息价 |
