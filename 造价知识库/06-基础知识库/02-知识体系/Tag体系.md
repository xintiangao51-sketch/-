# 造价基础知识库 Tag 体系

> 任务编号：HERMES-20260618-002  
> 版本：v1.0  
> 日期：2026-06-19  
> 用途：Obsidian 检索、RAG 检索、Agent 调用、知识卡片归档  
> 规则：统一使用中文分层 Tag，避免同义词混用。

---

## 一、Tag 命名规则

| 规则 | 标准 | 示例 |
|---|---|---|
| 分层命名 | `#维度/值` | `#地区/新疆` |
| 中文优先 | 专业词用中文，不中英混写 | `#业务/清单`，不用 `#BOQ` |
| 单一含义 | 一个 Tag 只表达一个维度 | `#阶段/预算` 不等于 `#业务/报价` |
| 低频合并 | 低频同义词合并到推荐 Tag | `#签证` 合并为 `#业务/签证` |
| 风险明确 | 风险级别使用红黄蓝 | `#风险/红` |

## 二、核心 Tag 维度

### 2.1 地区

| Tag | 用途 | 说明 |
|---|---|---|
| `#地区/新疆` | 新疆项目通用 | 涉及新疆定额、人工价、本地化专项 |
| `#地区/全国` | 全国通用规范或方法 | 不能直接替代新疆项目依据 |
| `#地区/乌鲁木齐` | 地州级资料 | 人工价、材料价、案例 |
| `#地区/伊犁` | 地州级资料 | 人工价、运输、冬季施工 |
| `#地区/克州` | 地州级资料 | 高原、边境、特殊施工 |
| `#地区/阿勒泰` | 地州级资料 | 严寒、冬施、运输 |
| `#地区/喀什` | 地州级资料 | 高原、风沙、抗震 |

### 2.2 专业

| Tag | 用途 |
|---|---|
| `#专业/建筑` | 土建、装饰、房建 |
| `#专业/安装` | 电气、给排水、暖通、消防、设备 |
| `#专业/市政` | 道路、管网、照明、附属 |
| `#专业/园林` | 绿化、景观、小品、铺装 |
| `#专业/公路` | 路基、路面、桥涵、交安 |
| `#专业/水利` | 渠道、堤防、水工建筑物 |
| `#专业/综合` | 跨专业规则、通用方法 |

### 2.3 阶段

| Tag | 用途 |
|---|---|
| `#阶段/估算` | 投资估算、指标测算 |
| `#阶段/概算` | 初设概算、概算控制 |
| `#阶段/预算` | 施工图预算、控制价 |
| `#阶段/招投标` | 招标清单、投标报价、清标 |
| `#阶段/施工过程` | 变更、签证、进度款、材料调差 |
| `#阶段/结算审计` | 竣工结算、审计、争议处理 |

### 2.4 业务

| Tag | 用途 |
|---|---|
| `#业务/清单` | 清单编制、项目特征、工程量清单 |
| `#业务/定额` | 定额套用、换算、补充定额 |
| `#业务/组价` | 综合单价、人材机、四维拆解 |
| `#业务/取费` | 管理费、利润、规费、税金、措施费 |
| `#业务/材料` | 材料询价、信息价、运杂费 |
| `#业务/变更` | 设计变更、工程变更、价款调整 |
| `#业务/签证` | 现场签证、工程量确认、证据链 |
| `#业务/索赔` | 工期索赔、费用索赔、反索赔 |
| `#业务/结算` | 竣工结算、审核、审计对账 |
| `#业务/合同` | 合同条款、付款、调价、争议 |
| `#业务/方案比选` | 技术经济比较、成本优化 |

### 2.5 依据

| Tag | 用途 | 注意事项 |
|---|---|---|
| `#依据/合同` | 合同条款、专用条款 | 必须记录条款号或页码，缺失标 `⚠合同条款待确认` |
| `#依据/招标文件` | 招标控制价、投标口径 | 记录文件名、章节、发布日期 |
| `#依据/政策文件` | 取费、税率、地方政策 | 记录发文机关、文号、发布日期；缺失标 `⚠政策依据待确认` |
| `#依据/FEE_POLICY` | 系统权威费率文件 | 仅作为当前系统口径，项目有明确文件时优先项目文件 |
| `#依据/市场询价` | 材料、机械、劳务询价 | 记录询价日期、供应商、税率、运输条件 |
| `#依据/历史项目` | 指标、案例、经验数据 | 不能直接作为最终定价依据，需复核适用性 |
| `#依据/规范` | 清单规范、计算规范、验收规范 | 不得虚构规范编号 |

### 2.6 新疆专项

| Tag | 用途 |
|---|---|
| `#专项/冬季施工` | 11 月至次年 3 月或合同约定冬施期 |
| `#专项/高原降效` | 海拔影响人工、机械效率 |
| `#专项/风沙防护` | 沙漠、戈壁、风沙区防护与降效 |
| `#专项/抗震` | 新疆设防烈度、抗震构造措施 |
| `#专项/边境项目` | 口岸、边防、远离城镇项目 |
| `#专项/运距` | 新疆长距离运输、二次倒运 |

### 2.7 风险

| Tag | 用途 | 判定口径 |
|---|---|---|
| `#风险/红` | 重大风险 | 可能导致重大经济损失、合同争议、索赔失败 |
| `#风险/黄` | 中等风险 | 可能造成成本增加、资料退回、审计争议 |
| `#风险/蓝` | 一般风险 | 格式、效率、资料完整性问题 |

### 2.8 状态

| Tag | 用途 |
|---|---|
| `#状态/已核验` | 来源明确且可追溯 |
| `#状态/待核验` | 内容可参考但不能定稿 |
| `#状态/待补资料` | 缺合同、图纸、价格、定额等资料 |
| `#状态/已归档` | 已完成 Hermes 质检或正式归档 |
| `#状态/历史版本` | 旧口径、旧资料、失效依据 |

## 三、错误 Tag 对照

| 错误写法 | 推荐写法 | 原因 |
|---|---|---|
| `#新疆造价` | `#地区/新疆` + `#业务/组价` | 地区和业务分开，便于检索 |
| `#BOQ` | `#业务/清单` | 中文统一，避免中英混用 |
| `#签证变更索赔` | `#业务/签证` + `#业务/变更` + `#业务/索赔` | 一个 Tag 不混多个业务 |
| `#高风险` | `#风险/红` | 风险分级统一红黄蓝 |
| `#管理费28` | `#业务/取费` + `#依据/FEE_POLICY` | 不在 Tag 写死费率 |
| `#2024定额` | `#业务/定额` + `#地区/新疆` | 定额版本写在正文来源字段 |

## 五、维护规则

1. 新增 Tag 前先查本文件，避免同义重复。
2. 项目专属 Tag 不写入全局体系，优先写在项目记忆或项目目录内。
3. 费率、单价、定额编号不得作为 Tag 常量。
4. 缺依据的内容必须同时打 `#状态/待核验` 或 `#状态/待补资料`。
5. Hermes 质检通过后方可把核心 Tag 追加到全局索引。
