# 2025版房屋建筑与装饰工程工程量计算规范

> **核心提炼**：GB 50854-2025 替代 GB 50854-2013，统一工程量计算规则，规范清单编制行为。

---

## 📋 核心术语定义

| 术语 | 定义 |
|------|------|
| **工程量计算** | 按设计图纸、施工组织设计及国家规定计算规则，进行工程数量计算，简称工程计量 |
| **房屋建筑** | 固定地点提供庇护的实体，分为工业建筑与民用建筑 |
| **工业建筑** | 生产用建筑物：车间、动力站、库房、运输设施等 |
| **民用建筑** | 非生产性建筑：住宅、办公楼、学校、医院、商店、体育馆等 |

---

## ⚖️ 工程计量核心规则

### 3.0.1 计量依据
1. 经审定的施工设计图纸及说明
2. 经审定的施工组织设计或施工方案
3. 经审定的其他有关技术经济文件

### 3.0.2 计价规范衔接
工程实施过程中的计量应按照现行国家标准《建设工程工程量清单计价规范》GB 50500 的相关规定执行。

### 3.0.3 计量单位选择
附录中有两个或以上计量单位的，结合工程实际确定其一。**同一工程项目计量单位应一致**。

### 3.0.4 有效位数规则
| 单位 | 保留位数 |
|------|---------|
| t（吨） | 小数点后3位，第4位四舍五入 |
| m、m²、m³、kg | 小数点后2位，第3位四舍五入 |
| 个、件、根、组、系统 | 取整数 |

### 3.0.5 工作内容包含规则
本规范各项目仅列出主要工作内容，除另有规定外，**视为已包括完成该项目所列或未列的全部工作内容**。

### 3.0.6 跨专业衔接规则
| 专业类别 | 执行规范 |
|---------|---------|
| 电气、给排水、消防等安装工程 | GB 50856 通用安装工程工程量计算规范 |
| 仿古建筑工程 | GB 50855 仿古建筑工程工程量计算规范 |
| 室外地（路）面、室外给排水 | GB 50857 市政工程工程量计算规范 |
| 爆破法施工石方工程 | GB 50862 爆破工程工程量计算规范 |

## 📝 工程量清单编制规则

### 4.1 一般规定

#### 4.1.1 编制依据
1. 本规范 + GB 50500 建设工程工程量清单计价规范
2. 国家或省级、行业建设主管部门颁发的计价依据和办法
3. 建设工程设计文件
4. 与工程项目有关的标准、规范、技术资料
5. 拟定的招标文件
6. 施工现场情况、工程特点及常规施工方案
7. 其他相关资料

#### 4.1.2 其他项目/规费/税金
应按照 GB 50500 的相关规定编制。

#### 4.1.3 补充项目规则
- 编制人可补充附录未包括的项目
- 报省级或行业工程造价管理机构备案
- 汇总报住建部标准定额研究所
- **编码规则**：01 + B + 三位阿拉伯数字，从 01B001 起顺序编制
- 同一招标工程不得重码
- **补充要件**：项目名称、项目特征、计量单位、工程量计算规则、工作内容
- 不能计量的措施项目：需附名称、工作内容及包含范围

### 4.2 分部分项工程

#### 4.2.1 编制五要素
项目编码、项目名称、项目特征、计量单位、工程量计算规则

#### 4.2.2 编码规则（十二位）
- 一至九位：按附录规定设置
- 十至十二位：根据项目名称和特征设置
- ⚠️ **同一招标工程项目编码不得重码**

#### 4.2.3 项目名称
按附录规定 + 拟建工程实际确定

#### 4.2.4 项目特征
按附录规定 + 拟建工程实际予以描述

#### 4.2.5 工程量计算
按附录规定的计算规则计算

#### 4.2.6 计量单位
按附录规定确定

#### 4.2.7 模板工程特殊规则（关键条款）
- 现浇混凝土工程项目"工作内容"已包括模板工程
- 措施项目中单列了现浇混凝土模板工程项目
- 招标人根据工程实际选用
- ⚠️ **若措施项目清单未编列模板项目 → 模板不单列，综合单价中必须包含模板费用**

#### 4.2.8 预制混凝土构件规则
- 按现场制作编制，"工作内容"已包括模板工程
- 若采用成品预制构件 → 构件成品价（含模板、钢筋、混凝土等所有费用）计入综合单价

#### 4.2.9 金属结构构件规则
- 按成品编制项目，构件成品价计入综合单价
- 若采用现场制作 → 包括制作的所有费用

#### 4.2.10 门窗工程规则（橱窗除外）
- 按成品编制项目，门窗成品价计入综合单价
- 若采用现场制作 → 包括制作的所有费用

## 🎯 关键要点总结

| 要点 | 核心内容 |
|------|---------|
| 有效位数 | t→3位，m/m²/m³/kg→2位，个/件等→整数 |
| 模板工程 | 可单列也可包含在综合单价，看措施清单是否列项 |
| 成品构件 | 预制砼、金属结构、门窗均按成品计价，现场制作需包含全部费用 |
| 跨专业衔接 | 按专业分别执行对应规范 |
| 补充项目 | 01B001起顺序编码，必须报备案 |
