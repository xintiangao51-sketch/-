## 🏗️ 台时费费用组成

### 第一类费用（折旧费+修理+替换设备）
| 费用项目 | 计算方法 |
|---------|---------|
| **折旧费** | 机械预算价格×(1-残值率)/耐用总台班 |
| **大修理费** | 一次大修理费×寿命期大修理次数/耐用总台班 |
| **经常修理费** | 大修理费×经常修理费系数 |
| **替换设备费** | 轮胎、钢丝绳、电缆、传动皮带等 |
| **安装拆卸费** | 机械安装拆卸及辅助设施费 |

### 第二类费用（可变费用）
| 费用项目 | 计算方法 |
|---------|---------|
| **人工费** | 机上人工工日数×人工单价 |
| **动力燃料费** | 燃料消耗量×燃料单价（汽油、柴油、电、煤、水） |

## 🎯 2024版核心变化要点

| 变化类型 | 主要内容 |
|---------|---------|
| **机械更新** | 新增现代化大型施工机械（大吨位吊车、盾构等） |
| **能耗标准** | 更新燃料消耗指标，符合国五国六排放标准 |
| **人工工日** | 调整机上人工工日消耗标准 |
| **折旧年限** | 根据最新设备寿命更新折旧年限 |
| **残值率** | 统一调整各类机械残值率标准 |
| **新能源** | 新增电动、混合动力工程机械台时费 |

## 🔍 与土建定额的差异

| 对比项 | 水利工程台时费 | 建设工程台班费 |
|-------|--------------|--------------|
| **计量单位** | 台时（小时） | 台班（8小时） |
| **人工工日** | 按工时计算 | 按工日计算 |
| **机械类型** | 含大型工程船舶、专用水工机械 | 以通用建筑机械为主 |
| **适用范围** | 水利水电、防洪、水土保持 | 房屋建筑、市政、安装 |

---

## 💡 使用注意事项

| 注意事项 | 说明 |
|---------|------|
| **单位换算** | 1台班 = 8台时，注意单位转换 |
| **地区调整** | 第二类费用（人工、燃料）按当地价格调整 |
| **停滞费用** | 机械停滞按折旧费+机上人工费计算 |
| **租赁机械** | 市场租赁价与定额台时费差异需调整 |
| **大型设备** | 特大型设备安拆费、进出场费单独计算 |

## 📌 新疆地区适用要点

| 要点 | 说明 |
|-----|------|
| **偏远地区运输** | 机械进出场运输费用需单独考虑 |
| **冬季施工** | 低温启动、防冻液等额外费用另行计算 |
| **高海拔降效** | 高原地区机械效率降效系数调整 |
| **燃料价格** | 新疆地区柴油价格与内地差异调整 |
