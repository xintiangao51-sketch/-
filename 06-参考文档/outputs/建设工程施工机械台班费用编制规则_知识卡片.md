# 建设工程施工机械台班费用编制规则

> **核心提炼**：住建部标准定额研究所主编，2015年9月1日起施行，与2015版消耗量定额配套使用，规定施工机械台班费用的组成、计算方法和费用标准，是定额编制和工程造价的基础依据。

## 🏗️ 台班费用组成结构

### 第一类费用（不变费用）
| 费用项目 | 说明 |
|---------|------|
| **折旧费** | 机械在规定使用年限内陆续回收原值的费用 |
| **大修理费** | 机械达到规定大修理间隔台班必须的大修理费用 |
| **经常修理费** | 机械日常保养和临时故障排除费用 |
| **安拆费及场外运费** | 机械安装、拆卸及场外运输费用 |

### 第二类费用（可变费用）
| 费用项目 | 说明 |
|---------|------|
| **人工费** | 机上司机和其他操作人员的人工费 |
| **燃料动力费** | 机械运转消耗的煤、油、电等费用 |
| **其他费用** | 养路费、车船使用税、保险费等 |

---

## 📊 台班单价计算公式

```
机械台班单价 = 折旧费 + 大修理费 + 经常修理费 
              + 安拆费及场外运费 + 人工费 
              + 燃料动力费 + 其他费用
```

## 🔍 与新疆定额的关系

| 关系项 | 说明 |
|-------|------|
| **基准作用** | 新疆2020版消耗量定额机械费用的编制基础 |
| **地区调整** | 新疆地区可根据当地人工、燃料价格调整第二类费用 |
| **运输距离** | 新疆地区机械场外运输距离可适当调整 |
| **冬施影响** | 冬季施工增加的机械降效费用另行计算 |

---

## ⚠️ 使用注意事项

| 注意事项 | 说明 |
|---------|------|
| **费用分类** | 第一类费用全国统一，第二类费用地区调整 |
| **机械停滞** | 停滞台班费按折旧费+人工费+其他费用计算 |
| **大型机械** | 特大型机械安拆费需单独计算 |
| **自有vs租赁** | 自有机械按台班单价，租赁机械按市场租赁价 |
| **增值税** | 营改增后机械台班单价需除税计算 |
