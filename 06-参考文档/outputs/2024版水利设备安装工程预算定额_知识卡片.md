# 2024版水利设备安装工程预算定额

> **核心提炼**：水利部水总〔2024〕323号发布，2025年4月1日起执行，替代水建管〔2003〕523号，覆盖水力机械、电气、金属结构、自动化控制设备等全专业安装工程。

---

## 📋 基本信息

| 项目 | 内容 |
|------|------|
| **发布单位** | 中华人民共和国水利部 |
| **文号** | 水总〔2024〕323号 |
| **发布日期** | 2024年12月9日 |
| **执行日期** | 2025年4月1日 |
| **主编单位** | 水利部水利建设经济定额站 |
| **体系定位** | 水利工程"一规六定"之安装专业定额 |
| **替代旧版** | 水建管〔2003〕523号中小型+水建管〔2001〕523号全套 |

---

## 📚 定额章节结构

| 章号 | 章节名称 | 主要内容 |
|:----:|---------|---------|
| **第一篇 | 总说明 | 费用组成、计算规则、人工材料机械标准 |
| **第一章** | 水力机械辅助设备安装 | 泵、风机、压缩机、阀门等 |
| **第二章** | 水轮机安装 | 各型水轮机、贯流式水轮机 |
| **第三章** | 水轮发电机安装 | 各型发电机、励磁系统 |
| **第四章** | 桥式起重机安装 | 桥机、门式起重机、启闭机 |
| **第五章** | 闸门安装 | 平面闸门、弧形闸门、人字门 |
| **第六章** | 压力钢管制作与安装 | 钢管制作、运输、安装 |
| **第七章** | 电气设备安装 | 变压器、断路器、开关柜 |
| **第八章** | 自动化控制设备安装 | PLC、自动化元件、监控系统 |
| **第九章** | 电缆敷设 | 电力电缆、控制电缆、桥架 |
| **第十章** | 接地装置 | 接地极、接地母线、接地测试 |
| **附录 | 施工机械台时费定额 | 安装专用机械台时费 |

---

## 🔗 与水土保持概算定额的差异

| 对比项 | 水利设备安装预算定额 | 水土保持工程概算定额 |
|-------|-------------------|-------------------|
| **精度** | 预算级，子目细 | 概算级，子目粗 |
| **适用阶段** | 施工图预算、招投标、结算 | 可研、初设概算 |
| **子目数量** | 多，覆盖全专业 | 少，主要常用设备 |
| **计算精度** | 高，按重量、规格、型号详细 | 粗，综合考虑扩大指标 |
