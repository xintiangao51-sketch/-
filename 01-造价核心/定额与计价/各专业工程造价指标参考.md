# 各专业工程造价指标参考（新疆地区）

> 综合新疆2024定额及2025年上半年市场价编制，适用于投资估算和初步设计概算。指标按乌鲁木齐基准价计算，各地州需按地区系数调整。

---

## 一、住宅工程

### 1.1 多层住宅（6层及以下，砖混/框架）

| 费用项 | 砖混结构 | 框架结构 |
|-------|---------|---------|
| 土建工程 | 1,150-1,350元/m² | 1,350-1,550元/m² |
| 装饰工程（毛坯） | 180-250元/m² | 200-280元/m² |
| 安装工程 | 280-350元/m² | 300-380元/m² |
| 措施费 | 180-220元/m² | 200-250元/m² |
| **合计** | **1,790-2,170元/m²** | **2,050-2,460元/m²** |

**费用占比分析（框架结构）**：
- 土建工程：62-65%
- 装饰工程：10-12%
- 安装工程：14-16%
- 措施费：9-11%

**新疆特殊说明**：
- 多层住宅在新疆城镇仍为主力，地基处理费比内地高15-20%（冻土+抗震）
- 砖混结构调整到乌鲁木齐以南地区仍有市场，北疆建议用框架
- 外墙保温费占土建成本8-10%（内地5-6%）

### 1.2 高层住宅（12-18层，剪力墙）

| 费用项 | 普通装修（毛坯交付） | 精装交付 |
|-------|-------------------|---------|
| 土建工程 | 1,600-1,850元/m² | 1,600-1,850元/m² |
| 装饰工程 | 200-300元/m² | 800-1,200元/m² |
| 安装工程 | 350-450元/m² | 380-480元/m² |
| 措施费 | 250-320元/m² | 260-330元/m² |
| **合计** | **2,400-2,920元/m²** | **3,040-3,860元/m²** |

**费用占比分析（毛坯交付）**：
- 土建工程：62-65%
- 装饰工程：8-10%
- 安装工程：14-16%
- 措施费：10-12%

**新疆特殊说明**：
- 高层住宅在乌鲁木齐、克拉玛依等城市占比70%以上
- 地暖费用含在土建中，约80-120元/m²
- 断桥铝窗费用约180-280元/m²（比内地高20-30%）
- 抗震钢筋HRB400E用量增加，比内地多8-12kg/m²

### 1.3 高层住宅（25-33层，剪力墙）

| 费用项 | 指标（元/m²） |
|-------|-------------|
| 土建工程 | 1,800-2,100 |
| 装饰工程（毛坯） | 200-300 |
| 安装工程 | 380-480 |
| 措施费 | 280-350 |
| **合计** | **2,660-3,230** |

**费用占比分析**：
- 土建工程：60-63%
- 装饰工程：8-10%
- 安装工程：15-17%
- 措施费：10-12%

**新疆特殊说明**：
- 超高部分（>100m）结构费用增加显著
- 消防系统要求更高，喷淋+报警系统增加50-80元/m²
- 新疆25层以上项目建议采用铝模+爬架体系，可节省模板费30%

### 1.4 别墅（2-3层）

| 费用项 | 独栋别墅 | 联排别墅 |
|-------|---------|---------|
| 土建工程 | 2,200-2,800元/m² | 1,800-2,200元/m² |
| 装饰工程 | 1,500-3,000元/m² | 800-1,500元/m² |
| 安装工程 | 500-700元/m² | 400-550元/m² |
| 措施费 | 200-300元/m² | 180-250元/m² |
| **合计** | **4,400-6,800元/m²** | **3,180-4,500元/m²** |

**新疆特殊说明**：
- 别墅项目在乌鲁木齐南山、水西沟、昌吉等区域较集中
- 采暖系统费用占总造价8-12%（壁挂炉或集中供暖）
- 地下室（如有）需额外增加1,500-2,500元/m²
- 屋面瓦（彩色水泥瓦或陶瓦）：120-200元/m²

## 九、使用说明

### 9.1 指标适用范围
- 投资估算阶段使用，误差率+-15%
- 初步设计概算阶段需结合设计图纸调整，误差率+-10%
- 施工图预算需按实际工程量计算

### 9.2 指标调整公式
```
调整后指标 = 基准指标 * 地州系数 * 时间系数 * 规模系数

- 地州系数：见上表
- 时间系数：以2025年1月为1.00，每季度调整+-2%
- 规模系数：建筑面积<5,000m²时*1.05-1.10，>50,000m²时*0.95-0.98
```

### 9.3 不含费用项
以下费用未包含在上述指标中，需单独计取：
- 土地费用（征地+拆迁）
- 前期费用（勘察、设计、监理、招标）
- 建设期贷款利息
- 预备费（基本+涨价）
- 建设管理费
- 专项评价费（环评、水保、安评等）
