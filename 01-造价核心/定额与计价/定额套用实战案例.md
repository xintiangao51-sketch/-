# 定额套用实战案例（10个真实场景）

> 基于新疆2024版定额及乌鲁木齐地区价格水平，涵盖土建、装饰、安装、市政全专业。

## 案例一：土方工程 —— 某小区基础开挖

### 工程场景
乌鲁木齐市某住宅小区，框剪结构12层，基坑开挖深度5.2m，三类土，基底面积1200m²，放坡系数1:0.67。地下水位-4.5m，需降水+排水。

### 套用定额子目
| 序号 | 定额编号 | 子目名称 | 工程量 | 单位 |
|-----|---------|---------|-------|------|
| 1 | 1-1-12 | 挖掘机挖土方（三类土） | 6,240 | m³ |
| 2 | 1-1-85 | 自卸汽车运土（运距5km以内） | 5,200 | m³ |
| 3 | 1-1-92 | 回填土方（人工+机械夯实） | 1,040 | m³ |
| 4 | 1-3-8 | 基底钎探 | 1,200 | m² |
| 5 | 1-3-15 | 基坑降水（井点降水，7天） | 1 | 项 |

### 换算调整说明
- 三类土不需调整（定额已综合）
- 挖土深度5.2m，按5m以内套用，超过部分按每增1m调整（1-1-18，+8%）
- 运距5km按定额1km+每增1km*4调整
- 回填土需夯实，调整为夯填子目（1-1-92）

### 综合单价计算过程
```
1. 挖掘机挖土方：6,240m³ * 8.56元/m³ = 53,414元
2. 挖深调整（5.2m按6m档）：53,414 * 1.08 = 57,687元
3. 自卸汽车运土：5,200m³ * 18.35元/m³ = 95,420元
   - 运距超出1km部分：5,200 * 3.85元/m³ * 4 = 80,080元
   - 合计：95,420 + 80,080 = 175,500元
4. 回填夯实：1,040m³ * 22.56元/m³ = 23,462元
5. 基底钎探：1,200m² * 3.85元/m² = 4,620元
6. 降水费用：1项 * 28,500元 = 28,500元

土方工程合计：57,687 + 175,500 + 23,462 + 4,620 + 28,500 = 289,769元
```

### 常见错误与避坑提示
| 错误类型 | 具体问题 | 后果 | 正确做法 |
|---------|---------|------|---------|
| 土方分类错误 | 将三类土按二类土套用 | 单价偏低15-20% | 根据地质报告确定土类别 |
| 放坡系数漏算 | 只算直壁开挖 | 工程量少算30-40% | 按规范放坡，加工作面 |
| 运距计算错误 | 只算挖方量未算外运量 | 运土费用缺失 | 挖方-回填=外运 |
| **新疆特有** | 未考虑冻土开挖 | 冬季施工无法执行 | 11月-3月需套用冻土开挖子目（单价*1.5） |
| 降水费用漏计 | 见水位未降水即开挖 | 边坡坍塌 | 地勘报告显示水位即须降水 |

## 案例二：混凝土工程 —— 办公楼框架柱

### 工程场景
乌鲁木齐某办公楼，框架结构8层，C40混凝土框架柱，截面600*600mm，层高4.2m，总柱混凝土量380m³。

### 套用定额子目
| 序号 | 定额编号 | 子目名称 | 工程量 | 单位 |
|-----|---------|---------|-------|------|
| 1 | 4-2-18 | 现浇混凝土框架柱（C40） | 380 | m³ |
| 2 | 4-2-18(换) | 混凝土标号换算（定额C30换C40） | 380 | m³ |

### 换算调整说明
- 定额按C30编制，实际C40需进行混凝土标号换算
- 换算法：定额混凝土含量*（C40价-C30价）
- C30信息价：385元/m³（乌鲁木齐2025年1月），C40信息价：425元/m³
- 价差：(425-385)*1.02(损耗系数)=40.8元/m³

### 综合单价计算过程
```
1. 定额子目4-2-18基价：856.30元/m³
2. 标号换算调整：1.02m³ * (425-385) = 40.8元/m³
3. 调整后基价：856.30 + 40.8 = 897.10元/m³
4. 模板费用（另计，4-3-28）：68.50元/m²接触面积
5. 柱模板接触面积：380m³ / 0.36m² * 4边 * 4.2m ≈ 17,733m²
   模板费用：17,733 * 65.80 = 1,166,831元

混凝土柱综合单价：897.10元/m³
模板综合单价：65.80元/m²接触面积
```

### 常见错误与避坑提示
| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 标号换算只换价不换量 | 混凝土含量不等于1m³ | 按定额含量1.015-1.025m³/m³换算 |
| 模板量按混凝土量估算 | 误差巨大 | 按接触面积计算 |
| 柱高超过3.6m漏计支撑费 | 超高费应按每超1m增加 | 套用超高增加费子目4-3-98 |
| **新疆冬季施工费** | 11-3月混凝土需冬施费 | 增加15-25元/m³（热水拌和+保温） |
| 泵送费漏计 | 商品混凝土需泵送 | 增加25-35元/m³ |

## 案例三：钢筋工程 —— 高层住宅剪力墙

### 工程场景
乌鲁木齐某高层住宅，剪力墙结构18层，HRB400E钢筋，抗震等级二级，总钢筋用量420t（含基础、墙、梁、板）。

### 套用定额子目
| 序号 | 定额编号 | 子目名称 | 工程量 | 单位 |
|-----|---------|---------|-------|------|
| 1 | 4-4-12 | 现浇构件钢筋制安（HRB400以内） | 420 | t |

### 换算调整说明
- 定额已综合考虑制作、安装、绑扎、焊接
- 钢筋主材按市场价调整：HRB400E乌鲁木齐2025年均价3,850元/t
- 定额钢筋含量1.03t/t（含损耗3%）

### 综合单价计算过程
```
1. 定额基价（不含主材）：685.30元/t
2. 钢筋主材价：1.03t * 3,850元/t = 3,965.50元/t
3. 综合基价：685.30 + 3,965.50 = 4,650.80元/t
4. 总价：420t * 4,650.80 = 1,953,336元
```

### 常见错误与避坑提示
| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 钢筋搭接和锚固未计 | 定额已含，但特殊设计需另算 | 常规不再增加 |
| 钢筋接头漏计 | 直螺纹套筒连接需单列 | 直径>=18mm套筒连接，约15-25元/个 |
| 钢筋损耗率3%未考虑 | 主材用量不足 | 按定额含量1.03计入 |
| **新疆运距价差** | 偏远地州钢筋价差大 | 和田比乌市高300-500元/t |
| 抗震钢筋（E级）未区分 | HRB400和HRB400E价差约200元/t | 按设计标注执行 |

## 案例四：砌体工程 —— 住宅填充墙

### 工程场景
昌吉市某住宅楼，框架结构，外墙采用300厚加气混凝土砌块，内墙200厚加气块，总砌体量560m³，M7.5水泥砂浆砌筑。

### 套用定额子目
| 序号 | 定额编号 | 子目名称 | 工程量 | 单位 |
|-----|---------|---------|-------|------|
| 1 | 3-2-15 | 加气混凝土砌块墙（300厚） | 320 | m³ |
| 2 | 3-2-16 | 加气混凝土砌块墙（200厚） | 240 | m³ |

### 换算调整说明
- 砌筑砂浆M7.5与定额M5有价差
- M7.5砂浆比M5高约15元/m³
- 300厚墙定额含量0.18m³砂浆/m³砌体，价差：0.18*(M7.5-M5)=2.7元/m³

### 综合单价计算过程
```
1. 3-2-15单价（300厚）：385.60元/m³
2. 砂浆标号调整：+2.7元/m³
3. 调整后单价：388.30元/m³
4. 300厚墙总价：320m³ * 388.30 = 124,256元
5. 200厚墙总价：240m³ * 375.80 = 90,192元
6. 砌体合计：214,448元
```

### 常见错误与避坑提示
| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 砌体模数与设计不符 | 加气块运至现场需切割 | 计入损耗3-5% |
| 构造柱/圈梁/过梁漏算 | 二次结构工程量约砌体量的15-20% | 单独列项 |
| 钢丝网防裂漏计 | 不同材料交接处需挂网 | 约8-12元/m²，总长约砌体面积20% |
| **新疆加气块质量** | 部分小厂加气块强度不达标 | 建议选用大厂品牌（天山、青松等） |
| 拉结筋植筋漏算 | 框架柱与填充墙拉结 | 每道墙2根6mm钢筋，约5元/根 |

## 案例五：防水工程 —— 地下室底板防水

### 工程场景
阿克苏市某商业综合体，地下室一层，底板面积3,200m²，采用SBS改性沥青防水卷材（II型，4mm厚），底板下+侧墙外防水。

### 套用定额子目
| 序号 | 定额编号 | 子目名称 | 工程量 | 单位 |
|-----|---------|---------|-------|------|
| 1 | 8-1-28 | SBS改性沥青防水卷材（热熔法，4mm） | 3,200 | m² |
| 2 | 8-1-28(换) | 侧墙防水（平面转立面） | 860 | m² |

### 换算调整说明
- 平面与立面防水人工费不同（立面+15%）
- 卷材搭接宽度：长边80mm，短边100mm，损耗率约15%
- 附加层按实际面积另计（阴角、阳角、管根等）

### 综合单价计算过程
```
1. 平面SBS防水基价：42.80元/m²
2. 立面调整：42.80 * 1.15 = 49.22元/m²
3. 平面总价：3,200m² * 42.80 = 136,960元
4. 立面总价：860m² * 49.22 = 42,329元
5. 附加层（按防水面积15%计）：(3,200+860)*0.15*52.60 = 32,041元
6. 防水合计：136,960 + 42,329 + 32,041 = 211,330元
```

### 常见错误与避坑提示
| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 附加层漏算 | 管根、阴阳角、施工缝 | 按实计算，约防水面积10-15% |
| 搭接宽度不足 | 短边搭接<100mm | 严格执行规范 |
| 平面立面未区分 | 统一套用平面价格 | 立面人工费*1.15 |
| **新疆低温施工** | 低于-10℃热熔法质量差 | 冬季改用自粘卷材或热风焊接 |
| 基层处理漏算 | 找平层+冷底子油 | 增加约8-12元/m² |

## 案例六：保温工程 —— 外墙外保温

### 工程场景
伊宁市某住宅小区，12栋11层住宅，外墙保温面积28,500m²，采用100mm厚EPS板（B1级）+抹面砂浆+耐碱玻纤网格布+涂料饰面。

### 套用定额子目
| 序号 | 定额编号 | 子目名称 | 工程量 | 单位 |
|-----|---------|---------|-------|------|
| 1 | 9-2-8 | EPS板外墙外保温（粘贴+锚固，50mm） | 28,500 | m² |
| 2 | 9-2-9 | 每增减5mm | 28,500 | m² |
| 3 | 9-2-42 | 耐碱玻纤网格布（标准型） | 28,500 | m² |
| 4 | 9-2-56 | 抹面砂浆（5mm） | 28,500 | m² |

### 换算调整说明
- 定额按50mm编制，实际100mm需增补50mm
- EPS板厚度调整：每增5mm套用9-2-9子目，需10次
- 锚固件数量：新疆大风区需加密至8个/m²（内地6个/m²）

### 综合单价计算过程
```
1. 50mmEPS保温基价：58.60元/m²
2. 厚度调整（增加50mm=10*5mm）：10*6.80元/m² = 68.00元/m²
3. 网格布：15.80元/m²
4. 抹面砂浆：28.50元/m²
5. 锚固件加密增加费：2个/m² * 0.85元/个 = 1.70元/m²
6. 综合单价：58.60 + 68.00 + 15.80 + 28.50 + 1.70 = 172.60元/m²
7. 保温工程总价：28,500m² * 172.60 = 4,919,100元
```

### 常见错误与避坑提示
| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 厚度调整只计一次 | 应分段逐级累加 | 每5mm递进，精确计算 |
| 锚固件数量不足 | 新疆大风区需加密 | 8个/m²，边角区域10个/m² |
| EPS板等级不符 | 普通B2级不满足消防 | 新疆要求B1级及以上 |
| 窗侧/阳台侧边漏算 | 侧面保温面积不小 | 按展开面积计算 |
| **新疆特有** | EPS板容重需>=20kg/m³（内地18kg/m³） | 材料验收严格控制 |

## 案例九：市政工程 —— 沥青道路

### 换算调整说明
- 沥青混凝土主材按乌鲁木齐市场价：AC-13为420元/t，AC-20为385元/t
- 路面宽度按实计算，不扣除井盖面积
- 水泥稳定碎石按5%水泥含量调整

### 综合单价计算过程
```
1. 级配碎石基层（20cm）：26,400m² * 28.50元/m² = 752,400元
2. 水泥稳定碎石（18cm，5%）：26,400m² * 42.80元/m² = 1,129,920元
3. AC-20下面层（6cm）：
   - 面积26,400m²，厚度6cm，密度2.45t/m³
   - 沥青量：26,400 * 0.06 * 2.45 = 3,881t
   - 单价：385元/t + 摊铺费35元/t = 420元/t
   - 总价：3,881 * 420 = 1,630,020元
4. AC-13上面层（4cm）：
   - 沥青量：26,400 * 0.04 * 2.45 = 2,587t
   - 单价：420元/t + 摊铺费35元/t = 455元/t
   - 总价：2,587 * 455 = 1,177,085元
5. 透层+粘层：26,400m² * 6.50元/m² = 171,600元
6. 路缘石安装（两侧）：2,400m * 85元/m = 204,000元

道路工程合计（路面部分）：5,065,025元
道路单方造价：5,065,025 / 26,400 = 191.86元/m²（路面面积）
道路每延米造价：5,065,025 / 1,200 = 4,221元/m（含两侧）
```

### 常见错误与避坑提示
| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 沥青用量计算错误 | 未按密度换算面积→重量 | 必须用密度和厚度转换 |
| 水泥稳定碎石水泥剂量 | 5%为质量比非体积比 | 注意配合比类型 |
| 检查井周边处理漏算 | 需增加加固措施 | 每座检查井约200元加固费 |
| **新疆高温车辙** | 表面层需用改性沥青 | AC-13改用SBS改性沥青，单价+150元/t |
| 路基压实度检测 | 需第三方检测 | 增加试验费约3-5元/m² |

## 案例十：措施项目 —— 某高层住宅综合措施费

### 工程场景
喀什市某住宅小区，3栋18层剪力墙结构，总建筑面积28,560m²，檐高54m。需计算全部措施项目费用。

### 套用定额子目
| 序号 | 定额编号 | 子目名称 | 工程量 | 单位 |
|-----|---------|---------|-------|------|
| 1 | 17-1-12 | 综合脚手架（钢管，檐高60m以内） | 28,560 | m² |
| 2 | 17-2-8 | 垂直运输（塔吊，檐高60m以内） | 28,560 | m² |
| 3 | 17-3-5 | 模板工程（综合） | 28,560 | m² |
| 4 | 17-4-12 | 冬雨季施工增加费 | 28,560 | m² |
| 5 | 17-4-28 | 临时设施费 | 28,560 | m² |

### 换算调整说明
- 喀什地区抗震设防烈度8度，模板支撑需加强
- 冬雨季施工费：喀什冬期4个月（11月-2月），按直接费的3.5%计
- 临时设施费按直接费的2.0%计

### 综合单价计算过程
```
1. 综合脚手架（钢管）：28,560m² * 28.50元/m² = 813,960元
2. 垂直运输（塔吊）：28,560m² * 38.60元/m² = 1,102,416元
3. 模板工程：28,560m² * 68.50元/m² = 1,956,360元
4. 冬雨季施工增加费：
   直接费合计（估算）= 12,500,000元（含人材机）
   冬施费：12,500,000 * 3.5% = 437,500元
5. 临时设施费：12,500,000 * 2.0% = 250,000元
6. 安全文明施工费：12,500,000 * 3.8% = 475,000元
7. 已完工程保护费：28,560m² * 5.80元/m² = 165,648元

措施项目合计：5,200,884元
措施费单方：5,200,884 / 28,560 = 182.13元/m²
措施费占直接费比例：5,200,884 / 12,500,000 = 41.6%
```

### 常见错误与避坑提示
| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 脚手架按建筑面积包干 | 实际需按外墙周长+高度计算 | 综合脚手架按建筑面积已含 |
| 模板只算一次 | 高层住宅标准层模板可周转 | 按接触面积计算，考虑周转次数 |
| 冬施费比例不足 | 新疆冬施费比内地高1-2倍 | 按实际冬季施工时间比例计取 |
| **喀什抗震加强** | 8度设防，模板支撑需加强 | 支撑体系费用增加15-20% |
| 塔吊基础及附着漏算 | 塔吊基础+附着预埋件 | 增加约8-12万元/台 |

## 附：定额套用八大通用避坑原则

1. **先看定额说明再套用**：每个章节前面都有适用范围和说明，忽略则可能套错子目
2. **主材与辅材区分**：定额基价是否含主材、主材是否另计，必须确认
3. **工程量的计量单位**：m³/m²/m/t/个不能混淆，换算是错误高发区
4. **换算调整必须在编制说明中注明**：以便审核人员复核
5. **新疆地州系数不能忘**：各地州人工、机械调整系数差异可达15%
6. **冬季施工费非全工期计取**：按实际冬季施工天数比例分段计算
7. **措施费不可漏项**：脚手架、模板、垂直运输、冬施、临设、安文六大件
8. **税金和规费按规定计取**：新疆增值税9%，附加税0.5%
