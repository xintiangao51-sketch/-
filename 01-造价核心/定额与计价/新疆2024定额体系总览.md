## 二、计价体系说明

### 2.1 计价模式

新疆建设工程计价采用**"定额量、市场价、竞争费"**的模式：

| 要素 | 定价方式 | 说明 |
|---|---|---|
| **定额量** | 政府规定 | 消耗量按定额执行，不得随意调整 |
| **市场价** | 市场竞争 | 人工、材料、机械单价按市场或信息价 |
| **竞争费** | 企业自主 | 管理费、利润可竞争，规费税金不可竞争 |

### 2.2 计价程序

#### 工料单价法计价程序
```
1. 直接工程费 = Σ(工程量 × 定额基价)
2. 措施费 = 直接工程费 × 措施费率 或 按实计算
3. 直接费 = 直接工程费 + 措施费
4. 间接费 = 直接费 × 间接费率 或 (人工费+机械费) × 费率
5. 利润 = (直接费 + 间接费) × 利润率 或 (人工费+机械费) × 利润率
6. 税金 = (直接费 + 间接费 + 利润) × 税率
7. 工程造价 = 直接费 + 间接费 + 利润 + 税金
```

#### 综合单价法计价程序（GB50500-2013清单）
```
综合单价 = 人工费 + 材料费 + 机械费 + 管理费 + 利润 + 风险费用
    ↓
分部分项工程费 = Σ(工程量 × 综合单价)
措施项目费 = 按定额计算或按项计算
其他项目费 = 暂列金额 + 暂估价 + 计日工 + 总承包服务费
规费 = (分部分项 + 措施 + 其他)中的人工费+机械费 × 规费费率
税金 = (分部分项 + 措施 + 其他 + 规费) × 税率
    ↓
工程造价 = 分部分项 + 措施 + 其他 + 规费 + 税金
```

## 三、取费标准框架

### 3.1 取费基数说明

| 专业工程 | 取费基数 | 说明 |
|---|---|---|
| 建筑工程 | 人工费 + 机械费 | 装饰工程同 |
| 安装工程 | 人工费 | 仅以人工费为基数 |
| 市政工程 | 人工费 + 机械费 | 道路、桥梁、管网同 |
| 园林绿化 | 人工费 + 机械费 | |

### 3.2 费率区间参考（2024版）

| 费用项目 | 建筑工程 | 装饰工程 | 安装工程 | 市政工程 | 备注 |
|---|---|---|---|---|---|
| **管理费** | 18%~28% | 20%~30% | 25%~35% | 16%~25% | 以人工+机械为基数 |
| **利润** | 10%~18% | 12%~20% | 15%~25% | 8%~15% | 以人工+机械为基数 |
| **规费** | 18%~22% | 18%~22% | 20%~25% | 18%~22% | 含社保、公积金、工伤 |

### 4.3 人工单价调整规则

- 人工调差只计取税金，不计取管理费、利润、规费
- 调差部分 = (发布人工单价 - 定额人工单价) × 定额工日数量
- 人工单价原则上每半年调整一次

## 六、定额总说明要点摘录

### 6.1 消耗量调整原则

1. 定额消耗量是编制招标控制价、投标报价、结算审核的依据
2. 投标报价时，企业可根据自身技术水平和管理能力调整消耗量
3. 定额中注有"×××以内"或"×××以下"者，均包括×××本身
4. 定额中注有"×××以外"或"×××以上"者，均不包括×××本身

### 6.2 材料消耗量

1. 定额材料消耗量包括主要材料、辅助材料、零星材料
2. 材料损耗率已包含在定额消耗量中
3. 周转性材料按摊销量计算
4. 混凝土、砂浆等半成品材料消耗量按定额配合比计算

### 6.3 机械消耗量

1. 机械台班消耗量按常用机械类型、规格综合取定
2. 机械台班单价包含折旧费、大修理费、经常修理费、安拆费及场外运费、人工费、燃料动力费、税费
3. 大型机械进出场费单独计算
