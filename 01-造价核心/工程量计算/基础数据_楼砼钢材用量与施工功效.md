# 基础数据：楼砼钢材用量与施工功效

> 来源于网络百科整理的常用估算经验数据，**非新疆定额口径**。仅用于投资估算/快速匡算，结算与清单计价必须以现行定额及图纸为准。⚠ 各值为内地经验口径，新疆工程需结合地区系数调整并核实。

## 一、各类住宅单位面积钢筋/混凝土用量

| 建筑类型 | 钢筋用量 | 混凝土用量 | 备注 |
|---------|--------|----------|------|
| 多层砌体住宅 | 30 kg/㎡ | 0.30-0.33 m³/㎡ | 抗震7度区规则结构 |
| 多层框架 | 38-42 kg/㎡ | 0.33-0.35 m³/㎡ | |
| 小高层11-12层 | 50-52 kg/㎡ | 0.35 m³/㎡ | |
| 高层17-18层 | 54-60 kg/㎡ | 0.36 m³/㎡ | |
| 高层30层(H=94m) | 65-75 kg/㎡ | 0.42-0.47 m³/㎡ | |
| 高层酒店式公寓28层(H=90m) | 65-70 kg/㎡ | 0.38-0.42 m³/㎡ | |
| 别墅 | 介于多层砌体与小高层之间 | 同左 | |

> 上述数据按抗震7度区规则结构设计；新疆多数地区抗震设防烈度≥7度，北疆部分地区8度，需按实际抗震设防调整。

## 二、施工预算经济指标（多层住宅）

| 指标 | 经验取值 |
|------|----------|
| 室外门窗面积占建筑面积 | 0.20-0.24 |
| 模板面积占建筑面积 | 约2.2 |
| 室外抹灰面积占建筑面积 | 约0.4 |
| 室内抹灰面积占建筑面积 | 约3.8 |

> 注：室外门窗不含单元门、防盗门。该指标为人工匡算用，与 [[01-造价核心/定额与计价/各专业工程造价指标参考]] 的新疆造价指标口径不同，二者不可混用。

## 三、建筑工人施工功效（经验值）

| 工种 | 日功效 |
|------|--------|
| 抹灰工 | 35 ㎡/天 |
| 砖工（红砖） | 1000-1800 块/天 |
| 砖工（空心砖） | 800-1000 块/天 |
| 瓷砖 | 15 ㎡/天 |
| 刮大白 | 第一遍300㎡/天，第二遍180㎡/天，第三遍压光90㎡/天 |

> ⚠ 施工人功效受季节、工艺、管理水平影响大，且未含新疆冬施降效。冬施期间人工/机械效率显著下降，需按冬施费规定另行调整。

## 待核实
- ⚠ 钢筋/混凝土单方用量为内地7度区经验值，新疆具体项目需按图纸配筋及抗震设防核实
- ⚠ 施工功效未区分季节与地区差异，新疆冬施需单独测算
- ⚠ 建材基础数据为通用值，取样口径与新疆地方材料标准差异待核实
