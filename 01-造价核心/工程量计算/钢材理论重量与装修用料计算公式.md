# 钢材理论重量与装修用料计算公式

> 常用钢材理论重量估算公式及装修材料用量测算公式。属经验速算口径，精度有限；正式算量应以国标理重表（如 GB/T 702、GB/T 9787 等）和图纸净用量为准。

## 一、钢材每米理论重量（经验公式）

| 类型 | 每米重量公式（kg/m） |
|------|---------------------|
| 角钢（等边） | 0.00785×(边宽+边宽-边厚)×边厚 |
| 等边角钢（另式） | 边宽×边厚×0.015 |
| 不等边角钢 | 0.00785×边厚×(长边宽+短边宽-边厚) |
| 圆钢/螺纹钢 | 0.00617×直径×直径 |
| 方钢 | 0.00786×边宽×边宽 |
| 六角钢 | 0.0068×对边直径×对边直径 |
| 八角钢 | 0.0065×直径×直径 |
| 扁钢 | 0.00785×厚度×边宽 |
| 无缝钢管/电焊钢管 | 0.02466×壁厚×(外径-壁厚) |
| 方管 | (边长+边长)×2×厚×0.00785 |
| 钢板/板材 | 每㎡=7.85×厚度 (kg/㎡) |
| 工字钢 | 0.00785×腰厚×[高+f(腿宽-腰厚)] |
| 槽钢 | 0.00785×腰厚×[高+e(腿宽-腰厚)] |

> 式中 f、e 为截面系数，需查型钢表取值。简化估算时常用查表法替代。

## 三、装修材料用量测算公式

### 3.1 地面砖用量
每百米用量 = 100 / [(块料长+灰缝宽)×(块料宽+灰缝宽)] × (1+损耗率)

损耗率一般 1-5%。例：0.5×0.5m 复古地砖，拼缝0.002m，损耗1%，100㎡用量 ≈ 393块。
地砖总价 = 地砖块数 × 单价

### 3.2 顶棚（吊顶）用量
顶棚板用量 = (长-屏蔽长)×(宽-屏蔽宽)
屏蔽尺寸为扣除的边缘遮挡量。

### 3.3 包门用量
包门材料用量 = 门外框长 × 门外框宽

### 3.4 壁纸用量
壁纸用量 = [(高-屏蔽长)×(宽-屏蔽宽)]×壁数 - 门面积 - 窗面积

### 3.5 实木地板用量
- 纵向用量 = 房间长度 / 地板长度
- 横向用量 = 房间宽度 / 地板宽度
- 除不尽用进位法（不可四舍五入）；纵向不到半块算半块，超过半块算一块
- 地板损耗 = 地板面积 - 住房面积；损耗率 = 损耗/住房面积（一般≤5%）

### 3.6 贴墙材料用量
精确公式：S = (L/M + 1)(H + h) + C/M
- S=所需长度(m)，L=扣门窗后四壁总长(m)，M=材料宽(m)
- H=所需高度(m)，h=图案重复距离(m)，C=门窗上下所需面积(㎡)
- 或粗算：房间面积×2.5 ≈ 贴墙用料数
- 实地测量法更精确：按材料宽度数墙周长幅数 × 高度

### 3.7 涂料用量估算法
涂料公斤数 = 房间面积(㎡)÷4 + 墙壁高度(dm)÷4
例：20㎡，墙高16dm → 20÷4 + 16÷4 = 9 公斤（粉刷两遍）

## 待核实
- ⚠ 型钢理论重量公式中的 f/e 系数需查国标型钢表确认，不可凭公式直接用于结算
- ⚠ 装修用料损耗率为经验值，实际损耗受材料规格、排版、工艺影响
- ⚠ 有色金属比重与国标取值差异待核对
