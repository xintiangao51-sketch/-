## 常见项目类型费率表（竖版）

### 房建项目（框架/剪力墙）

```
综合单价构成（示例：C30 混凝土梁）：
  人工费：120 元/m³
  材料费：320 元/m³（C30 商品混凝土 324 元/m³ 含税）
  机械费：15 元/m³
  → 小计：455 元/m³
  企业管理费：(120+15) × 15.08% = 20.4 元/m³（按上限计取）
  利润：(120+15) × 18% ≈ 24.3 元/m³
  → 综合单价 ≈ 500 元/m³（不含增值税）
```

### 安装项目（电气/水暖）

```
综合单价构成（示例：配电箱安装）：
  人工费：180 元/台
  材料费：80 元/台
  机械费：15 元/台
  → 小计：275 元/台
  企业管理费：180 × 16.87% = 30.4 元/台（安装专业按纯人工）
  利润：180 × 22% = 39.6 元/台
  → 综合单价 ≈ 345 元/台
```

## 公式速查

```
管理费 = 计费基数 × 管理费费率%
利润   = 计费基数 × 利润率%
规费   = 人工费 × 规费率%
增值税 = (分部分项+措施+其他-可抵扣进项) × 9%
总造价 = 分部分项 + 措施 + 其他 + 规费 + 税金
```
