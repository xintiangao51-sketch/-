# 新疆造价公开依据清单 · 拓展应用指南

## 场景一：接到新项目报价请求  🔴 必执行

### 第一步：地基锁定（输出"项目地基卡"）

```markdown
| 字段 | 内容 |
|------|------|
| 项目名称 | [填写] |
| 所在地州 | [填写，如：乌鲁木齐市/巴州/伊犁州] |
| 工程类别 | [房建/市政/安装/公路/水利] |
| 计价口径 | [招标控制价/投标报价/结算/审价] |
| 计价标准版本 | [GB/T 50500-2024 / GB 50500-2013] |
| 合同日期 | [YYYY-MM-DD] |
| 计价基准期 | [约定采用的造价信息期号] |
```

### 第二步：定额覆盖检查

| 检查项 | 查什么 | 参考卡片 |
|--------|--------|----------|
| 主定额是否覆盖？ | 房建→2020版房建定额 / 市政→2020版市政定额 / 安装→2020版安装补充+国标2015 | [[10-索引与导航/知识库总索引#KC-104]] |
| 有无补充定额可用？ | 装配式？→装配式试行 / 机器人施工？→智能建造试行 / 排水养护？→排水养护定额 | [[10-索引与导航/知识库总索引#KC-106]] |
| 地州有无本地估价表？ | 巴州/伊犁州/克州/博州/沙湾市见已知口径；其余地州须查住建局 | [[10-索引与导航/知识库总索引#KC-105]] |

### 第三步：人材机查询路径选择

```
项目所在地州有月度信息价？
  ├─ 是 → 以地州信息价为第一优先
  └─ 否 → 取乌鲁木齐市同期信息价 + 合同约定调价条款
```

参考：[[10-索引与导航/知识库总索引#KC-286]]

### 第四步：费率核取

每个费率百分比绑定来源：`[费率名称] = [X%]，来源：[文号/合同条款]，核实日期：[YYYY-MM-DD]`

参考：[[10-索引与导航/知识库总索引#KC-227]]

### 第五步：输出报价说明

报价说明中必须包含：
1. 计价依据（定额版本 + 地州通告文号）
2. 人工单价来源（哪个通告，第几期）
3. 材料价格基准期
4. 取费文件（文号 + 发布日期）
5. ⚠ 待核实项清单（如有）

## 场景二：合同审查时溯源取费  🟡 建议执行

### 检查清单

- [ ] 合同中报价依据是否引用具体文号？（不能只写"按新疆现行定额"）
- [ ] 社会保障费/意外伤害保险/排污费是否单独列项？（造价管理办法强制要求）
- [ ] 人工单价是否标注来源通告？
- [ ] 是否有"价格调整"条款？调整机制是否绑定具体信息价期号？
- [ ] 计价标准版本：合同签订日在 2025-09-01 之后，是否约定 GB/T 50500-2024？

参考：[[10-索引与导航/知识库总索引#KC-107]] + [[10-索引与导航/知识库总索引#KC-227]]

## 场景三：结算审价时验证定额套用  🟡 建议执行

### 审价快速对照表

| 被审文件中的表述 | 应核查 | 参考 |
|------------------|--------|------|
| "按 2024 版定额" | ⚠ 2024 版三本主定额不存在→核实实际套用版本 | KC-104 |
| "按自治区定额" | 项目所在地州有无本地估价表？→有则须按地州 | KC-105 |
| 定额套用装配式子目 | 项目是否确实使用装配式工艺？→否则不可套用试行定额 | KC-106 |
| 费率固定百分比 | 百分比是否绑定具体文件？→无绑定则 ⚠ 待核实 | KC-227 |
| 人工单价取整 | 是否引用了最近一期人工单价通告？→没有则 ⚠ 待核实 | KC-286 |

## 场景四：投标时验证招标控制价合理性  🟡 建议执行

### 反查路径

1. **拆解控制价** → 提取所用定额子目列表
2. **比对本清单** → 逐项确认能否在公开来源中找到对应依据
3. **差异标记**：
   - 控制价有、公开依据无 → 可能使用内部估价表，需澄清
   - 控制价无、公开依据有 → 可能漏项，需质疑
   - 费率不一致 → 检查是否为地州特殊取费
4. **输出差异报告**：`控制价 vs 公开依据 · 差异分析表`
