# 工程造价基础认知概览

工程造价的两种含义、基本职能、计价内容环节、审核方法与管理的速览。属概念框架，作为新人入门与术语统一的锚点。

## 一、工程造价两种含义
| 含义 | 视角 | 内容 |
|------|------|------|
| 第一种含义 | 投资者/业主 | 建设某工程花费的全部费用（固定资产再生产+无形资产+铺底流动资金的一次性费用总和） |
| 第二种含义 | 承包商/市场 | 工程价格，即工程承发包交易行为的承包价/合同价 |

## 二、基本职能
| 职能 | 说明 |
|------|------|
| 评价职能 | 评价投资效益与经济合理性 |
| 调控职能 | 调节投资规模、结构 |
| 预测职能 | 预测投资额与成本 |
| 控制职能 | 控制投资、纠偏 |

## 三、造价计价内容（按建设阶段）
| 阶段 | 计价文件 |
|------|---------|
| 项目建议书/可研 | 投资估算 |
| 初步设计 | 设计概算 |
| 技术设计 | 修正概算 |
| 施工图设计 | 施工图预算 |
| 工程施工 | 工程结算（进度结算） |
| 竣工 | 竣工决算 |

> 计价的特征由基本特点（大额性、个别性/差异性、动态性、层次性、兼容性）决定。

## 四、职业要求
工民建/土木/建筑经济/工程造价/预算等专业大专以上；熟悉识图、结构构造、材料设备、定额与取费标准、工程量计算规则、经济法规与合同；能独立完成估、概、预、结算；并具沟通协调、执行能力。

## 五、预算审核内容
- 基本信息：核对工程概况、取费类别、税率等基本数据
- **工程量审核**：图算与预算工程量对比、计算口径、计量单位、重复/漏项
- **套用单价审核**：定额子目是否正确、换算/补充定额是否合理、错套高套
- **费用审核**：取费基础与费率、材差、税费计算程序

## 六、五种审核方法
| 方法 | 要点 |
|------|------|
| 全面审核法 | 按图纸+定额逐项全面算，精度高耗时大，用于审核周期长的项目 |
| 重点审核法 | 抓造价大、易错子目重点核，工作量小效果好 |
| 对比审核法 | 单方造价/分部比例/专业比例/工料消耗指标横向对比，发现异常 |
| 分组计算 | 利用同组相同/相近基数，审一个判其他，速度快 |
| 筛选法 | 单方工程量/价格/用工基本值表对照，标准不同则调整 |

## 七、造价管理（两层含义）
- 建设工程投资费用管理：预测、计算、确定、监控造价及其变动
- 工程价格管理：微观（企业成本控制/计价/竞价）+宏观（政府法律/经济/行政手段调控市场）
