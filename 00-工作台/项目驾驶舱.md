## 六、新疆项目特有的管控要点

### 6.1 冬季施工管控要点

| 管控事项 | 具体要求 | 依据文件 |
|---|---|---|
| 冬季施工起止日期 | 11月15日至次年3月15日（北疆）/ 11月1日至次年3月31日（阿勒泰等高寒地区） | 新疆2024定额说明 |
| 冬季施工措施费 | 按定额人工费+机械费的__%计取（分地区不同） | 详见常用参数看板 |
| 混凝土施工要求 | 入模温度不低于5℃，防冻剂用量须报审 | GB50666-2011 |
| 冬季停工提前通知 | 每年9月底前确定当年冬季施工方案 | 项目管理惯例 |
| 越冬维护 | 停工项目须编制越冬维护方案并报验 | 项目管理惯例 |

### 6.2 新疆特殊材差管控

| 材料类别 | 管控要点 | 风险提示 |
|---|---|---|
| 砂石料 | 运输距离50km以上需签认运距，环保整治期价格上涨 | 部分地州禁采期价格翻倍 |
| 保温材料（XPS/EPS） | 新疆节能标准75%要求，保温层厚度较内地大 | 厚度增加导致辅材全面增加 |
| 抗冻混凝土 | 所有外露混凝土构件需使用抗冻混凝土 | 等级按F150-F300不等 |
| 塑钢/断桥铝窗 | 新疆标准要求三玻两腔或四玻三腔 | 乌鲁木齐已执行四玻三腔 |
| 供热管材 | 地暖管PE-RT / 暖气片供暖用钢制散热器 | 高寒地区管材等级要求高 |

### 6.3 人工费波动管控

| 管控要点 | 具体措施 | 备注 |
|---|---|---|
| 农忙季节（5-6月） | 提前1个月锁定劳务队伍，签订保供协议 | 新疆农忙影响较大 |
| 古尔邦节/肉孜节 | 少数民族工人休假7-10天，提前安排 | 每年日期不固定 |
| 冬休前抢工 | 9-10月人工费上涨15-25%，提前锁定 | 劳务市场供需决定 |
| 跨地州用工 | 南疆到北疆务工人员需考虑交通和住宿 | 工效低于本地工 |

### 6.4 地区差异管控

| 地区 | 造价特点 | 管控重点 |
|---|---|---|
| 乌鲁木齐 | 信息价全面，市场竞争充分 | 重点关注人工费、商品混凝土价 |
| 昌吉/石河子 | 材料价略低于乌市，运费较低 | 确保使用当地信息价 |
| 伊犁/博乐 | 水泥、砂石资源丰富，价格较低 | 关注运费和运输距离 |
| 喀什/和田 | 材料价高15-20%，运距远 | 重点审核运距签证 |
| 阿勒泰/塔城 | 冬季超长，冬季施费比重高 | 提前规划冬季施工方案 |
| 吐鲁番/哈密 | 高温施工增加费，防风要求高 | 关注夏季高温补贴 |
| 克拉玛依/独山子 | 工业项目多，安装工程造价高 | 关注甲供材扣回 |

### 7.2 模板使用指引

1. **复制模板：** 每周新建一个副本，按实际数据填写
2. **优先级标注：** 使用 P0/P1/P2/P3 四级，红色标注高风险项
3. **风险标识：** 🟢正常 🟡关注 🔴预警，见红即报
4. **数据来源：** 各项目台账、计量报告、采购报表
5. **归档要求：** 周报按 `YYYY-WW-项目名称-周报` 格式命名存档
