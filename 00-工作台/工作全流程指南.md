# 工作全流程指南 —— 从收图到结算的完整操作流程

> 版本：v1.0 | 更新日期：2026-06-19 | 适用：新疆地区工程造价全过程管理

---

## 概述

本指南覆盖工程造价工作从接收到施工图纸到最终结算归档的全流程。每个阶段包含操作要点、常见错误避坑、新疆特殊注意事项。适用于房屋建筑、市政、装饰装修等各类型项目。

**整体流程概览：**

```
收图 → 识图 → 提量 → 清单编制 → 组价 → 调价策略 → 对量审核 → 结算编制
```

---

## 第一阶段：收图→识图→提量

### 1.1 操作要点

| 序号 | 步骤 | 操作内容 | 成果文件 |
|---|---|---|---|
| 1 | 收图确认 | 核对图纸目录、版本、日期、签章，签收图纸移交单 | 图纸接收记录 |
| 2 | 图纸清点 | 建筑图、结构图、水暖电图、装饰图逐张清点，确认完整 | 图纸清点清单 |
| 3 | 图纸审查 | 发现图纸矛盾、尺寸冲突、标高不一致等问题，形成答疑清单 | 图纸疑问清单 |
| 4 | 建立模型 | 广联达/新点建模，按楼层、分区建立 | 算量模型文件 |
| 5 | 土建提量 | 混凝土、钢筋、模板、砌体、抹灰逐项提取 | 工程量计算书 |
| 6 | 装饰提量 | 楼地面、墙面、天棚、门窗逐项提取 | 装饰工程量表 |
| 7 | 安装提量 | 给排水、电气、暖通逐系统提取 | 安装工程量表 |
| 8 | 自核互校 | 自检+互检，提量人与复核人签字 | 工程量复核记录 |

### 1.2 常见坑与避坑

| 易错点 | 错误表现 | 正确做法 |
|---|---|---|
| 版本混淆 | 用了旧版图纸提量 | 提量前确认图纸版次表，以最新版为准，旧版标注作废 |
| 漏提构造柱/圈梁 | 只算了主体未算二次结构 | 建模时按规范要求设置构造柱，马牙槎模板单独计算 |
| 钢筋搭接漏算 | 只算净长未算搭接 | 按规范长度统计搭接，直径≥22mm采用机械连接 |
| 装饰做法理解偏差 | 按照图纸说明做法提量错误 | 逐层核对建筑做法表，一个做法对应一个清单项 |
| 土方工作面/放坡系数用错 | 土方量偏差大 | 按新疆定额规定的工作面和放坡起点深度执行 |

### 1.3 新疆特殊注意事项

| 序号 | 注意事项 | 说明 |
|---|---|---|
| 1 | 冻土深度 | 新疆各地冻土深度不同，基础埋深须考虑冻土线（乌鲁木齐1.2m，阿勒泰1.5m） |
| 2 | 风荷载取值 | 新疆大风区较多（吐鲁番、百里风区），需按风荷载规范调整 |
| 3 | 砌体材料 | 新疆普遍使用加气混凝土砌块和空心砖，实心粘土砖已禁用 |
| 4 | 节能标准 | 75%节能标准，保温层厚度较内地大30-50%，建模注意 |
| 5 | 抗渗抗冻混凝土 | 基础底板、外露构件均需抗渗抗冻，等级按设计要求 |
| 6 | 当地材料选用 | 砂石料源运距、块石料场位置影响土方外运和回填方案 |

## 第二阶段：清单编制阶段

### 2.1 操作要点

| 序号 | 步骤 | 操作内容 | 成果文件 |
|---|---|---|---|
| 1 | 列清单项 | 按GB50500-2013清单计价规范，结合设计图纸逐项列出 | 工程量清单初稿 |
| 2 | 项目特征描述 | 每个清单项详细描述材质、规格、厚度、强度等级等 | 清单项目特征表 |
| 3 | 计量单位确认 | 按规范用m²、m³、t、m、项等标准单位 | — |
| 4 | 工程量填入 | 将提量成果填入清单对应项 | 清单工程量表 |
| 5 | 措施清单编制 | 组织措施费+技术措施费，脚手架、模板、垂直运输等 | 措施项目清单 |
| 6 | 其他项目清单 | 暂列金额、暂估价、计日工、总承包服务费 | 其他项目清单 |
| 7 | 自检互查 | 逐项核对清单完整性，填写清单自检表 | 清单编制检查记录 |

### 2.2 列项技巧

| 技巧 | 说明 |
|---|---|
| 分层列项 | 按楼层/分区组织清单，便于施工阶段进度款拆分 |
| 主次分明 | 主要项目（钢筋、混凝土、砌体）详细列，零星项目合并 |
| 特征简准 | 项目特征描述既要准确又不过于冗长，关键参数必写 |
| 参考类似项目 | 前期类似项目清单可作为参考，但需核实差异 |
| 暂估价设置 | 大宗材料和专业分包工程设置暂估价，后期认价 |

### 2.3 漏项检查清单

| 检查项目 | 常见漏项 | 检查方法 |
|---|---|---|
| 土方工程 | 边坡支护、排水降水、地基处理 | 查看基础图+地勘报告 |
| 砌体工程 | 构造柱、圈梁、过梁、砌体加筋 | 查看结构设计说明 |
| 混凝土工程 | 后浇带、设备基础、排水沟 | 查看建筑+结构图 |
| 钢筋工程 | 砌体拉结筋、预埋件、抗震构造 | 查看结构设计总说明 |
| 屋面工程 | 排气孔、雨水口、泛水收边 | 查看屋面节点详图 |
| 装饰工程 | 滴水线、防滑条、变形缝 | 查看建筑节点图 |
| 安装工程 | 管道支架、保温防腐、调试费 | 查看安装系统图+设计说明 |
| 措施项目 | 季节性施工、成品保护、二次搬运 | 根据施工组织设计 |

### 2.4 新疆特殊注意事项

| 序号 | 注意事项 | 说明 |
|---|---|---|
| 1 | 新疆2024定额清单说明 | 清单编制按新计价规范(2024版)，与国标GB50500有差异处优先执行新疆规定 |
| 2 | 地区类别划分 | 分一类地区（乌市、克拉玛依）和二类地区（其他地州），费率有差异 |
| 3 | 清单列项附加 | 新疆要求单独列出"冬季施工措施费""高原降效费"等特殊措施项 |
| 4 | 材料暂估价 | 新疆信息价未覆盖的材料需设置暂估价，后期按认价调整 |
| 5 | 定额换算说明 | 新疆定额与国标定额换算规则不同，套用时注意换算条件 |

## 第三阶段：组价阶段

### 3.1 操作要点

| 序号 | 步骤 | 操作内容 | 成果文件 |
|---|---|---|---|
| 1 | 定额套用 | 根据清单项目特征选择对应的2024新疆定额子目 | 预算书初稿 |
| 2 | 材料价导入 | 导入当期新疆建设工程信息价，补充市场询价 | 材料价格表 |
| 3 | 人工费调整 | 按各地州当期人工单价调整人工费（人工调增计取税金） | 人工费调整表 |
| 4 | 机械费调整 | 按新疆机械台班单价调整，燃料动力费按市场价 | 机械费调整表 |
| 5 | 管理费利润 | 按工程类别和企业类别取管理费和利润 | — |
| 6 | 措施费计取 | 组织措施费+技术措施费按定额规定计取 | — |
| 7 | 规费税金 | 按现行标准计取规费（社保+公积金）和税金(9%) | — |
| 8 | 汇总出价 | 生成单位工程概预算表，检查合理性 | 单位工程造价汇总表 |

### 3.2 定额套用技巧

| 技巧 | 说明 |
|---|---|
| 就近原则 | 特征最接近的定额优先套用，避免挂靠套用 |
| 换算标记 | 所有定额换算在预算书中标注换算内容，便于审核 |
| 系数调整 | 注意定额说明中的标高系数、厚度系数、运距系数等 |
| 新旧衔接 | 2024新定额中部分子目已整合，注意与2021定额的差异 |
| 补充定额 | 无对应定额子目时，编制补充定额并按程序报批 |

### 3.3 市场价导入注意事项

| 材料类别 | 导入方式 | 注意要点 |
|---|---|---|
| 大宗材料 | 当期信息价 | 水泥、钢筋、商品混凝土优先采用信息价 |
| 装饰材料 | 市场询价 | 瓷砖、卫浴、灯具等品牌差异大，按品牌规格询价 |
| 安装材料 | 信息价+询价 | 管材、线缆信息价覆盖较好，设备按品牌询价 |
| 新疆特色材料 | 市场询价 | 当地产石材、保温材料按厂家报价，附运距费 |

### 3.4 新疆组价特殊要点

| 要点 | 说明 |
|---|---|
| 人工费调增规则 | 人工调增部分只计税金，不计管理费利润规费（新疆特有规则） |
| 各地区人工差异 | 乌鲁木齐二类工148元/工日，克州126元/工日，差异约17% |
| 特殊材料运费 | 南疆三地州材料运费高于信息价10-15%，需另计 |
| 高原降效 | 海拔2500m以上项目，人工机械效率降低，按系数调整 |
| 冬季施工措施费 | 按各地州冬季施工天数及温度分区差异化计取 |

## 第四阶段：调价策略阶段

### 4.1 不平衡报价技巧

| 策略 | 操作方式 | 适用情况 | 风险提示 |
|---|---|---|---|
| 早收款 | 前期项目（土方、基础）单价调高5-10% | 资金流紧张的项目 | 结算时可能被审减 |
| 后期让利 | 后期项目（装饰、室外）单价适当调低 | 平衡总价不超限 | 后期实际量增加时吃亏 |
| 量增调高 | 预计实际量大于清单量的项目单价调高10-15% | 有经验的施工方案判断 | 量未增加时风险自担 |
| 量减调低 | 预计实际量小于清单量的项目单价调低 | 设计深度不够的暂定项目 | 量未减少时利润损失 |
| 新增项目 | 无定额可套的项目报高价 | 新技术新工艺 | 可能被业主指定分包 |
| 暂估价联动 | 配合暂估价项目的施工报价调高 | 指定分包配合的项目 | — |

### 4.2 总价控制策略

| 策略 | 操作方式 | 优缺点 |
|---|---|---|
| 下浮法 | 总价下浮一定比例（3-8%）后投标 | 简单明了，但利润较薄 |
| 主材控制 | 压低主材价，提高人工和措施费 | 容易通过审核，但材料采购有压力 |
| 组合策略 | 部分项调高，部分项调低，内部平衡 | 技术含量高，需要经验支撑 |

### 4.3 新疆投标调价特殊考虑

| 序号 | 考虑因素 | 影响与对策 |
|---|---|---|
| 1 | 南疆项目运输成本 | 运距3倍于北疆，调价时充分考虑运费系数 |
| 2 | 冬季停工损失 | 每年11月-次年3月停工，机械租赁费损失计入 |
| 3 | 劳务人员流动 | 农忙季节人工费上涨15-25%，报价时预留弹性 |
| 4 | 当地材料采购 | 部分偏远地区只能采购到当地材料，质量差异大 |
| 5 | 维稳费用 | 部分南疆项目需考虑维稳安保费用 |

## 第五阶段：对量审核阶段

### 5.1 常见争议点

| 争议类型 | 甲方/审计方观点 | 施工方观点 | 解决建议 |
|---|---|---|---|
| 钢筋搭接长度 | 按平法图集最小值 | 按实际施工做法+规范要求 | 以施工方案+规范为准 |
| 土方放坡系数 | 按定额规定不放坡或1:0.33 | 按实际施工1:0.5-0.75 | 以批准的施工方案为准 |
| 脚手架搭设范围 | 按定额计算规则 | 按实际搭设范围 | 分清定额包含和增补范围 |
| 垂直运输机械 | 按定额综合考虑 | 按实际使用台班 | 大型机械有进退场费 |
| 材料损耗率 | 按定额标准损耗率 | 按实际损耗率 | 以定额损耗率为基础，超耗论证 |
| 超高增加费 | 按定额计算规则 | 按实际施工高度 | 查定额说明中的超高分段 |

### 5.2 新疆对量特殊争议

| 争议点 | 新疆特殊性 | 处理原则 |
|---|---|---|
| 冬季施工措施费 | 冬季施工天数各地不同 | 以批准的冬季施工方案及现场记录为准 |
| 高原降效 | 海拔≥2500m才计取 | 以海拔高程测量报告为准 |
| 特殊材料运距 | 新疆幅员辽阔，运距差异大 | 以材料运距签证为准 |
| 抗冻混凝土等级 | 各地对抗冻要求不同 | 以设计图纸为准 |
| 人工调差 | 调差政策变动频繁 | 以最新调差文件执行日期为准 |

### 5.3 对量策略建议

| 策略 | 说明 |
|---|---|
| 先易后难 | 先对无争议项，再对有争议项，最后攻坚 |
| 证据为王 | 所有争议以设计图纸、施工方案、现场签证为准 |
| 抓大放小 | 金额占比80%的主要项目重点对，小项可协商让步 |
| 分批确认 | 每对完一部分立即签署确认单，避免整体搁置 |
| 会议纪要 | 每次对量会议形成纪要双方签字，避免反复 |

## 第六阶段：结算编制阶段

### 6.1 资料清单

| 序号 | 资料名称 | 要求 | 重要性 |
|---|---|---|---|
| 1 | 施工合同及补充协议 | 原件扫描 | ★★★★★ |
| 2 | 招标文件及投标文件 | 含工程量清单、商务标 | ★★★★★ |
| 3 | 竣工图纸（含电子版） | 经审核合格的竣工图 | ★★★★★ |
| 4 | 设计变更通知单 | 编号连续、签章完整 | ★★★★★ |
| 5 | 现场签证单 | 各方签章齐全，时间明确 | ★★★★★ |
| 6 | 工程联系单 | 编号连续 | ★★★★☆ |
| 7 | 材料认价单 | 甲供材+乙供材分别提供 | ★★★★☆ |
| 8 | 会议纪要 | 与造价相关的全部纪要 | ★★★☆☆ |
| 9 | 施工组织设计 | 经批准的施工方案 | ★★★★☆ |
| 10 | 隐蔽工程验收记录 | 描述清晰，签章完整 | ★★★★☆ |
| 11 | 竣工验收报告 | 各单位签章齐全 | ★★★★★ |
| 12 | 工程量计算书 | 清晰可查的计算过程 | ★★★★☆ |
| 13 | 进度款支付台账 | 历次支付明细 | ★★★★★ |
| 14 | 甲供材领用台账 | 领用记录+核销单 | ★★★★★ |

### 6.2 结算编制要点

| 序号 | 要点 | 操作说明 |
|---|---|---|
| 1 | 量价核对 | 以竣工图纸为基础重新算量，与原清单量对比分析差异 |
| 2 | 变更分类整理 | 设计变更+签证+联系单分类编号，逐项计算费用 |
| 3 | 材料价差调整 | 按合同约定调整材料价差（5%以内不调，超出调差） |
| 4 | 索赔费用计算 | 收集索赔证据，按合同约定的程序和时限提交 |
| 5 | 甲供材扣回 | 按实际领用量和合同约定价格扣回，超领按1.2倍扣回 |
| 6 | 质保金扣除 | 结算总价3%预留质保金，质保期满后退还 |
| 7 | 税金调整 | 如施工期内税率变化，按税务政策调整 |
| 8 | 结算书编制 | 按合同约定的格式编制结算书，附全部计算依据 |

### 6.3 结算审核常见问题

| 问题 | 表现 | 预防措施 |
|---|---|---|
| 竣工图与实际不符 | 竣工图标注与现场不一致 | 结算前现场复核关键尺寸 |
| 签证时效性 | 事后补签的签证不被认可 | 及时签证，避免积压 |
| 量价重复 | 不同变更中有重复工程量 | 逐项梳理，避免重复计取 |
| 资料缺失 | 关键签证找不到 | 建立资料台账，专人管理 |
| 价格争议 | 材料价格取定不一致 | 保留同期信息价文件 |

### 6.4 新疆结算特殊要求

| 序号 | 要求 | 说明 |
|---|---|---|
| 1 | 冬季施工费用结算 | 须提供冬季施工方案批准文件及温度记录 |
| 2 | 人工费调差 | 按新疆住建厅发布的调差文件执行，注意文件有效期 |
| 3 | 信息价 | 结算材料价以新疆建设工程信息价为准，信息价未覆盖的以市场询价 |
| 4 | 规费结算 | 须提供社保、公积金缴纳凭证，否则不计取 |
| 5 | 南疆补贴 | 在南疆四地州施工的项目，注意计算边远地区补贴 |

## 全流程时间节点控制

| 阶段 | 参考天数 | 说明 |
|---|---|---|
| 收图→识图→提量 | 15-30天 | 复杂项目适当延长 |
| 清单编制 | 7-14天 | 标准层项目可缩短 |
| 组价 | 10-20天 | 安装工程较土建耗时 |
| 调价策略 | 3-7天 | 结合招标策略确定 |
| 对量审核 | 15-45天 | 争议大的项目耗时较长 |
| 结算编制 | 20-40天 | 资料齐全是前提 |

> **注：** 以上时间为经验参考值，实际时间根据项目复杂程度、人数配置、委托方要求调整。新疆项目因地域广阔、交通运输条件差，建议在时间安排上预留10-20%的缓冲期。
