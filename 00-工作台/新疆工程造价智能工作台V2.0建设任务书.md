## 二、项目愿景

基于现有知识库资产，搭建一个**本地化、AI驱动、可交互**的造价智能工作台，实现从"资料堆积"到"知识资产"的跃迁，让新疆造价工程师在30秒内获得任一项目的报价基准、定额依据、风险提示、历史对标。

## 三、建设范围与详细要求

### 模块1：项目知识图谱构建
【目标】将现有非结构化项目资料转化为可查询的结构化知识节点

#### 1.1 项目元数据标准化
遍历全部项目文件夹，为每个项目建立标准化元数据表：

| 字段 | 说明 | 必填 |
|---|---|---|
| 项目ID | 自动生成，格式：XJ-年份-序号 | ✅ |
| 项目名称 | 全称+简称 | ✅ |
| 所在地市 | 乌鲁木齐/哈密/喀什/伊犁等 | ✅ |
| 专业类型 | 房建/市政/公路/水利/工业 | ✅ |
| 建筑面积/规模 | ㎡/km/万元 | ✅ |
| 招标方式 | 公开招标/邀请/竞争性谈判 | ✅ |
| 合同类型 | 固定总价/固定单价/成本加酬金 | ✅ |
| 中标价/单方造价 | 元、元/㎡ | ✅ |
| 工期/冬休期 | 月 | ✅ |
| 中标单位 | 企业名称+资质等级 | ✅ |
| 下浮率 | % | ✅ |
| 核心风险点 | 3~5条提炼 | ✅ |
| 知识库路径 | 相对路径 | ✅ |

#### 1.2 造价核心指标提取
为每个项目提取造价核心指标：
- 各专业工程造价占比（建筑/装饰/电气/暖通/给排水/消防等）
- 人材机占比结构（按专业分部）
- 措施费占比（安全文明、冬施、二次搬运等）
- 暂列金额、暂估价占比
- 设备费占比及主要设备清单

#### 1.3 项目关联图谱
建立项目间关联图谱：
- 同地区项目对标链接
- 同专业项目对标链接
- 同中标单位项目链接

### 模块3：造价AI助手提示词体系（核心产出）
【目标】建立10+个标准化造价工作提示词，一键调用即可完成专业任务

#### 3.1 清单组价助手提示词
```
【输入】清单编码、项目名称、项目特征、工程量、工程地点、工期
【输出】匹配新疆2024定额子目、人材机拆分、管理费利润规费计算、综合单价分析、风险提示
【要求】自动套用新疆取费标准，人工调增只计税，标注待核实项
```

#### 3.2 变更签证计价助手提示词
```
【输入】签证内容、现场情况、合同约定、原清单单价
【输出】变更计价依据、新增单价组价、量差计算、价差调整、索赔费用计算、签证风险等级
【要求】按合同条款优先，无约定按定额，标注争议点
```

#### 3.3 投标报价策略助手提示词
```
【输入】最高限价、评标办法、项目特征、竞争对手分析
【输出】最优报价区间、不平衡报价建议、风险清单项、成本底线测算、报价下浮率建议
【要求】结合新疆中标下浮率大数据
```

#### 3.4 成本纠偏助手提示词
```
【输入】目标成本、实际发生成本、偏差项明细
【输出】偏差原因分析（量差/价差/效率）、纠偏措施、预计最终成本、关键控制点
【要求】±5%偏差红线预警
```

#### 3.5 合同审查助手提示词
```
【输入】合同文本或关键条款
【输出】按🔴高风险/🟡中风险/🔵低风险分级标注，绑定具体法条编号，给出修改建议
【要求】重点审查价款、工期、变更、索赔、付款、质保、违约条款
```

#### 3.6 方案比选助手提示词
```
【输入】两个及以上施工方案描述、工程量、工期要求
【输出】各方案造价测算、工期对比、风险对比、综合评分、推荐意见
【要求】考虑新疆冬施、运输、高原等特殊条件
```

#### 3.7 工程量审核助手提示词
```
【输入】图纸尺寸描述、清单工程量、计算规则
【输出】工程量复核计算、差异原因、核增/核减金额、审核意见
【要求】标注计算依据和公式
```

#### 3.8 进度款审核助手提示词
```
【输入】已完工程量、合同付款节点、累计已付款、变更签证金额
【输出】本次应付金额计算、付款比例核对、扣款事项、付款审核意见
```

### 模块4：项目对标与价格指数系统
【目标】建立新疆工程造价大数据对标能力

#### 4.1 单方造价指数表（按专业、按地区）
- **房建类**：住宅/办公/商业/学校/医院/工业厂房（元/㎡）
- **市政类**：道路/给排水/供热/燃气（元/km或元/㎡）
- **公路类**：高速公路/国省干道/农村公路（元/km）
- **水利类**：渠道/管道/建筑物（元/m或元/座）

#### 4.2 造价指标分析模板
- 新建项目输入基本信息后，自动匹配3~5个对标项目
- 自动计算偏离度，标注异常指标
- 生成对标分析报告框架

#### 4.3 价格波动预警
- 主要材料价格波动超过±5%自动预警
- 人工单价调整跟踪
- 定额、费率政策变更跟踪

## 六、风险与注意事项

⚠ **红线规则，严禁违反**

1. 定额子目缺少时，标注"⚠定额待补"并建议相似子目参考
2. 价格数据缺少时，标注"⚠待询价"并给出询价渠道建议
3. 图纸信息不完整时，标注"⚠需补充图纸尺寸"
4. 合同条款解读有歧义时，标注"⚠法条需复核"并给出不同解读方向
5. 所有成本测算必须拆分为人工/辅材/主材/机械，禁止只给总价
6. **禁止固定使用管理费28%、利润18%、规费22%**，必须按项目实际动态调整
7. **人工调增部分只计税金**，不计管理费、利润、规费

## 八、执行原则

1. **结论先行**：先给判断、金额、风险等级，再给依据和推导
2. **表格优先**：表格 > 清单 > 段落，预算/报价/审查内容必须结构化
3. **新疆适配**：默认按新疆工程环境、材料运输、冬施、高原/风沙/地震设防等条件考虑
4. **数据驱动**：不编造定额、价格、图纸尺寸、合同条款；缺依据必须标 `⚠待核实`
5. **可落地**：输出要能直接复制进 Excel、Word、会议纪要、签证单、报价说明
6. **去客套**：禁止"请问是否需要""您看可以吗"等空话
