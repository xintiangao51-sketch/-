# 原始资料索引

> **目的**：本页面是 D:\知识库 与 D:\高治国资料\知识库原始资料\ 之间的**桥梁**。
> 
> 知识库存放**精炼后的 MD 卡片**，原始资料目录存放**未被提炼的 PDF/Excel/Word**。
> **D:\知识库 保持纯 MD**，避免被非 MD 文件污染。

---

## ⚡ 一句话结论

`D:\高治国资料\知识库原始资料\` 是 `D:\知识库\` 的**原料仓库**。
知识库的每张 KC 卡片背后，都能在原始资料目录找到 1-3 份原始文件作为佐证。

## 五、3 个 PowerShell 脚本（待你 Windows 端执行）

| 脚本 | 路径 | 作用 | 预计耗时 |
|------|------|------|:------:|
| N1 创建目录 | `D:\知识库\.scripts\N1_创建原始资料目录.ps1` | 创建 100+ 子目录 | 5 秒 |
| N2 迁移业务资料 | `D:\知识库\.scripts\N2_迁移归档库业务资料.ps1` | 99-归档库 → 原始资料 | 30 分 |
| N3 删除零价值 + 自检 | `D:\知识库\.scripts\N3_删除零价值_自检.ps1` | 删发票/学位证书 + 自检 | 5 分 |

执行顺序：
```powershell
# 1. 创建目录
powershell -ExecutionPolicy Bypass -File D:\知识库\.scripts\N1_创建原始资料目录.ps1

# 2. 复制 README（老高已写到 D:\知识库\.scripts\N4_README_原始资料目录.md，复制到 D:\高治国资料\知识库原始资料\README.md）

# 3. 迁移
powershell -ExecutionPolicy Bypass -File D:\知识库\.scripts\N2_迁移归档库业务资料.ps1

# 4. 删零价值
powershell -ExecutionPolicy Bypass -File D:\知识库\.scripts\N3_删除零价值_自检.ps1
```
