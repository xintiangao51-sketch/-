## 四、迁移执行批次与 PowerShell 脚本

### 计划：4 批次执行

| 批次 | 内容 | 文件数 | 估计大小 | 耗时 |
|:---:|------|:-----:|:-------:|:---:|
| **P1** | 01-造价核心 非MD文件 | ~400 | ~600 MB | ~10 分钟 |
| **P2** | 03-合同与法务 非MD文件 | ~700 | ~450 MB | ~10 分钟 |
| **P3** | 04-施工方案 非MD文件 | ~300 | ~120 MB | ~5 分钟 |
| **P4** | 05-投标报价 非MD文件 | ~500 | ~1.8 GB | ~15 分钟 |

### 第 1 批：P1_P2_合同法务迁移.ps1（先做合同，量最大）

```powershell
# P1 03-合同与法务 全部非MD → 原始资料/03-合同法务资料/
$src = "D:\知识库\03-合同与法务"
$dst = "D:\高治国资料\知识库原始资料\03-合同法务资料"

# 合同范本
Get-ChildItem "$src\合同范本" -Include *.doc,*.docx,*.pdf -Recurse -File -ErrorAction SilentlyContinue |
  ForEach-Object { Move-Item $_.FullName "$dst\合同范本\" -Force }

# 法规
Get-ChildItem "$src\法规" -Include *.doc,*.docx,*.pdf -Recurse -File -ErrorAction SilentlyContinue |
  ForEach-Object { Move-Item $_.FullName "$dst\法规\" -Force }

# 索赔案例
Get-ChildItem "$src\索赔案例" -Include *.doc,*.docx,*.pdf -Recurse -File -ErrorAction SilentlyContinue |
  ForEach-Object { Move-Item $_.FullName "$dst\合同审核清单\" -Force }
```

### 第 2 批：P2_P3_施工方案迁移.ps1

```powershell
# P2 04-施工方案 全部非MD → 原始资料/00-项目资料/
$src = "D:\知识库\04-施工方案"
$dst = "D:\高治国资料\知识库原始资料\00-项目资料\施工方案"

Get-ChildItem $src -Include *.doc,*.docx,*.xls,*.xlsx,*.pdf -Recurse -File -ErrorAction SilentlyContinue |
  ForEach-Object { 
    $subPath = $_.FullName.Replace($src, "").TrimStart("\")
    Move-Item $_.FullName "$dst\$subPath" -Force
  }
```
