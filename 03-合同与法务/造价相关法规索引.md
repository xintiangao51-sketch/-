### 1.2 中华人民共和国建筑法

| 章节 | 主要内容 | 造价相关要点 |
|---|---|---|
| 第二章 建筑许可 | 施工许可证、从业资格 | 开工条件 |
| 第三章 建筑工程发包与承包 | 发包、承包、禁止行为 | 禁止转包、违法分包 |
| 第四章 建筑工程监理 | 监理的范围和职责 | 监理的权利义务 |
| 第五章 建筑安全生产管理 | 安全生产责任 | 安全文明施工 |
| 第六章 建筑工程质量管理 | 质量责任、质量保修 | 质量保修期 |
| 第七章 法律责任 | 违法发包、转包等的处罚 | 违法后果 |

**重点法条：**
- 第28条：禁止承包单位将其承包的全部建筑工程转包给他人
- 第29条：建筑工程总承包单位可以将承包工程中的部分工程发包给具有相应资质条件的分包单位
- 第60条：建筑物在合理使用寿命内，必须确保地基基础工程和主体结构的质量
- 第62条：建筑工程实行质量保修制度，保修范围和最低保修期限由国务院规定

## 二、行政法规层级

### 2.1 建设工程质量管理条例（国务院令第279号）

| 条款 | 内容 | 造价相关要点 |
|---|---|---|
| 第16条 | 竣工验收条件 | 竣工验收应具备的条件 |
| 第25条 | 施工单位的质量责任 | 施工单位不得转包、违法分包 |
| 第32条 | 施工质量缺陷的返修 | 施工单位对施工中出现质量问题的建设工程或者竣工验收不合格的建设工程，应当负责返修 |
| 第39条 | 质量保修制度 | 建设工程实行质量保修制度 |
| 第40条 | 最低保修期限 | 基础设施、地基基础和主体结构为设计文件规定的合理使用年限；屋面防水5年；供热供冷2个采暖期/供冷期；电气管线、给排水管道、设备安装和装修工程2年 |
| 第41条 | 保修责任 | 建设工程在保修范围和保修期限内发生质量问题的，施工单位应当履行保修义务，并对造成的损失承担赔偿责任 |

### 2.2 建设工程安全生产管理条例（国务院令第393号）

| 条款 | 内容 | 造价相关要点 |
|---|---|---|

| 第22条 | 施工单位安全费用 | 施工单位对列入建设工程概算的安全作业环境及安全施工措施所需费用，应当专款专用 |
| 第27条 | 安全技术交底 | 建设工程施工前，施工单位负责项目管理的技术人员应当对有关安全施工的技术要求向施工作业班组、作业人员作出详细说明 |

### 2.3 建设工程勘察设计管理条例（国务院令第662号修订）

- 设计文件的编制要求
- 设计变更的管理
- 设计文件的审批

### 3.2 建筑工程施工发包与承包计价管理办法（住建部令第16号）

| 条款 | 内容 |
|---|---|
| 第6条 | 全部使用国有资金投资或者以国有资金投资为主的建筑工程，应当采用工程量清单计价 |
| 第8条 | 最高投标限价应当依据工程量清单、工程计价有关规定和市场价格信息等编制 |
| 第14条 | 发承包双方应当在合同中约定，发生下列情形时合同价款的调整方法 |

### 3.3 建设工程价款结算暂行办法（财建〔2004〕369号）

- 工程竣工价款的结算
- 索赔价款结算
- 合同价款争议的解决

### 3.4 最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释（一）（法释〔2020〕25号）

| 条款 | 内容 | 造价相关应用 |
|---|---|---|
| 第1条 | 合同无效的情形 | 未取得资质、超越资质、借用资质等 |
| 第2条 | 合同无效但验收合格的处理 | 参照合同约定折价补偿 |
| 第3条 | 合同无效且验收不合格的处理 | 修复后合格的可以请求折价补偿 |
| 第17条 | 欠付工程款利息 | 当事人对欠付工程价款利息计付标准有约定的，按照约定处理；没有约定的，按照同期同类贷款利率或者同期贷款市场报价利率计息 |
| 第18条 | 利息起算时间 | 从应付工程价款之日开始计付 |
| 第19条 | 工程量争议 | 按照施工过程中形成的签证等书面文件确认 |
| 第20条 | 竣工结算文件的认可 | 发包人收到竣工结算文件后，在约定期限内不予答复，视为认可 |
| 第21条 | 黑白合同 | 当事人就同一建设工程另行订立的建设工程施工合同与经过备案的中标合同实质性内容不一致的，应当以备案的中标合同作为结算工程价款的根据 |
| 第22条 | 固定价合同的鉴定 | 当事人约定按照固定价结算工程价款，一方当事人请求对建设工程造价进行鉴定的，不予支持 |
| 第26条 | 实际施工人权利保护 | 实际施工人以转包人、违法分包人为被告起诉的，人民法院应当依法受理 |
| 第27条 | 建设工程价款优先受偿权 | 装饰装修工程的承包人享有优先受偿权 |
| 第35条 | 优先受偿权的范围 | 承包人就逾期支付建设工程价款的利息、违约金、损害赔偿金等主张优先受偿的，人民法院不予支持 |
| 第36条 | 优先受偿权的顺位 | 建设工程价款优先受偿权优于抵押权和其他债权 |
| 第41条 | 优先受偿权的期限 | 承包人应当在合理期限内行使建设工程价款优先受偿权，但最长不得超过十八个月，自发包人应当给付建设工程价款之日起算 |

### 3.5 建设工程质量保证金管理办法（建质〔2017〕138号）

| 条款 | 内容 |
|---|---|
| 第7条 | 发包人应按照合同约定方式预留保证金，保证金总预留比例不得高于工程价款结算总额的3% |
| 第8条 | 缺陷责任期从工程通过竣工验收之日起计 |
| 第10条 | 缺陷责任期内，发包人在使用过程中，发现已接收的工程存在新的缺陷或已修复的缺陷部位或部件又遭损坏的，承包人应负责修复 |
