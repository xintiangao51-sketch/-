## 📌 一、核心结论（一句话版）

**按发票上的税额抵扣，与税率高低无关。** 企业实际缴纳的增值税 = 销项税额 − 进项税额。只要业务真实、取得合规的增值税专用发票，13% 的进项税可以直接抵扣 9% 的销项税，**不存在"亏 4 个点"的说法**。

---

## 📐 二、基础原理

### 2.1 价外税三性

| 特性 | 含义 | 实操意义 |
|------|------|---------|
| **透明性** | 价款与税款分离 | 报价单应注明"不含税价+税额" |
| **链条性** | 上缴=下抵，环环相扣 | 上下游税率差异不影响抵扣 |
| **差额性** | 仅对"增值额"课税 | 进项大于销项时形成留抵 |

### 2.2 核心公式

```
应纳税额 = 当期销项税额 − 当期进项税额 − 上期留抵税额
留抵税额 = 当期进项税额 − 当期销项税额（结果为负时）
```

### 2.3 法规依据

| 法规/公告 | 生效日期 | 关键内容 |
|-----------|---------|---------|
| 《增值税暂行条例》第八条 | 1994.1.1（多次修订） | 购进货物/服务的进项税准予从销项税中抵扣 |
| 财税〔2016〕36号 | 2016.5.1 | 建筑业、销售不动产适用 11%→**9%** |
| 财税〔2018〕32号 | 2018.5.1 | 税率调整过渡：按纳税义务发生时间判定 |
| 财政部 税务总局 海关总署公告2019年第39号 | 2019.4.1 | 16%→**13%**、10%→**9%**；全行业留抵退税 |
| 国家税务总局公告2019年第31号 | 2019.4.1 | 增值税发票综合服务平台勾选认证 |
| 国家税务总局公告2017年第11号 | 2017.6.1 | 建筑业跨县市区预缴 |
| 财政部 税务总局公告2022年第21号 | 2022.4.1 | 大规模留抵退税（小微+六大行业） |

## 🎯 三、建筑业税率适用全图

### 3.1 业务类型 × 计税方法

| 业务类型 | 一般计税 | 简易计税 | 选择规则 |
|---------|---------|---------|---------|
| **建筑服务**（新建/改建/扩建） | **9%** | 3%（差额） | 可二选一，**36个月内不得变更** |
| **销售不动产** | 9% | 5%（差额征收） | 房企老项目可选 |
| **出租不动产**（2016.4.30前取得） | 9% | 5%（差额） | 老项目可选择 |
| **清包工** | 9% | 3% | 甲供工程适用 |
| **甲供工程** | 9% | 3% | 甲供材占比决定选择 |
| **工程监理/勘察/设计** | **6%** | — | 现代服务业，非建筑业 |
| **货物销售**（建材单独销售） | **13%** | — | 兼营情形，单独核算 |

> ⚠️ **关键提醒**：建筑业税率 9% 适用于服务主合同；货物销售 13% 适用于单独卖建材。混合销售判定参见 [[深度解读了2026年增值税法新规]]。

### 3.2 简易计税 vs 一般计税决策树

```
        ┌─────────────────────────┐
        │ 是否符合简易计税条件？    │
        └──────────┬──────────────┘
                   │
       ┌───────────┴───────────┐
       │YES                   │NO
       ▼                       ▼
┌──────────────────┐    ┌──────────────┐
│ 计算两种方案税负  │    │ 一般计税 9%  │
└────────┬─────────┘    └──────────────┘
         │
         ▼
┌─────────────────────────────────────────┐
│ 临界点公式：                            │
│ • 一般计税：0.09R − 0.13C              │
│ • 简易计税：0.03R÷(1+3%) ≈ 0.0291R     │
│ • 相等时：C/R ≈ 47%                     │
│                                         │
│ 决策：                                  │
│ • 采购/收入 ≥ 47% → 一般计税更优       │
│ • 采购/收入 < 47% → 简易计税更优       │
└─────────────────────────────────────────┘
```

> 💡 **临界点推导**：令 0.09R − 0.13C = 0.0291R，解得 C/R ≈ 0.4685。
> 一旦选择简易计税，**36个月内不得变更**，需慎重决策！

## ✅ 四、抵扣资格判定

### 4.1 必须满足"四流一致"

| 维度 | 具体要求 |
|------|---------|
| **合同流** | 采购合同主体与发票抬头一致 |
| **发票流** | 一般计税取专票；简易计税不能抵扣进项 |
| **资金流** | **对公账户**转账，禁止现金大额交易 |
| **货物流/服务流** | 出入库单、验收单、施工记录、监理签字 |

> 🔴 **红线**：四流不一致 = 不得抵扣 + 涉嫌虚开风险

### 4.2 不得抵扣的进项税（财税〔2016〕36号第二十七条）

- 集体福利、个人消费
- 餐饮服务、居民日常服务
- **贷款服务**（利息、手续费、咨询费）——建筑业重灾区
- 非正常损失（管理不善导致被盗、霉烂）
- 购进的贷款服务、餐饮服务、居民日常服务、娱乐服务

### 4.3 时效要求

- **勾选认证**：360 天内（国家税务总局公告2019年第31号）
- **发票保存**：10 年

## 🛠️ 五、实操流程

### 5.1 发票管理 SOP

```
取得发票 → 查验真伪（增值税发票综合服务平台）
    → 360 天内勾选认证
    → 申报期内填报申报表
    → 归档保存（10 年）
```

### 5.2 申报表填报（一般计税）

| 表位 | 项目 | 数据来源 |
|------|------|---------|
| 第 1 栏 | 按适用税率计税销售额 | 收入台账 |
| 第 2 栏 | 销项税额 | 销售额 × 9% |
| 第 12 栏 | 进项税额 | 已认证发票合计 |
| 第 22 栏 | 上期留抵税额 | 上月申报表 |
| 第 24 栏 | 应纳税额 | 销项 − 进项 − 留抵 |

### 5.3 跨县市区预缴（重要！）

**适用情形**：建筑业纳税人在机构所在地以外的县市区提供建筑服务

**预缴公式**：
```
预缴税额 = (全部价款和价外费用 − 支付的分包款) ÷ (1 + 9%) × 2%
```

**注意事项**：
- 预缴 ≠ 申报，预缴后回机构所在地申报
- 预缴可抵减应纳税额
- **未预缴将面临滞纳金**

> 📚 政策依据：国家税务总局公告2017年第11号

## 💰 六、留抵退税全攻略

### 6.1 退税条件（财政部2022年21号公告）

| 主体类型 | 条件 | 退税比例 |
|---------|------|---------|
| **小微企业** | 增量留抵 > 0 | **全额**退还 |
| **六大行业** | 制造业、科研/技术服务、电力/热力/燃气/水生产、软件信息技术、生态环保、交通运输/仓储邮政 | **全额** |
| **其他行业** | 增量留抵 > 50 万 + 6 个月连续零或负 + 3% 进项构成 | 60% |

### 6.2 增量留抵计算

```
增量留抵 = 当期期末留抵 − 2019 年 3 月底期末留抵
```

### 6.3 申请材料

- 《退（抵）税申请表》
- 完税凭证（增值税申报表）
- 退税所属期完税证明
- 行业证明材料（如制造业资质）

> 💡 **关键洞见**：当进项持续大于销项时，企业相当于获得**国家提供的无息贷款**！建筑业（材料密集型）常常出现这种情况。

## 📊 七、案例测算（实战版）

### 7.1 基础案例（原笔记示例）

**场景**：某建筑企业（一般纳税人）

| 项目 | 数值 | 计算 |
|------|------|------|
| 不含税工程收入 | 1000 万 | — |
| 销项税额 | 90 万 | 1000 × 9% |
| 不含税材料采购 | 800 万 | — |
| 进项税额 | 104 万 | 800 × 13% |
| **应纳税额** | **−14 万** | 90 − 104 |

**结论**：当期无需缴纳增值税，形成 **14 万留抵税额**，可结转下期抵扣或申请留抵退税。

## 🚨 八、稽查重点与红线清单

### 8.1 高频风险点

1. **虚开发票**——无真实交易代开、虚开（刑事风险）
2. **资金回流**——对私转账、现金交易配合发票
3. **四流不一致**——合同、发票、付款主体不一致
4. **跨期确认**——故意延后或提前纳税义务
5. **虚假简易计税**——不符合条件却按 3% 申报
6. **甲供材违规选择**——以清包工名义逃避纳税

### 8.2 合规建议

- 建立"供应商黑名单"，杜绝"开票公司"
- 大额对私付款必须有合理商业理由 + 完税证明
- 跨期合同**必须**明确税率调整条款
- 定期内部审计（建议每季度）
- 留抵退税前**自查进项真实性**

## ❓ 九、常见误区澄清

| 误区 | 正解 | 依据 |
|------|------|------|
| 13% 进项只能抵扣 13% 销项 | 抵扣的是"税额"不是"税率"，可全额抵扣 9% 销项 | 增值税暂行条例第八条 |
| 存在 4 个点的税差亏损 | 实际税负取决于"增值率"，非税率差 | 价外税原理 |
| 小规模 3% 专票不能抵扣 | 一般纳税人取得小规模 3% 专票，可按票面税额抵扣 | 财税〔2016〕36号 |
| 甲供工程必须按 9% | 可选择简易计税 3%，但 36 个月不得变更 | 财税〔2016〕36号 |
| 留抵退税很难申请 | 小微/六大行业基本条件即可，2022 年起大幅放宽 | 财政部2022年21号公告 |
