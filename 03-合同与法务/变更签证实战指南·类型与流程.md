# 变更签证实战指南·类型与流程

> 枢纽页：[[变更签证实战指南]]（60 行框架速览）
> 原版章节：v1.0 §一 变更的 5 种类型及处理程序
> 适用场景：收到变更通知单，需判断属哪一类、按何流程处理

---

## 设计变更

**定义：** 设计单位对原施工图设计文件的修改、补充和完善。

**常见原因：**
- 设计图纸与现场实际不符（新疆常见：地质条件与勘察报告不符）
- 设计错误、遗漏
- 发包人要求调整功能或标准
- 优化设计以降低造价或方便施工

**处理程序：**
```
发起（设计单位/发包人/承包人）→ 设计变更通知单 → 总监审核 → 发包人审批 → 下发施工单位 → 施工实施 → 签证确认
```

**关键节点：**
- 设计变更通知单须在施工前下发（避免返工）
- 收到变更后7天内承包人应提交变更价款报告

---

## 施工变更

**定义：** 因施工条件变化或发包人要求调整施工方案引起的变更。

**常见原因：**
- 施工场地条件变化（新疆常见：开挖后发现地质条件与勘察不符）
- 施工顺序、施工方法调整
- 发包人要求的施工标准提高
- 现场发现地下障碍物、管线

**处理程序：**
```
发现变化 → 承包人提出变更申请 → 监理/发包人确认 → 变更方案论证 → 审批 → 实施 → 签证确认
```

---

## 工程量清单偏差

**定义：** 招标工程量清单与施工图纸实际工程量之间的差异。

**常见原因：**
- 清单编制漏项、错项
- 工程量计算偏差（新疆定额的计量规则与国标清单差异）
- 项目特征描述不准确

**处理程序：**
```
发现偏差 → 承包人复核并提交差异报告 → 发包人/造价咨询复核 → 确认 → 签证
```

**注意：** 根据 GB50500-2013，工程量偏差超过 15% 时，综合单价应进行调整。

---

## 现场签证

**定义：** 承包人与发包人（监理）对施工过程中涉及合同价款之外的责任事件所做的签认证明。

**适用范围：**
- 合同外零星用工、零星机械
- 突发事件的抢修、抢险
- 发包人临时要求的合同外工作
- 隐蔽工程现场确认
- 施工条件变化导致的额外工作

**处理程序：**
```
事件发生 → 现场记录（拍照/测量）→ 24小时内通知监理 → 三方现场确认 → 签证单签署 → 结算
```

**关键时效：** 24 小时通知红线——超期可能被拒签。

---

## 工期变更

**定义：** 因非承包人原因导致的施工工期调整。

**常见原因：**
- 发包人原因（场地移交延迟、图纸提供延迟、甲供材延迟）
- 不可抗力（新疆：极端低温、大风、暴雨、地震）
- 冬歇期延长或提前
- 政策因素（重大活动停工、环保要求停工）

**处理程序：**
```
事件发生 → 7天内提交工期延期申请及证据 → 监理审核 → 发包人确认 → 工期签证
```
