# 建设工程合同与法务核心速查

> 老高造价工作台 | 适用：新疆地区建设工程

## 一、法律框架

| 层级 | 文件 | 状态 |
|------|------|------|
| 基础法 | 《民法典》合同编·建设工程合同章（第788-808条） | 现行 |
| 司法解释 | 《最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释（一）》法释〔2020〕25号 | 现行 |
| 计价规范 | GB50500-2013《建设工程工程量清单计价规范》 | 现行 |
| 招标投标 | 《招标投标法》+《招标投标法实施条例》 | 现行 |

## 二、合同风险三级标注体系

### 🔴 红线（禁用/废标风险）

| 风险点 | 法条依据 | 处理 |
|--------|---------|------|
| 无资质/超越资质承揽 | 民法典第791条 + 建工司法解释第1条 | 必须拒绝 |
| 必须招标未招标 | 招标投标法第3条 | 必须拒绝 |
| 转包/违法分包 | 民法典第791条 | 必须拒绝 |
| 未取得规划许可擅自开工 | 建工司法解释第3条 | 必须拒绝 |

### 🟡 黄线（需调整/中风险）

| 风险点 | 法条依据 | 建议 |
|--------|---------|------|
| 计价方式约定不明 | 民法典第795条 | 明确固定单价/总价/可调价 |
| 付款节点模糊 | 建工司法解释第27条 | 按形象进度明确节点 |
| 变更签证流程不完整 | 民法典第798条 | 补充14天签证时限条款 |
| 材料价差调整机制缺失 | 民法典第533条（情势变更） | 加入±5%调价触发 |

### 🔵 蓝线（提示/低风险）

| 风险点 | 说明 |
|--------|------|
| 竣工结算期限过长 | 发承包双方约定不超过60日 |
| 质保金比例偏高 | 不得超过工程价款结算总额的3%（2017修订） |
| 争议解决方式 | 建议约定工程所在地法院管辖 |
