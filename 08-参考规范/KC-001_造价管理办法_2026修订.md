## 核心变化（对标旧版）

### 1️⃣ 施工过程结算全面推行（第20条）
- 经双方确认的过程结算文件 = 竣工结算组成部分
- 竣工后原则上不再重复审核 → 大幅缩短结算周期
- 实操影响：每期进度款必须做过程结算，不能再拖到竣工统一算

### 2️⃣ 信息化建设（第11条）
- 分类发布造价指标指数
- 建立市场价格监测预警机制
- 利好方向：信息价采集系统升级、历史数据积累

### 3️⃣ 信用管理（第31条）
- 建立信用档案 + 分级分类监管
- 信用差的造价咨询企业/个人将面临更严格检查

### 4️⃣ 禁止"以审代结"（第21条）
- 国有资金工程**不得以"未完成审计"为由拒绝或拖延结算**
- 重大利好：结算审核终于有法可依，不再被审计无限期拖住

## 对造价工作的影响

| 场景 | 影响 |
|------|------|
| 投标报价 | 安全生产措施费单独列项、不得删减 |
| 过程结算 | 每期都要做→工作量增加，但竣工后工作量大幅减少 |
| 结算审核 | 双方确认后不得单方换审 |
| 合同起草 | 第19条11项必备条款，缺项可能导致合同效力问题 |
| 造价咨询 | 禁止转包、禁止同时服务招标人和投标人 |
