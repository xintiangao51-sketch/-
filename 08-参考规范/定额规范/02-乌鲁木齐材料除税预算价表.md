## 三、价表编制说明

### 3.1 编制公式

```
材料预算价格 = (材料原价 + 运杂费 + 运损费) × (1 + 采购及保管费率)
```

### 3.2 费率

| 费用 | 费率 |
|:---|:---|
| 采购费率 | 1% |
| 保管费率 | 1.5% |
| **采购及保管费率** | **2.5%** |
| 运输损耗率 | 0.5%（部分材料） |

### 3.3 运距范围

乌鲁木齐城区范围：
- 东至：绕城高速公路
- 南至：天山区南界和绕城高速公路
- 西至：昌吉市东界
- 北至：五家渠市行政区界、米东区长山子镇、柏杨河哈萨克族乡南界

> 超出城区范围的运输部分可计算超运距费用，仅计增值税销项税额
