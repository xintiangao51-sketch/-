# 定额知识卡片模板

> 返回 [[08-参考规范/定额规范/01-新疆2024定额]]

---

## 模板说明

本模板用于创建定额知识卡片，每张卡片应包含：

| 字段 | 说明 | 必填 |
|:---|:---|:---:|
| tags | 标签分类 | ✅ |
| 定额编号 | 如 1-1-1 | ✅ |
| 定额名称 | 子目名称 | ✅ |
| 工作内容 | 适用范围 | ✅ |
| 单位 | 计量单位 | ✅ |
| 人工 | 人工消耗量 | ✅ |
| 材料 | 材料消耗量 | ✅ |
| 机械 | 机械消耗量 | ✅ |
| 备注 | 调整说明 | |

---

## 卡片示例

### 示例1：人工挖土方

```yaml
---
tags: [定额, 土方工程, 人工, 第一章]
定额编号: 1-1-1
定额名称: 人工挖土方（一、二类土）深度2m以内
工作内容: 挖土、装土、抛土于坑边1m以外
单位: m³
人工: 0.24工日
材料: -
机械: -
调整: 深度超过2m时，每加深1m增加0.04工日
地区: 全疆适用
---
```

### 示例2：砖基础

```yaml
---
tags: [定额, 砌筑工程, 砖基础, 第四章]
定额编号: 4-1-1
定额名称: 砖基础（M10水泥砂浆）
工作内容: 砖基础砌筑
单位: m³
人工: 0.96工日
材料: 砖523块、M10砂浆0.242m³
机械: -
调整: 基础宽度不同时可换算
地区: 全疆适用
---
```
