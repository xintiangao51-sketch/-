# 1-1-1 人工挖土方（一、二类土）深度2m以内

## 基本信息

| 字段 | 值 |
|:---|:---|
| 定额编号 | 1-1-1 |
| 定额名称 | 人工挖土方（一、二类土）深度2m以内 |
| 工作内容 | 挖土、装土、抛土于坑边1m以外 |
| 单位 | m³ |
| 人工消耗量 | 0.24工日 |
| 材料消耗量 | - |
| 机械消耗量 | - |

## 说明

1. 一、二类土包括：砂土、粘性土、种植土
2. 深度超过2m时，每加深1m增加0.04工日
3. 坑底面积超过20m²时，按机械挖土计算
