## 四大核心变化

### 🔴 变化1：规费不再单独列项
- 养老保险、失业保险、医疗保险、工伤保险、住房公积金
- → **分别计入人工单价和企业管理费**
- 影响：综合单价构成改变，报价对比时需注意口径统一

### 🔴 变化2：模板工程划入分部分项
- 模板工程**不再是措施项目**
- → 列入分部分项工程项目清单
- 影响：措施项目费范围缩小，模板需要逐项组价

### 🟡 变化3：措施项目统一以"项"计量
- 单价措施项目和总价措施项目均以"项"计量
- 需进行费用构成分析（人/材/机/管/利）

### 🟡 变化4：市场化定价导向
- 最高投标限价：结合人工/材料/机械市场价格信息编制
- 投标报价：可依据企业定额和市场价格信息**自主报价**
- 影响：不再完全依赖政府定额，企业自主报价能力成为核心竞争力

## 新疆2026-07-01执行通知要点

| 内容 | 要求 |
|------|------|
| 适用范围 | 新建/扩建/改建的房屋建筑和市政基础设施工程 |
| 过渡期 | 2026-07-01前已发布招标文件的，按原规定执行 |
| 危大工程 | 深基坑/高支模等措施费列入施工技术专项措施项目暂估价 |
| 防尘 | 建筑垃圾运输处置费单独列项 |
| 费率调整 | 企业管理费执行调整后费率（详见通知附件1） |

## 实操影响清单

| 造价环节 | 影响 | 行动建议 |
|---------|------|---------|
| 清单编制 | 模板→分部分项，措施项目以"项"计量 | 更新模板，调整清单结构 |
| 组价 | 规费不再单列，需调整综合单价分析表 | 更新Excel模板，修改取费程序 |
| 投标报价 | 可自主报价，企业定额价值上升 | 建立企业定额库 |
| 最高投标限价 | 必须依据市场价格 | 加强信息价采集 |
| 结算 | 过程结算正式写入国标 | 每期做过程结算 |
| 过渡期项目 | 7月1日前已签约项目按旧标准 | 区分新旧项目，避免混淆 |
