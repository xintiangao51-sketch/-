# KC-008 GB/T 50856-2024 通用安装工程工程量计算标准

GB/T 50500-2024 新清单计价标准的配套工程量计算标准之一（共9本同步实施）。适用于通用安装工程工程量清单项目设置与计算规则。

## 基本信息
| 项目 | 内容 |
|------|------|
| 标准号 | GB/T 50856-2024（推荐性） |
| 名称 | 通用安装工程工程量计算标准 |
| 性质 | 取代 GB 50856-2013，由强制性转为推荐性 |
| 页数 | 195 页 |
| 原文位置 | 知识库根目录 PDF（⚠纯扫描件，无文本层；PDF书签为无意义页码标签） |
| 配套 | GB/T 50500-2024 清单计价标准，2025-09-01全国施行/新疆2026-07-01施行 |

## 与2013版关系（⚠待OCR核实条款差异）
- 性质由强制性 GB → 推荐性 GB/T
- 与新清单计价标准的"模板工程划入分部分项""措施项目以项计量""规费不单列"等变化对应的工程量计算口径调整
- 各专业附录清单项目编码与计算规则的更新/增减

## 规范结构框架（通用安装工程通行框架，⚠待OCR核实）
标准通常含总则、术语、基本规定、工程量清单编制及按专业划分的附录（机械设备/热力设备/静置设备/电气设备/建筑智能化/自动化仪表/通风空调/工业管道/给排水采暖燃气/消防/通信网络/刷油防腐蚀绝热等）。⚠ 各附录准确编号与条文以OCR原文核实为准，正文条款严禁引用未核实内容。
