# 新疆各地州人工价格汇总

---

## 一、人工单价体系说明

### 1.1 人工单价构成

```
人工单价 = 基本工资 + 工资性补贴 + 辅助工资 + 职工福利费 + 劳动保护费
```

### 1.2 人工分类

| 工种分类 | 说明 | 适用范围 |
|---|---|---|
| 一类工 | 高级技工、特种作业人员 | 特殊工艺、专业技术要求高的工序 |
| 二类工 | 普通技术工人 | 建筑安装工程主体施工，最常用 |
| 三类工 | 普通工、壮工 | 土石方、搬运等辅助工作 |

### 1.3 新疆地区调整规则

1. **人工调差只计税金**：人工单价调整部分只计取税金，不计取管理费、利润、规费
2. **调差公式**：调差金额 =（发布单价 - 定额单价）× 定额工日数量
3. **调整周期**：原则上每半年调整一次，特殊情况随时调整
4. **调整依据**：新疆住建厅发布的人工单价调整文件

## 三、人工单价历史走势

### 3.1 近年乌鲁木齐二类工单价变化

| 年份 | 二类工单价（元/工日） | 年涨幅 | 备注 |
|---|---|---|---|

| 2026（预测） | 150~155 | +1.4%~3.4% | 预计平稳上涨 |

### 3.2 价格走势分析
- **长期趋势**：人工单价持续上涨，年均涨幅约4%~6%
- **近年特点**：涨幅有所收窄，趋于平稳
- **地区差异**：南北疆差异明显，最高与最低相差约25%
- **影响因素**：
  1. 经济发展水平
  2. 劳动力供给情况
  3. 物价水平
  4. 政策调整
  5. 季节性因素（冬季停工）

## 六、人工价格风险预警

### 6.1 价格波动预警阈值

| 波动幅度 | 预警等级 | 应对措施 |
|---|---|---|
| ±3%以内 | 正常 | 不调整，风险自留 |
| ±3%~5% | ⚠ 关注 | 密切关注市场变化 |
| ±5%以上 | 🔴 预警 | 启动价格调差程序，准备索赔资料 |

### 6.2 人工价格风险应对

| 风险情况 | 应对策略 |
|---|---|
| 合同约定可调价 | 按合同约定及时申请调整 |
| 合同约定固定价 | 投标报价时充分考虑涨价风险 |
| 工期较长项目 | 约定人工价格调差机制 |
| 短期项目 | 风险自留，报价时考虑涨价系数 |

## 七、人工单价使用说明

### 7.1 定额套用说明
1. 定额中的人工工日按8小时工作制计算
2. 加班加点按规定另计加班工资
3. 夜班施工增加夜班补贴

### 7.2 计算示例

**示例：乌鲁木齐某工程，二类工100工日，人工单价148元/工日**
```
定额人工费 = 100工日 × 148元/工日 = 14,800元

如冬季施工（降效系数1.15）：
冬施人工增加费 = 14,800元 × 15% = 2,220元
冬施人工费合计 = 14,800 + 2,220 = 17,020元

如高原施工（海拔2500m，降效系数1.20）：
高原人工增加费 = 14,800元 × 20% = 2,960元
高原人工费合计 = 14,800 + 2,960 = 17,760元
```
