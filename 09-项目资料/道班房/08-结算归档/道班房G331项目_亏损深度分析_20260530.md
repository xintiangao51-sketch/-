## 一、核心结论

```
合同含税价: 24,708,076 元（2471万，3%增值税）
承包方:     新疆富年建筑工程有限公司（负责人：高治国）
甲方:       湖南建投交通建设有限公司 G331二标项目部
地点:       阿勒泰可可托海山区（K248+500 / K284+800）
工期:       2024.8.16 → 2026.6.30（有效施工＜8个月，冬休7个月）
状态:       主体完工，剩余工程未施作，已出现亏损，结算争议中
```

**亏损根因用一句话：甲方指定高价混凝土（1030 vs 清单994元/m³）+ 甲方结算量小于实际完成量 + 抢工/冬施/山区运输三重超支。**

## 二、亏损原因拆解（按重要性排序）

### 🔴 P0 — 指定混凝土单价倒挂（最大亏损源）

| 项目 | 清单综合均价 | 实际采购价（中建交通三标拌合站） | 价差 |
|------|------------|---------------------------|------|
| 混凝土 | 994.36 元/m³ | 1,030 元/m³ | **+35.64 元/m³** |

- 甲方为抢工指定使用三标拌合站混凝土，单价远超清单成本
- 亏损报告中明确表述：**"该情况已严重导致我部实际成本的增加，是我部目前成本测算下亏损的主要因素"**
- 新增配套：罐车(4台196天24.5万已付?)、天泵(75,000元/月×3月=22.5万)、发电机组

### 🔴 P0 — 工程量计量差异（甲方少计）

亏损报告明确指出：
> "我部实际完成的工程量与甲方在2025年12月结算计价表中统计的实际完成量存在**巨大出入**，尤其是在土石方开挖、外墙保温、混凝土浇筑等核心分项工程上"

- K284基底承载力不足：增加开挖换填方量 + 混凝土增加**312.35m³**（变更签证）
- 清单外新增：室内吊顶工程、K284增加基础挖深
- 甲方12月结算计价表 vs 实际完成——双方量差未解决

### 🟡 P1 — 抢工措施费

- 2025年5月16日复工即收到抢工指令
- 新增人员进场路费、加班工资、周转材料额外投入
- 新增罐车、天泵、发电机组 —— 均甲方指令

### 🟡 P1 — 冬季施工额外成本

- 2024.10.15~2025.5.15 冬休（山区气温零下）
- 冬施措施费属于额外支出，清单未含
- 2024年冬休前抢工 + 2025年复工抢工，两轮

### 🟢 P2 — 山区运输超支

- 可可托海山区，道路交通运输不便
- 材料运输、机械进退场费用远超正常水平
- 运输费欠付60,500元（衣慢别克）

## 六、亏损报告中甲方的结算方案与诉求

### 6.1 两步走结算方案

**第一步（紧急）：**
- 暂按清单总价 **2,470.81万元** 办理中期结算
- 按 **85%** 比例支付 = **2,100万元**
- 优先代付农民工工资 **160万**
- 支付机械租赁费 **90万**

**第二步（2026.5复测后）：**
- 甲方牵头，监理+我方三方复测
- 土石方按实际开挖体积计算（含放坡/排水沟/换填）
- K284变更混凝土312.35m³全额计取
- 机电安装因材料优化/功能升级的差异调整

### 6.2 成本补偿诉求

1. **抢工措施费**：新增工人路费/加班工资/周转材料
2. **冬季施工措施费**：额外支出
3. **山区运输及场地处理补贴**
4. **混凝土单价修正**：从994→1030元/m³
5. **不均衡报价项调整**：实际发生与市场单价偏差大的项目

### 6.3 最终承诺

- 2026年6月30日前完成全部剩余工程并通过竣工交验
- 积极配合复测、审计、结算

## 七、鼎梁筑视角的风险评估

### 7.1 亏损定性

| 维度 | 判断 |
|------|------|
| 是否亏损 | **已确认亏损**（报告标题即含"亏损"二字） |
| 亏损金额 | 待定（需复测确定量差+价差补偿到位后折算） |
| 亏损主因 | 甲方指定高价混凝土 + 计量差异 + 抢工冬施 |
| 可挽回性 | 部分可挽回——核心看甲方是否认可量差和补偿 |

### 7.2 风险点

| 风险 | 等级 | 说明 |
|------|------|------|
| 农民工上访风险 | 🔴 | 欠付160.8万，年关压力大 |
| 机械费欠款 | 🔴 | 111.4万，涉及多人，李富关一人59.8万 |
| 甲方不接受量差 | 🟡 | 12月结算已出现"巨大出入"，复测是关键 |
| 混凝土价差不补偿 | 🟡 | 甲方指定的，按理应补偿，但需确认 |
| 剩余材料140万 | 🟢 | 已购在库，复工即可用，无新增成本 |
| 2026复测时间 | 🟢 | 5月中旬已到，复测应可推进 |

### 7.3 建议策略（鼎梁筑视角）

1. **短期（本周）：**
   - 催甲方尽快启动复测（5月中旬已到）
   - 优先解决农民工160万——用工资支付保障条例施压
   
2. **中期（复测阶段）：**
   - 带图纸+施工记录+影像资料参与复测
   - 重点守住：土石方量/K284变更312m³/混凝土价差
   
3. **策略建议：**
   - 混凝土价差是有力的谈判筹码——甲方指定供应商，非我方选择
   - K284变更有签证支撑，312.35m³必须全额
   - 不平衡报价调整属于合理诉求，但需要市场询价佐证
