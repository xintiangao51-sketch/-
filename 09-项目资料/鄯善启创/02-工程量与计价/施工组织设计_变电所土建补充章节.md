# 变电所土建工程施工方案补充章节

## （新疆启创电子电极箔建设项目·220KV变电站土建部分）

> 本章为施工组织设计的补充章节，专门针对220KV变电站土建工程。
> 变电站设备购置及安装调试不在本章范围内。

## 第十二章 220KV变电站土建施工方案

### 12.1 工程概况

#### 12.1.1 建设规模

| 项目 | 内容 |
|------|------|
| 变电站等级 | 220KV |
| 投标总价 | ¥8,368.11万元 |
| 其中设备购置 | ¥6,642.74万元（79.4%，含税13%） |
| 其中建安及其他 | ¥1,725.37万元（20.6%） |
| 建安主要内容 | 控制楼、主变基础、GIS基础、构支架、电缆沟、场区道路围墙、接地网 |

#### 12.1.2 土建施工范围

1. **控制楼**：框架结构，约600㎡，含建筑、结构、给排水、暖通、照明及装修
2. **主变压器基础**：4组，大体积混凝土基础，含油池及防火墙
3. **GIS设备基础**：2座，精度要求高（水平度≤2mm/m）
4. **构支架基础及安装**：钢构支架Q355B，含避雷针、母线构架
5. **电缆沟**：钢筋砼结构，1.2m×1.2m，约350m
6. **场区道路**：C30砼路面，约2500㎡
7. **围墙及大门**：砖砌+铁艺围栏，约500m
8. **接地网**：铜包钢接地系统，约600m
9. **事故油池**：钢筋砼地下构筑物
10. **站区给排水**：生活给水、消防给水、排水系统

### 12.2 施工总体部署

#### 12.2.1 施工原则

变电所土建按照**"先地下后地上、先深后浅、先主体后围护"**原则组织。

关键线路：控制楼基础→控制楼主体→设备基础→构支架安装→电缆沟→场区道路→围墙→接地网→竣工验收。

#### 12.2.2 施工流向

```
场地平整→控制楼基础→控制楼主体结构(含装修)
         →主变基础(4组同步)→油池及防火墙
         →GIS基础(2座同步)
         →构支架基础→钢构支架安装
         →电缆沟施工
         →场区道路砼浇筑
         →围墙砌筑及铁艺安装
         →接地网敷设(与基础同步预埋)
         →站区给排水→零星收尾→验收
```

#### 12.2.3 与电气安装的配合

- 基础施工阶段：预埋电气接地引上线、电缆保护管
- 构支架安装后：配合电气专业进行设备安装就位
- 电缆沟完成后：配合电缆敷设
- 控制楼装修后：配合电气二次设备安装

### 12.3 主要施工方案

#### 12.3.1 控制楼施工

**结构形式**：现浇钢筋混凝土框架结构，建筑面积约600㎡，单层/局部两层。

**基础工程**：
- 柱下独立基础C30，基础埋深≥1.5m（冰冻线以下）
- 基础防腐：刷环氧煤沥青涂层（土壤腐蚀性防护）
- 防雷接地引下线预埋：柱内≥Φ16主筋焊接引出

**主体结构**：
- 框架柱C30，框架梁C30，现浇板C30
- 填充墙：加气混凝土砌块，M5混合砂浆
- 屋面：SBS防水+挤塑板保温，有组织排水

**装修标准**：
- 地面：防滑地砖（控制室、办公室）/ 水泥砂浆（备品间）
- 内墙：无机涂料
- 外墙：真石漆
- 吊顶：矿棉板（控制室）/ 铝合金条板（走廊）

#### 12.3.2 主变压器基础施工

**工程特点**：
- 4组主变基础，单组尺寸约8m×6m×2.5m（预估）
- 大体积混凝土，单组混凝土量约120m³
- 含事故油池（钢筋砼地下构筑物，有效容积≥单台主变油量的60%）
- 防火墙：钢筋砼墙体，高度≥6m

**施工要点**：

1. **基础开挖**：机械大开挖，放坡1:0.5，基底预留300mm人工清底
2. **垫层**：C15混凝土100mm厚，每边宽出基础100mm
3. **钢筋**：HRB400E，双层双向，保护层40mm（有腐蚀性地基）
4. **预埋件**：
   - 主变基础顶面预埋钢板（20mm厚），水平度≤2mm/m
   - 接地引上线Φ16圆钢不少于2处/台
   - 电缆保护管预埋（Φ100-Φ150镀锌钢管）
5. **混凝土**：C30商品混凝土，掺抗裂防水剂
6. **大体积温控**：
   - 入模温度≤28℃（鄯善高温季节需夜间浇筑+冰水拌合）
   - 冷却水管：DN32钢管，间距1.0m，通循环水
   - 测温监控：预埋热电偶，前3天每2h一次
   - 养护：覆膜+土工布保湿≥14天
7. **油池**：底板及池壁C30P8抗渗混凝土，内壁防腐处理（环氧煤沥青）

#### 12.3.3 GIS设备基础施工

**精度要求**：
- 基础顶面水平度：≤2mm/m（全长度）
- 预埋件位置偏差：≤±5mm
- 相邻基础高差：≤2mm

**施工方法**：

1. 采用二次灌浆法施工：
   - 第一次浇筑至基础顶面以下50mm（预留灌浆层）
   - 预埋调平螺栓/垫铁，精确调平
   - 第二次采用C40无收缩灌浆料灌注50mm找平层
2. 模板：定型钢模板，确保表面平整
3. 测量：采用全站仪+精密水准仪，每个基础不少于4点复核
4. 养护：灌浆后覆膜保湿养护≥7天

#### 12.3.4 构支架施工

**钢结构构支架**：Q355B，含：
- 进线构架（门型构架）
- 母线构架
- 设备支架（断路器、隔离开关、CT、PT等）
- 避雷针（独立式，高度≥30m）

**施工流程**：
```
基础施工(预埋地脚螺栓)→构支架工厂加工→热镀锌防腐→
运输进场→现场拼装→吊装就位→螺栓紧固→二次灌浆→
避雷针安装→整体调整→验收
```

**吊装方案**：
- 主吊：25t汽车吊
- 构架柱分段吊装（超过12m分两段）
- 构架梁整体吊装
- 避雷针分段组装后整体起吊（需编制专项方案）

**质量控制**：
- 地脚螺栓预埋：采用定位模板固定，浇筑前后两次复核
- 垂直度：用两台经纬仪正交方向观测，偏差≤H/1000且≤25mm
- 热镀锌：锌层厚度≥86μm（当地大气腐蚀性较强）
- 高强螺栓：10.9S级，扭矩法施工，初拧50%终拧100%

#### 12.3.5 电缆沟施工

**结构**：钢筋砼沟道，净空1.2m×1.2m，壁厚150mm，底板厚200mm。

**施工要点**：
1. 沟槽开挖：机械开挖，人工修整，排水坡度1%
2. 垫层：C15混凝土100mm厚
3. 底板及沟壁：一次浇筑成型，C30P6抗渗混凝土
4. 模板：沟壁内侧采用定型钢模，保证内壁平整
5. 预埋件：电缆支架预埋铁件（间距800mm）、接地扁钢通长敷设
6. 盖板：预制钢筋砼盖板C30，配吊环，场外预制场内安装
7. 防水：沟壁外侧刷冷底子油+热沥青两道
8. 排水：沟底设排水坡度≥0.5%，低点设集水坑

#### 12.3.6 场区道路及围墙

**道路**：
- 结构：C30混凝土路面，厚200mm
- 基层：300mm级配砂石（戈壁料）
- 路宽：主路4m，消防通道4m
- 施工：机械摊铺+振动梁整平+抹光机收面
- 切缝：间距4-6m，沥青灌缝

**围墙**：
- 基础：砖砌大放脚基础，埋深≥1.2m
- 墙体：M5水泥砂浆砌MU10砖，厚240mm，高2.5m
- 压顶：C20混凝土压顶
- 铁艺围栏：热镀锌方管+钢板网，高1.5m（围墙以上部分）
- 大门：电动推拉门（宽6m）+人员通道小门

#### 12.3.7 接地网施工

**系统要求**：
- 接地电阻：≤0.5Ω（220KV变电站要求）
- 材质：铜包钢（防腐性能优于镀锌扁钢，鄯善戈壁土壤腐蚀性强）

**施工要点**：
1. 水平接地体：-40×4铜包钢，埋深≥0.8m，网格间距≤10m
2. 垂直接地极：Φ20铜包钢，长2.5m，间距≥5m
3. 焊接：放热焊接（热熔焊），接头防腐处理
4. 接地引上线：Φ16圆钢，每台设备不少于1处
5. 与主接地网连接：≥2处可靠连接
6. 测试：接地电阻测试（如不满足≤0.5Ω需增加接地极或降阻剂）

### 12.6 与主厂区施工的协调

| 协调事项 | 处理方式 |
|----------|----------|
| 混凝土供应 | 与主厂区共用商品混凝土供应，高峰期错峰浇筑 |
| 钢筋加工 | 共用主厂区B/C区钢筋加工棚 |
| 施工用电 | 从主厂区一级配电柜引出专线 |
| 施工用水 | 从主厂区施工用水管网接入 |
| 测量控制 | 采用统一的场区坐标控制网 |
| 临时道路 | 与主厂区施工环路衔接 |
| 土方平衡 | 变电所区域土方纳入全场土方统一调配 |

> ⚠️ 变电所土建施工方案需设计图纸到位后细化。构支架吊装、避雷针安装超过30m需编制专项施工方案并报审。接地电阻不满足要求时需补充降阻方案。
