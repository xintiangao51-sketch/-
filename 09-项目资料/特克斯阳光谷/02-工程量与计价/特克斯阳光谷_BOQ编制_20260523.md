# 特克斯阳光谷 全专业工程量清单编制 — 任务总结

**时间**: 2026-05-23 10:49–11:14

## 任务目标
为用户「众汇·阳光谷(南区)建设项目」编制全专业安装工程量清单，含人材机单价分离，采用新疆伊犁地区定额。

## 环境搭建
- 安装 Python 3.12.8（华为云镜像）
- 安装 pdfplumber / pandas / openpyxl
- 确认 5 个造价专业技能已部署（drawing-analyzer / zaojia-dinge / cwicr-cost-calculator / cost-prediction / cn-transport-cost）
- 发现 zaojia-dinge、cost-prediction、weide-cost-assistant 目录权限锁定

## 数据处理
- 读取 67 个 BOQ 文件（6 专业 × 多楼栋）
- 读取 7 个材料采购/询价表
- PDF 图纸为扫描件（图片型），无法直接文本提取
- 从现有招标清单提取 2,375 条工程量数据
- 匹配材料价格数据库（100+ 种材料）
- 按新疆伊犁地区费率计算人材机分解
