# 阳安二线项目资料

## 项目概况
阳安二线铁路工程，中铁二十五局承建，涉及隧道、桥梁、路基等工程。

## 内容概览
- 施工组织设计（动态施组）
- 隧道进尺统计（新缯西河隧道、余家坡隧道）
- 拌合站超标分析
- 房屋拆迁统计
- 激励约束考核办法
- 劳动竞赛评比
- 征地拆迁资料
- 安全质量资料
