# 📁 09-项目资料

> 项目资料集中管理区。95 个项目按名称扁平排列，Dataview 动态分年度/分地区/分状态。

---

## 📊 动态项目总览

### 按状态分组

```dataview
TABLE project_name as "项目", contract_amount as "合同额(万)", region as "地区"
FROM "09-项目资料"
WHERE file.name != "项目资料-总览"
FLATTEN file.tags as tag
WHERE contains(tag, "项目")
GROUP BY status
SORT status ASC
```

### 按地区分组（动态）

```dataview
TABLE length(rows) as "项目数"
FROM "09-项目资料"
WHERE region
GROUP BY region
SORT length(rows) DESC
```

### 活跃项目一览

```dataview
TABLE project_name as "项目", status as "状态", contract_amount as "合同额(万)"
FROM "09-项目资料"
WHERE type = "index" AND contains(tags, "项目") AND status != "归档"
SORT file.ctime DESC
```
