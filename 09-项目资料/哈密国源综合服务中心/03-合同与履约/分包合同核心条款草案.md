# 方案B分包合同核心条款草案

> 承接范围：建筑工程 + 装饰工程 + 消防水工程
> 分包额估算：3,615万元 | 预估利润：392万元 | 利润率：10.8%

## 一、合同当事人

| 项 | 内容 |
|---|---|
| 总包方（甲方） | 广西建工集团第四建筑工程有限责任公司 |
| 分包方（乙方） | （你的公司名称） |
| 工程名称 | 国源电力哈密煤电公司综合能源集控服务中心工程施工 |
| 招标编号 | CEZB260002551 |

## 二、分包范围

### 2.1 承包范围

乙方承包以下专业的施工（包工包料，不含甲方指定设备）：

1. **建筑工程**（含地下工程、主体结构、二次结构、屋面、防水等）
   - 投标报价：2,262万元
   - 分包价：2,035.8万元（扣10%管理费）
   - 建筑面积约19,300㎡，折合单价1,055元/㎡

2. **装饰工程**（含室内外装修、楼地面、墙面、天棚等）
   - 投标报价：1,574万元
   - 分包价：1,416.6万元（扣10%管理费）
   - 折合单价734元/㎡

3. **消防水工程**（含消火栓系统、喷淋系统、消防水泵房等）
   - 投标报价：180万元
   - 分包价：162万元（扣10%管理费）

### 2.2 不含范围（由甲方自行采购/施工）

- 全部设备采购（配电箱、开关柜、水泵、热泵、电梯等，合计约393万元）
- 专业工程暂估价项目（迁改、泛光照明、智能会务、室内弱电、抗震支架、净水设备，合计480万元）
- 电气/给排水/通风空调/弱电/消防报警/人防/太阳能/室外工程等安装专业
- 暂列金额400万元

### 2.3 材料供应方式

| 类别 | 供应方式 | 备注 |
|---|---|---|
| 钢材（钢筋/型钢） | 乙方自购 | 按哈密信息价+调差 |
| 商品混凝土 | 乙方自购 | 按哈密信息价+调差 |
| 水泥/砂石/砌块 | 乙方自购 | 按哈密信息价+调差 |
| 装饰主材（石材/瓷砖/涂料等） | 乙方自购 | 按招标文件品牌要求 |
| 消防管道/阀门/喷头 | 乙方自购 | 按设备表品牌要求 |
| 模板/脚手架/周转材料 | 乙方自供 | 含在扩大劳务单价中 |

## 三、合同价格

### 3.1 合同方式

**固定综合单价合同**，按实际完成工程量结算。

### 3.2 综合单价组成

综合单价包含：人工费、材料费、机械费、管理费、利润、规费、税金，以及以下全部费用：

- 冬季施工措施费（冬休期11.30-3.1约3个月）
- 偏远地区附加费
- 创优措施费（满足新疆优质工程评审条件）
- 临时设施费（含临建搭拆、临时用水用电接口）
- 材料设备场内二次搬运费
- 已完工程保护费
- 安全文明施工费

### 3.3 材料价格调差（核心条款）

1. 主要材料（钢材、商品混凝土、水泥、砂石）价格参照**哈密市现行建设工程造价信息**执行
2. 施工期间材料价格波动超过**±5%**时，超出部分按实调整
3. 调差周期：按季度，以哈密市造价信息发布的季度均价为基准
4. 信息价缺失的材料，由乙方提供3家以上供应商报价，经甲方审核确认后执行
5. 乙方不得以材料价格上涨为由拖延工期

### 3.4 暂估价项目

本项目暂估价（480万元）不在乙方承包范围内，由甲方自行组织采购。乙方不承担暂估价格波动风险。

## 四、付款条款

### 4.1 进度款

| 节点 | 付款比例 | 说明 |
|---|---|---|
| 每月进度款 | 已完工程量的**75%** | 与甲方收到业主付款同步 |
| 竣工验收 | 累计付至**90%** | 验收合格后30日内 |
| 结算审计 | 累计付至**95%** | 审计完成后60日内 |
| 质保金 | **5%** | 质保期满后30日内返还 |

### 4.2 付款前提

甲方在收到业主对应进度款后7个工作日内，向乙方支付同比例进度款。甲方不得以业主未付款为由拖欠乙方进度款超过30日。

### 4.3 质保金

- 质保金比例为结算总价的5%
- 质保期：自竣工验收合格之日起2年（防水5年）
- 质保期满且无未解决质量缺陷后30日内一次性无息返还

## 五、工期

### 5.1 总工期

合同签订后**22个月**（含冬休期11.30-3.1约3个月），实际施工期约15个月。

### 5.2 里程碑节点

| 节点 | 工期 | 产值占比 |
|---|---|---|
| 临建+基础 | 2个月 | 16% |
| 地下室主体 | 3个月 | 32% |
| 地上主体结构 | 6个月 | 28% |
| 二次结构+砌筑 | 3个月 | 8% |
| 装饰装修 | 5个月 | 12% |
| 消防水安装 | 随主体穿插 | 4% |

### 5.3 冬休期

冬休期（暂定11月30日至次年3月1日）计入总工期。冬休期间：

- 乙方安排留守人员不少于5人，负责现场看护、成品保护、材料保管
- 留守人员费用由乙方承担，已含在合同总价中
- 冬休期结束后7日内复工，复工后15日内恢复至冬休前施工强度

## 六、质量与创优

### 6.1 质量标准

合格，并满足新疆维吾尔自治区优质工程评审条件。

### 6.2 创优责任

- 乙方须配合甲方完成创优所需的全部施工资料、过程记录、细部处理
- 创优专项费用50万元已在甲方其他费用清单中列支，由甲方统筹安排
- 因乙方施工质量原因导致未达创优标准的，乙方承担返工费用
- 因非乙方原因导致未创优的，不扣减乙方合同价款

## 七、安全文明施工

### 7.1 安全文明施工费

按建筑安装工程费（不含设备费）的**2.5%**计取，已含在合同总价中。乙方须按甲方及监理要求使用，专款专用。

### 7.2 智慧工地

智慧工地基础配置费由甲方统一投入，乙方配合安装和使用。

## 八、变更与签证

### 8.1 设计变更

设计变更由甲方/监理发出书面指示，乙方在收到指示后7日内提交变更报价书。

### 8.2 变更估价原则

1. 合同中有相同项目的，按合同综合单价执行
2. 合同中有类似项目的，参照合同综合单价执行
3. 合同中没有的，按以下原则：
   - 依据新疆2024版定额及哈密市当期信息价编制
   - 报价浮动率 L = (1 - 中标价/招标控制价) x 100%
   - 变更综合单价 = 定额综合单价 x (1 - L)

### 8.3 现场签证

乙方在签证事件发生后**48小时内**提交签证申请，逾期不补。

## 九、违约责任

### 9.1 工期违约

- 逾期竣工：每逾期一天，按合同总价的**0.05%**支付违约金
- 违约金上限：合同总价的**5%**

### 9.2 质量违约

- 不合格工程无条件返工，费用由乙方承担
- 因质量缺陷造成损失的，乙方承担全部赔偿责任

### 9.3 甲方违约

- 逾期付款超过30日的，按同期银行贷款利率支付利息
- 甲方原因导致停工超过15日的，补偿乙方现场人员工资和机械停滞费

## 十、争议解决

双方协商解决；协商不成的，向工程所在地人民法院提起诉讼。

## 十一、特别约定

1. **农民工工资保障**：乙方须开立农民工工资专用账户，确保农民工工资按月足额发放。如因农民工工资纠纷造成甲方损失，乙方承担全部赔偿责任。
2. **技术负责人要求**：乙方拟派技术负责人须持有一级建造师证书。
3. **禁止转包**：乙方不得将承包工程转包给第三方。经甲方书面同意，可将部分劳务作业分包给具备资质的劳务企业。
4. **保险**：乙方须为施工现场人员购买意外伤害保险，为施工设备购买财产保险。
5. **审计配合**：乙方须配合甲方完成国家能源集团内部审计及外部审计。
