# 方案B分包谈判攻防清单

> 承接：建筑+装饰+消防水 | 分包额3,615万 | 利润392万 | 利润率10.8%

## 一、谈判总体定位

### 你的底牌

| 项 | 内容 |
|---|---|
| 中标单位 | 广西建工集团第四建筑工程有限责任公司（桂林） |
| 中标价 | 7,481万元 |
| 中标单位总部 | 桂林，距哈密约4,000公里 |
| 中标单位现状 | 3年营收从75亿降至31亿，资产负债率83-87% |
| 他们的痛点 | 跨省施工、当地无团队、急需本地合作伙伴 |
| 你的优势 | 本地团队、熟悉哈密市场、有新疆施工经验 |

### 广西四建为什么需要你

1. **距离**：总部在桂林，哈密项目是典型的"飞地工程"，管理成本高
2. **资金压力**：营收腰斩、负债高企，现金流紧张，需要本地分包分担垫资
3. **人员**：拟派项目经理蓝晓华（桂林人），技术负责人刘伟国（内江项目调来），核心团队不熟悉新疆
4. **合规**：招标文件明确不允许转包，但可以"分包"，需要有资质的分包商
5. **工期**：22个月含冬休期，实际施工期约15个月，工期紧

### 你能给他们什么

- 本地劳务团队，降低异地管理成本
- 熟悉哈密材料市场，拿到更好的材料价
- 分担约200万垫资压力
- 配合他们完成创优目标
- 处理当地政府关系和审批手续

## 二、逐条攻防策略

### 条款1：分包范围

| 你的主张 | 对方可能要求 | 攻防要点 |
|---|---|---|
| 只接建筑+装饰+消防水 | 加压给排水、电气 | 坚决不接安装专业。理由：安装类设备占比高、利润薄甚至亏损。给排水-18%、消防报警-8.5%、通风空调-87.7%，接了就是白干倒贴 |
| 材料自购权 | 总包统购 | 坚持材料自购，这是你的利润来源之一。统购你就只剩劳务费，利润率降到3%以下 |
| 不含设备采购 | 含设备 | 设备采购393万+暂估价480万，资金占用大、风险高，留给总包 |

**谈判话术**：我们只接我们最擅长的土建和装饰，安装部分你们找专业队伍更合适。消防水虽然量不大，但跟我们土建有交叉，一起做效率更高。

### 条款2：合同价格

| 你的主张 | 对方可能要求 | 攻防要点 |
|---|---|---|
| 固定综合单价合同 | 总价包干 | 绝不接受总价包干。本项目22个月含冬休，材料价波动、设计变更风险大，总价包干会把所有风险转嫁给你 |
| 扣10%管理费 | 扣12-15% | 底线10%，超过10%就没有利润。报价时已经按10%预留，12%以上直接谈不下去 |
| 材料调差 | 不调差 | 这条是命脉。哈密钢材信息价波动可达20%+/年，不调差你扛不住。争取±5%以内不调，超出按实调整 |
| 冬施费单独列项 | 含在综合单价中 | 冬施费约150-200万，含在综合单价里会被摊薄。争取按实际发生签证 |

**关键计算**：
- 分包价 = 中标价 x 90% = 3,615万
- 你的成本（扩大劳务）= 约3,223万（含主材）
- 利润空间 = 392万 = 10.8%
- 如果管理费涨到12%：利润降到292万 = 8.1%
- 如果管理费涨到15%：利润降到151万 = 4.2%（风险不可接受）

### 条款3：付款条款

| 你的主张 | 对方可能要求 | 攻防要点 |
|---|---|---|
| 月进度75% | 月进度70% | 与业主付款（80%）挂钩，75%是合理比例。你的垫资约200万，如果只付70%，垫资涨到350万 |
| 竣工付至90% | 竣工付至85% | 竣工验收是大节点，必须拿到大头。85%意味着还有15%约540万压在后面 |
| 质保金5% | 质保金8-10% | 5%是行业惯例。本项目总价3615万，每多1%质保金就多36万压2年 |
| 甲方收到业主款后7日内支付 | 甲方收到业主款后30日内 | 7日是底线，30日等于你要多垫1个月的款 |

**垫资测算**：
- 正常施工期垫资约200-274万
- 冬休期留守费用约200万
- 如果付款条件恶化（70%进度+30日延迟），垫资可能涨到400-500万
- 你的资金承受能力决定了谈判的回旋余地

### 条款4：暂估价风险

| 你的主张 | 对方可能要求 | 攻防要点 |
|---|---|---|
| 暂估价不在分包范围内 | 包含暂估价 | 480万暂估价（迁改290万+泛光50万+智能会务30万+室内弱电50万+抗震支架30万+净水30万）全部不接。这些是专业分包的活，你干不了也不该干 |
| 暂估价±10%风险由甲方承担 | 乙方承担 | 合同澄清已明确：暂估价超10%以内承包方承担，10%以上发包方承担。你作为分包方，根本不该碰暂估价 |

### 条款5：创优费用

| 你的主张 | 对方可能要求 | 攻防要点 |
|---|---|---|
| 50万创优费由甲方统筹 | 乙方承担 | 创优费已在其他费用清单中列支，由甲方统筹。你只配合施工质量达标，不承担创优经费 |
| 未创优不扣乙方价款 | 未创优扣减 | 合同条款约定"未达国优标准扣分项报价10%"，这是总包对业主的承诺。分包合同必须明确：因乙方质量原因未创优才扣，非乙方原因不扣 |

### 条款6：冬施费

| 你的主张 | 对方可能要求 | 攻防要点 |
|---|---|---|
| 冬施费按实际签证 | 含在总价中 | 澄清文件明确冬施费包含在投标报价中。但你作为分包，如果含在综合单价里，冬休期3个月没有产值，冬施费就被稀释了。争取按实际发生+签证 |
| 冬休期留守5人 | 冬休期留守10人+ | 留守5人足够（2人看场+2人材料保管+1人安全巡查），每多1人每月约8000元成本 |

## 三、谈判顺序建议

1. **先谈范围**（最核心）：锁定建筑+装饰+消防水，把安装和设备甩出去
2. **再谈价格**：管理费10%、材料调差±5%触发
3. **然后谈付款**：75%进度+5%质保金
4. **最后谈细节**：冬施费、创优、违约金等

## 四、你的谈判筹码使用顺序

| 阶段 | 筹码 | 释放方式 |
|---|---|---|
| 开场 | 本地经验+劳务团队 | 主动展示，建立"我们是最合适的本地合作伙伴"印象 |
| 范围谈判 | 愿意分担垫资 | "我们可以垫200万，前提是范围控制在土建装饰" |
| 价格谈判 | 材料采购优势 | "哈密当地材料我们比你们从桂林调配便宜15-20%" |
| 付款谈判 | 工期承诺 | "我们保证15个月实际施工期内完成，配合你们22个月总工期" |
| 收尾 | 长期合作意愿 | "这个项目做好了，后面国家能源集团的新疆项目可以继续合作" |

## 五、红线条款（不可让步）

1. 管理费不超过10%
2. 必须是单价合同，不接受总价包干
3. 材料调差条款必须写入
4. 不接安装类（电气/通风/弱电/消防报警/人防/太阳能）
5. 不接设备采购
6. 质保金不超过5%
7. 进度款不低于75%

## 六、可让步条款（用来交换红线）

1. 工期违约金可接受0.05%/天（上限5%）
2. 冬休期留守人数可谈
3. 结算审计时间可放宽到90天
4. 可承担部分临时设施建设
5. 可配合总包的审计和检查工作
