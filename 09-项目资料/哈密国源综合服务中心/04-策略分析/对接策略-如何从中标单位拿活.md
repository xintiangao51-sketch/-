## 第二章：对手画像 — 广西建工四建现在是什么状态

### 2.1 基本盘

| 维度 | 事实 | 对你的含义 |
|------|------|-----------|
| 资质 | 特级总承包，1,288 位建造师 | 管理能力没问题 |
| 业绩 | 1,702 项中标，0 不良记录 | 信用记录干净 |
| 新疆经验 | **零**，从未进过新疆 | ← **你的第一个切入点** |
| 哈密当地资源 | 无劳务库、无材料商、无政府关系 | ← **你的第二个切入点** |

### 2.2 🔴 财务真实状况

| 指标 | 数据 | 防线 |
|------|------|:------:|
| 负债率 | 88.54% | 🔴 已破 80% 警戒线 |
| 现金比率 | 0.08 | 🔴 每 100 元短期债务仅有 8 元现金 |
| 票据逾期 | 3.41 亿 | ⚠ 已发生信用事件 |
| 营收 | 腰斩（1317→650 亿） | 🔴 持续失血 |
| 2024 净亏损 | 8.25 亿 | 🔴 亏损运营 |

> **翻译成白话**：四建非常缺钱。7,480 万对这体量的公司是**救命项目**，不是锦上添花。他们会拼命保履约（因为丢不起国能项目），但**对下付款一定会拖**。

### 2.3 他们的分包习惯

- **模式**：包工不包料（清包工），材料总包买
- **付款**：**无预付款**，月进度结 80%
- **结算**：按实工程量计件
- **安全**：事故责任和损失**全部由劳务方承担**
- **招标**：通过广西建工集采平台发标，取"合理低报价"
- **管理**：公司管钱和招采，项目部管现场履约

## 第三章：切入点分析 — 你该打哪张牌

### 3.1 四建在这个项目上的四大痛点

| 痛点 | 严重程度 | 你能怎么帮 |
|------|:------:|-----------|
| 🤝 **不熟悉哈密** — 没本地劳务、没材料渠道、没政府关系 | 🔴 致命 | 提供当地资源、协助审批、介绍靠谱材料商 |
| 💰 **现金流极度紧张** — 没钱垫、想拖付 | 🔴 严重 | 接受分期付款（换取更高单价或锁定更多工程量） |
| ❄️ **没冬施经验** — 广西企业从未在 -20℃ 施工 | 🟡 重要 | 提供冬施方案、保温措施经验 |
| 📋 **暂估价/手续费用搞不定** — 迁改、弱电、创优等不专业 | 🟡 重要 | 承接暂估价包干、代办创优流程 |

### 3.2 推荐切入方向（按优先级排序）

#### 🥇 首选：设备材料供应

| 为什么 | 详情 |
|--------|------|
| **安全性** | 材料款相对劳务费容易收回——有材料留置权 |
| **体量** | 设备费 357 万 + 土建装修材料数千万 |
| **门槛** | 靠渠道和价格优势，不需要养工人 |
| **核心单品** | ① 空气源热泵 12 台 × 18 万 = 216 万（中宇品牌需询价）② 奥的斯电梯 4 台 × 22 万 = 88 万 ③ 三科配电箱柜 15 台 = 32 万 ④ 商品混凝土、钢材、装修材料 |

#### 🥈 次选：弱电智能化 + 暂估价打包

| 为什么 | 详情 |
|--------|------|
| **利润最高** | 弱电施工 56.55 万 + 室内弱电暂估 50 万 + 智能会务暂估 30 万 + 泛光暂估 50 万 = **可打包约 186 万** |

| **后续锁定** | 暂估价后续必招标，提前和四建谈好——你帮他们设计方案、做暂估认价，他们给你优先权 |

#### 🥉 第三：消防工程专业分包

| 为什么 | 详情 |

| **体量适中** | 消防水 116.4 万 + 消防报警 127.3 万 + 室外消防水 20.9 万 + 设备 9 万 = **约 274 万** |
| **专业壁垒** | 需消防资质，普通劳务队做不了 |

## 第四章：怎么谈 — 谈判策略

### 4.1 开局定调：你不是来抢活的，是来帮他们活下来的

> 不要一上来就谈分包价格。先谈**你能帮他们解决什么麻烦**。

**第一次接触的建议话术框架**：

```
"四建的领导，我知道这是你们第一次进新疆哈密。
这地方和内地不一样——8度抗震、冬季零下20、材料运费贵、
哈密当地建管流程有自己的一套。

我是哈密本地的，这几个事我可以帮你们搞定：
- 当地劳务组织（我手上有现成的班组）
- 材料本地采购（我知道谁的价格实、谁断供风险小）
- 冬施方案（新疆工地冬天不停工的套路我熟）
- 政府手续（建设局、消防、环保的人头我熟）

你们把精力放在主体结构上——那是你们的强项。
配套的事交给我，我帮你们省心。"
```

### 4.2 谈判筹码清单

| 你有的筹码 | 为什么四建在乎 |
|-----------|--------------|
| 哈密当地资源（劳务+材料+机械） | 他们从广西调人调机械成本高、周期长 |
| 政府关系（建设局/消防/环保/水利） | 7 项"以建设单位名义"的手续他们搞不定 |
| 冬施经验 | 22 个月跨 2 个冬季，广西队伍没见过 -20℃ |
| 资金容忍度（接受付款延迟） | 他们现金流极度紧张，这是**最大的筹码** |
| 暂估价/创优/手续代办能力 | 480 万暂估 + 92.8 万其他费用他们没经验做 |

### 4.3 把"付款容忍度"换成你要的东西

四建缺钱 → 你想拿活 → **用付款条件换工程量和单价**。

| 你让步 | 你换取 |
|--------|--------|
| 接受无预付款 | → 要求锁定更大工程量或更多专业 |
| 接受月结 80% 延迟至 90 天 | → 单价上浮 5%~8% |
| 愿意垫付人工费 3 个月 | → 要求签排他协议（指定专业只给你） |
| 接受背靠背付款（业主付四建、四建再付你） | → 降低四建管理费抽成比例 |

> ⚠ **底线**：无论怎么让步，**不签无限垫资条款**。合同里必须明确：垫资上限金额、最长垫资期限、逾期利息。

### 4.4 建议的接触路径

```
第一步：不要走广西建工集采平台投公开标
        → 那是纯比价，你打不过广西当地低价队伍

第二步：直接找到四建哈密项目筹备组负责人
        → 蓝晓华（项目经理）或四建新疆协调人
        → 通过哈密煤电公司（业主方）侧面了解四建筹备动态

第三步：以"当地配套服务方"身份接触
        → 不谈分包，先谈你能帮他们落地哈密
        → 建立信任后再谈具体的分包范围

第四步：促成"议标"或"竞争性谈判"
        → 不要在公开平台拼价格
        → 通过前期价值展示，让四建主动邀你报价
```

## 第五章：怎么报价 — 分项报价策略

### 5.1 报价前的核心判断：四建各专业有多少预算空间

以下金额从内部报价表反推，是**四建的合同收入线**，你的报价不能等于这个数——要给他们留管理费和利润。

| 专业 | 四建合同收入（万元） | 合理分包价区间（万元） | 四建留存率 |
|------|:------:|:------:|:------:|
| 建筑工程（主体） | 1,308.80 | 1,046~1,177 | 10%~20% |
| 装饰工程 | 983.31 | 786~885 | 10%~20% |
| 电气安装 | 336.51 | 269~303 | 10%~20% |
| 消防报警 | 127.33 | 102~115 | 10%~20% |
| 消防水 | 116.40 | 93~105 | 10%~20% |
| 通风空调（安装部分） | 195.74 | 157~176 | 10%~20% |
| 弱电及监控 | 56.55 | 45~51 | 10%~20% |
| 室外工程（全部） | 900.66 | 720~811 | 10%~20% |
| 设备供应 | 357.12 | **不得低于你进货价+运费+5%** | — |
| 暂估价（弱电/泛光/迁改/抗震/净水/会务） | 480.00 | 按实际成本+合理利润报价 | 5%~10%（管理费） |

### 5.2 报价策略：根据专业特征区别定价

| 策略类型 | 适用专业 | 报价定位 | 理由 |
|----------|----------|:------:|------|
| **技术溢价型** | 弱电智能化、消防、设备供应 | 可以偏高 5%~10% | 四建不专业，更看重技术可靠 |
| **服务捆绑型** | 室外配套 + 手续代办 | 中等偏上 | 你解决了他们的属地难题 |
| **规模换量型** | 主体劳务（如果争取到的话） | 贴近市场均价 | 量大但利润薄，靠规模赚钱 |
| **关系定价型** | 暂估价分包 | 中等 | 关键是锁定优先权，后续价可谈 |

### 5.4 报价红线

| 红线 | 内容 |
|:----:|------|
| ❌ | 不要按四建合同收入的 100% 报价——他们要赚管理费 |

| ❌ | 不要报"低于成本"的价来抢活——四建会怀疑你的履约能力 |
| ✅ | 报价留 3%~5% 谈判空间，让四建有"砍下来"的成就感 |

## 第七章：风险提示

### 7.1 最大的三个风险

| 风险 | 概率 | 影响 | 应对 |
|------|:------:|:------:|------|
| **四建拖欠付款** | 🔴 高 | 致命——垫付人工/材料款 6 个月以上可能拖垮你 | ①合同约定逾期利息（日万分之五）②分阶段控制垫资上限③材料尽量做甲指乙供（业主直接付材料款） |
| **安全责任全责条款** | 🟡 中 | 严重——四建合同模板约定安全事故全部由劳务方承担 | ①必须买足建筑意外险+第三者责任险②合同里加不可抗力条款③安全交底留书面痕迹 |
| **工程量/变更签证不认** | 🟡 中 | 可能亏本——超量部分四建以"没签补充协议"为由拒付 | ①每道工序前后拍照留证②变更指令必须有书面签字才施工③月结单累计超量预警 |

### 7.2 决定成败的五个判断

| # | 判断问题 | 如果你的答案是"否" |
|:--:|----------|------------------|

| 2 | 你和哈密建设局/消防队有直接关系吗？ | 主打技术型分包（弱电/消防），不要打手续代办牌 |

| 5 | 你在哈密有 30 人以上的稳定劳务班组吗？ | 只做专业分包，不要碰主体/装修劳务 |
