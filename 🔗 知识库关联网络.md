```
    │
    ├── 费用定额 → 管理费28% / 利润18% / 规费22% / 增值税9%

    │       │
    │       └──▶ 合同审查 [03] → 合同价款调整依据
    │
    ├── 消耗量定额 → 各专业工程的人工/材料/机械消耗标准
    │       │
    │       ├──▶ 工程量计算 [01] → 工程量计算规则
    │       │
    │       └──▶ 变更索赔 [01] → 变更估价依据
    │
    └── 信息价 → 各地州人工/材料指导价
            │
            └──▶ 材料价格 [01] → 市场价与信息价对比
```

## 七、快捷查询路径

### 场景1：做一个新项目的报价
```
① 查同类项目：→ [09] 项目资料 找到类似规模项目
② 选定额体系：→ [01-定额与计价] 确定适用定额
③ 查人工单价：→ [01-材料价格] 或 D:\信息价\
④ 取费标准：→ [01-取费与税金] 管理28%/利润18%/规费22%/税金9%
⑤ 合同模板：→ [03-建设合同模板] 选用合适范本
⑥ 参考策略：→ [05-投标编制/报价策略]
```

### 场景2：处理变更索赔
```
① 查合同条款：→ [03] 对应项目的合同文件
② 查变更时限：→ 卡片#004 变更索赔流程
③ 定额依据：→ [01-定额与计价] 变更估价
④ 参考案例：→ [09] 类似项目的变更案例
```

### 场景3：审计复核
```
① 拿报审清单：→ [09-项目资料/XX项目/02-工程量与计价]
② 对照定额：→ [01-定额与计价]
③ 检查单价：→ 卡片#016 偏差预警
④ 核实合同：→ [03-合同与法务]
⑤ 出审计报告：→ 卡片#013 造价工作台引擎
```
