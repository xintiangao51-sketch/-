## 与知识库协同关系

| 知识库模块 | 定位 | 侧重点 |
|-----------|------|--------|
| 知识库/造价知识库 | 专业知识提炼中台 | 知识提炼、标准化模板、Prompt体系、方法论 |
| 知识库/01-造价核心 | 项目实操数据 | 项目文件归档、图纸、清单、合同等原始资料 |
| 知识库/02-业务中台 | 业务处理中间层 | 计价、算量、材料价格、合同等业务模块 |
| 知识库/09-项目资料 | 项目档案 | 具体项目全过程资料归档 |

> **协同关系**：知识库 提供原始项目数据 → 造价知识库 提炼为标准化知识模板 → 共同支撑智能工作台运行
