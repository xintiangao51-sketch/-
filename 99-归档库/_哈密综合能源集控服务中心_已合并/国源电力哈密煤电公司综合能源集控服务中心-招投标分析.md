## 3. 项目建设背景

哈密煤电公司为国家能源集团全资三级子公司、国神公司全资子公司，也是国神公司最大的煤电一体化单位。

**现有运营项目：**

| 类别 | 项目/业务 |
|------|-----------|
| 矿井/矿山 | 大南湖一号矿井、大南湖二号露天矿 |
| 电厂 | 花园电厂、大南湖热电厂 |
| 新能源 | 新能源分公司 |
| 供热 | 承担哈密市区集中供热面积 856 万㎡，约占市区总供热面积三分之一 |

**建设目的：**

| 建设内容 | 目的 |
|----------|------|
| 新能源集控中心 | 按"区域集控、运维智能化"原则，提升新能源分公司智能化、智慧化管控运维水平，降低风电场、光伏场就地值班值守工作量 |
| 明珠供热调度中心 | 解决哈密煤电公司明珠供热在市区内无固定调度大厅和收费大厅的问题 |
| 综合能源集控服务中心 | 实现新能源全域集控、强化供热服务功能、推动资源集约共享 |

### 判断结论

该项目为 **地上 7 层 + 地下 1 层公共建筑/办公综合体**，含主体、装饰、室外水电暖、围墙栏杆、停车场等附属工程。

按新疆哈密地区、地下室 3500㎡、水暖电配套齐全、智能集控功能用房考虑，**中标单方约 3876 元/㎡ 属于偏合理区间**。

**后续重点关注：**
- 弱电智能化是否单独列项
- 集控大厅装修标准及设备接口
- 供热调度功能房专项做法是否包含在施工范围内
- 清单控制价及各专业分部分项金额

## 13. 风险与关注点

1. **安全负责人 C 证有效期**：李欣 C3 证 2026-11-30 到期，工期 22 个月至 2028 年中，施工期间需完成续期或更换人员报备。

3. **弱电智能化边界**：新能源集控中心核心功能为智能化管控，若弱电智能化单独招标，施工总包与智能化分包界面需明确。
4. **供热调度功能房**：明珠供热调度中心含专项功能用房，涉及供热调度系统设备，设备是否甲供需确认。
