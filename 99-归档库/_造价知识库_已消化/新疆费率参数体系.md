## 一句话摘要
新疆造价核心参数：管理费28%+利润18%+规费22%+增值税9%，人工费调增只计税金。

## 费率参数（权威来源：FEE_POLICY.md）

| 费项 | 费率 | 计算基数 | 状态 |
|------|:---:|------|:----:|
| 管理费 | **28%** | 人工费 + 机械费 | ✅锁定 |
| 利润 | **18%** | 人工费 + 机械费 | ✅锁定 |
| 规费 | **22%** | 人工费 | ✅锁定 |
| 增值税 | **9%** | 税前总造价 | ✅确认 |

### 关键规则
- **人工费调增部分只计取税金**，不计管理费/利润/规费
- 定额版本：新疆2020版消耗量定额 + 2022年估价表 + 2024人工调价

## 成本计算标准公式

```
直接费 = 人工费 + 辅材费 + 主材费 + 机械费
管理费 = (人工费 + 机械费) × 28%
利润   = (人工费 + 机械费) × 18%
规费   = 人工费 × 22%
税金   = (直接费 + 管理费 + 利润 + 规费) × 9%
总造价 = 直接费 + 管理费 + 利润 + 规费 + 税金
```

### 成本四维拆解
全部成本分析必须按四维分解：**人工 / 辅材 / 主材 / 机械**
- 偏差红线：任一单项偏离超过 ±5% → 必须自动标注
