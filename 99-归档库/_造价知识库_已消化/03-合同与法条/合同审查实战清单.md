# 合同审查实战清单

> 适用场景：建设工程施工合同（含新疆地区项目）审查工作
> 版本：v1.0 | 更新日期：2026年6月
> 适用依据：《建设工程施工合同（示范文本）》（GF-2017-0201）、新疆维吾尔自治区建设工程造价管理办法

---

## 一、通用条款审查要点

### 1.1 价格条款审查

| 序号 | 审查项 | 审查标准 | 风险等级 |
|------|--------|----------|----------|
| PR-01 | 合同价款形式 | 明确约定为固定总价/固定单价/可调价格，不得模棱两可 | ★★★★★ |
| PR-02 | 价款包含范围 | 明确包含哪些费用（人工、材料、机械、管理费、利润、规费、税金），排除项应逐一列明 | ★★★★★ |
| PR-03 | 风险范围 | 固定价格合同应明确风险范围（如5%以内材料涨跌由承包人承担），超出范围的处理方式 | ★★★★★ |
| PR-04 | 价格调整机制 | 材料价格波动超过约定幅度时的调价公式、基准价、信息价来源 | ★★★★★ |
| PR-05 | 暂估价/暂列金额 | 明确暂估价项目的确定方式、招标主体、结算规则 | ★★★★ |
| PR-06 | 工程量偏差调整 | 实际工程量与清单偏差超过15%时，综合单价调整方法 | ★★★★ |
| PR-07 | 措施费包干范围 | 明确措施费是否包干，包干范围及调整条件 | ★★★★ |
| PR-08 | 规费/税金约定 | 是否按实际发生缴纳，政策性调整是否可调 | ★★★★ |
| PR-09 | 总包服务费 | 费率、计费基数、包含的服务内容 | ★★★ |
| PR-10 | 预付款 | 比例（新疆地区一般不低于10%）、支付条件、扣回方式 | ★★★★ |

**审查实操要点：**
- 新疆地区2024版定额发布后，新项目应明确约定采用新定额
- 材料价格来源应指定具体哪一期《新疆工程造价信息》（如：乌鲁木齐地区2026年第2期信息价）
- 审查"价格包含所有风险"之类的一刀切条款，坚决要求明确风险范围

### 1.2 工期条款审查

| 序号 | 审查项 | 审查标准 | 风险等级 |
|------|--------|----------|----------|
| TM-01 | 合同工期 | 是否合理（含冬歇期），应考虑西藏/新疆地区实际施工天数 | ★★★★★ |
| TM-02 | 开工日期 | 明确以开工令/实际进场/合同签署哪个为准 | ★★★★★ |
| TM-03 | 竣工日期 | 明确竣工验收合格日/提交验收报告日/实际移交日 | ★★★★★ |
| TM-04 | 工期顺延条件 | 冬施停工、重大活动停工、恶劣天气、图纸延迟、甲供材延迟等 | ★★★★★ |
| TM-05 | 工期索赔程序 | 提出书面通知的时限要求（通常14天内），逾期是否失权 | ★★★★★ |
| TM-06 | 赶工措施费 | 发包人要求提前竣工时的赶工费用计算标准 | ★★★★ |
| TM-07 | 分期交付 | 是否允许分期竣工验收、分期结算 | ★★★ |
| TM-08 | 节点工期 | 是否有奖惩措施，节点工期是否合理 | ★★★ |

**新疆特别关注：**
- 北疆地区（乌鲁木齐、昌吉、伊犁等）每年11月15日至次年3月15日为常规冬歇期，有效施工天数仅210-240天
- 南疆地区（喀什、和田等）冬歇期较短，但夏季高温（6-8月）需考虑高温施工降效
- 高海拔地区（帕米尔高原、昆仑山北麓）需考虑低氧环境施工降效

### 1.3 付款条款审查

| 序号 | 审查项 | 审查标准 | 风险等级 |
|------|--------|----------|----------|
| PY-01 | 进度款支付比例 | 新疆地区常规为已完成工程量的80%-85%，不得低于75% | ★★★★★ |
| PY-02 | 支付周期 | 按月支付/按节点支付，明确每月25日/28日报量、次月15日前付款 | ★★★★★ |
| PY-03 | 审批时限 | 发包人审核期限（通常14-28天），逾期视为认可条款 | ★★★★★ |
| PY-04 | 质保金 | 总额不超过结算价款的3%（2019年起新疆执行此标准），质保期符合《建设工程质量管理条例》 | ★★★★★ |
| PY-05 | 结算审核时限 | 发包人结算审核时限（500万以下20天，500-2000万30天，2000-5000万45天，5000万以上60天） | ★★★★ |
| PY-06 | 逾期支付利息 | 利率标准（不低于LPR），起算时间 | ★★★★ |
| PY-07 | 农民工工资专户 | 是否设立农民工工资专用账户，比例不低于进度款的20% | ★★★★★ |
| PY-08 | 履约保证金 | 比例（通常不超过合同价10%），退还条件及时间 | ★★★★ |

### 1.4 质量/质保条款审查

| 序号 | 审查项 | 审查标准 | 风险等级 |
|------|--------|----------|----------|
| QA-01 | 质量标准 | 明确为国标合格/优良/特定标准，避免"绝对合格"等表述 | ★★★★★ |
| QA-02 | 验收程序 | 隐蔽工程验收、分部分项验收、竣工验收程序及时限 | ★★★★ |
| QA-03 | 质保期 | 基础设施/地基主体为设计文件年限（不低于50年），防水5年，装修/管道2年 | ★★★★ |
| QA-04 | 缺陷责任期 | 一般为12个月或24个月，不应超过24个月 | ★★★★ |
| QA-05 | 质量保证金返还 | 缺陷责任期满后14天内返还，不得与质保期挂钩 | ★★★★ |
| QA-06 | 第三方检测 | 检测费用承担方，争议时检测机构选定方式 | ★★★ |

### 1.5 违约责任条款审查

| 序号 | 审查项 | 审查标准 | 风险等级 |
|------|--------|----------|----------|
| LI-01 | 工期延误违约金 | 日费率合理（通常万分之二至万分之五），有上限（不超过合同总价的5%-10%） | ★★★★★ |
| LI-02 | 质量违约金 | 应与实际损失匹配，避免过高 | ★★★★ |
| LI-03 | 发包人违约 | 逾期付款、延迟提供图纸/场地等违约责任 | ★★★★★ |
| LI-04 | 违约金上限 | 合同双方违约金上限对等 | ★★★★ |
| LI-05 | 损失赔偿范围 | 是否包括间接损失（通常排除间接损失） | ★★★★ |
| LI-06 | 连带责任 | 是否要求总包对分包行为承担连带责任（须在法律范围内） | ★★★ |

**审查重点：** 新疆地区司法实践中，法院对过高违约金调整幅度较大（通常以实际损失130%为上限），但合同中仍应约定合理违约金比例。

## 二、新疆地区专用条款特别审查

### 2.1 冬季施工条款（必审项 ⭐⭐⭐⭐⭐）

- 是否明确定义冬施起止时间（建议按气象标准而非固定日期）
- 冬施增加费用计算方式（按定额冬施费率 OR 按实际发生签证）
- 冬施措施费是否单独列项或包含在综合单价内
- 因极端低温（低于-25℃）无法施工时，是否按不可抗力处理
- 冬施混凝土外加剂、保温材料费用是否单独计价
- **案例警示：** 某乌鲁木齐项目合同约定"冬施费用包含在总价内"，实际冬季施工期长达5个月，冬施费用超200万元，承包人亏损严重。建议改为"冬施费用按实际发生按实签证"。

### 2.2 高原/高海拔条款

- 海拔2500米以上区域是否计取高原降效系数（参考新疆定额规定）
- 高原地区人工降效、机械降效计算方式
- 高原地区医疗保障费用归属
- 高原地区施工效率折减率（如海拔3000m以上人工降效系数1.15-1.25）

### 2.3 运输距离条款

- 新疆地广人稀，材料运输距离普遍较长
- 是否约定超出XX公里外的超运距费用
- 自采材料（砂石料）运距确认方式
- **实操要点：** 新疆项目材料平均运距是内地项目的3-5倍，必须单独约定运距范围

### 2.4 劳务用工条款

- 是否明确劳务分包/自有劳务模式
- 农民工工资按月足额发放的保障措施
- 新疆地区少数民族用工管理及语言支持费用
- 劳务人员往返路费承担

### 2.5 风沙防护条款

- 南疆风沙季（3-5月）施工防护措施费用
- 风沙对工程质量（抹灰、混凝土养护等）影响及补救费用
- 因大风（8级以上）停工是否列入工期顺延

### 2.6 地震设防条款

- 新疆为高烈度地震区（乌鲁木齐8度设防）
- 抗震构造措施费用确认
- 地震发生时已完工程保护责任

## 四、高风险条款标准修改建议

### 4.1 "以审计为准"条款（工程行业第一杀手条款）

**原文：** "工程竣工结算以发包人委托的政府审计或社会审计结果为准，审减金额超过5%部分审计费由承包人承担。"

**风险分析：** 最高人民法院已明确（建工解释第24条）"以审计为准"条款无效。审计结果往往延迟1-3年，且审计单位无合同约束，审减随意性大。

**标准修改建议：**
> "工程竣工结算由双方在竣工后28天内核对确认。如核对不一致，可共同委托造价咨询机构审定。任何一方对审定结果仍有异议的，可向有管辖权的人民法院或仲裁机构申请鉴定。承包人理解并配合发包人接受政府审计检查，但审计结果不作为结算依据。"

### 4.2 "包死风险"条款

**原文：** "本合同为固定总价合同，除设计变更和签证外，任何市场价格波动、政策调整等风险均由承包人承担。"

**风险分析：** 2021年以来新疆地区钢材、水泥等主要材料价格波动剧烈（如2021年钢材从3800元/t涨至6200元/t），此条款将导致承包方巨大亏损。

**标准修改建议：**
> "本合同为固定总价合同，风险范围为：材料价格波动在±5%（含）以内由承包人承担，超出±5%部分按以下方式调整：[调整公式]。主要材料范围包括：钢筋、商品混凝土、水泥、砂石料、砖砌块、钢管、型钢、防水材料、保温材料。调整基价为投标期（2026年第X期）新疆工程造价信息发布的乌鲁木齐地区信息价。"

### 4.3 "无限工期延误违约金"条款

**原文：** "每延误一天按照合同总价的千分之二支付违约金，无上限。"

**风险分析：** 按此条款，一个5000万项目延误100天，违约金高达1000万元（合同价的20%）。根据《民法典》及司法解释，违约金超过实际损失30%即视为过高，法院可调减。

**标准修改建议：**
> "因承包人原因造成工期延误的，每延误一天按合同总价的万分之三（但不超过合同总价的5%）支付违约金。违约金总额上限为合同总价的5%。发包人逾期支付工程款的，每逾期一日按应付未付金额的万分之三支付逾期利息。"

### 4.4 "无限质量保修"条款

**原文：** "承包人对本工程质量终身负责，无论何种原因造成的质量问题，承包人均应无条件修复并承担全部费用。"

**风险分析：** 无限扩大承包人责任，违反《建设工程质量管理条例》关于质保期和保修范围的规定。

**标准修改建议：**
> "质量保修范围、期限、责任按《建设工程质量管理条例》第40条及《房屋建筑工程质量保修办法》执行。因承包人施工原因造成的质量缺陷，由承包人负责修复。因设计、使用不当、不可抗力或地基不均匀沉降等原因造成的质量问题，由责任方承担。"

### 4.5 "预付款无限扣回"条款

**原文：** "预付款自工程开工后逐月按进度款比例的50%扣回，直至扣完为止。"

**风险分析：** 预付款与进度款同时高比例扣回，可能导致承包人资金链断裂。新疆地区冬歇期长时间无收入，但预付款仍被扣回，承包人无现金流支付来年复工费用。

**标准修改建议：**
> "预付款自工程款支付达到合同总价的20%后开始扣回，分5个月等额扣回。冬歇期间暂停预付款扣回，复工后继续。"

## 五、分级审查体系

### 5.1 必改项（★★★★★）

条款性质界定：
1. 违反法律法规强制性规定的条款
2. 涉嫌排除对方主要权利的格式条款
3. 责任分配严重失衡的条款
4. 结算依据不确定/审核时限无约定的条款
5. 逾期违约责任无上限或过高的条款
6. 新疆地区未约定冬施/地方性调整的条款
7. 以审计结论作为结算依据的条款
8. 农民工工资支付保障缺失的条款

**处理原则：** 必须谈判修改，不修改不签约。如发包方坚持不改，应在合同中做特别标注，并在报价中预留风险费用。

### 5.2 建议修改项（★★★★）

条款范围：
1. 不利于己方但法律有保护性规定的条款
2. 约定不明容易产生争议的条款
3. 时限过短可能导致失权的条款
4. 新疆地区特有且费用影响较大的条款

**处理原则：** 优先争取修改，如对方不接受可保留，但须内部确认风险可控。

### 5.3 关注项（★★★及以下）

条款范围：
1. 有争议但影响较小的条款
2. 可通过加强管理弥补的条款
3. 行业惯例且风险可控的条款

**处理原则：** 一般不需修改，但在履约过程中应重点关注，提前做好管理措施。

```
│  □ 定额依据：明确采用新疆2024版定额                             │
│  □ 信息价来源：明确指定《新疆工程造价信息》期数                 │
│  □ 冬歇期进度款：冬歇期是否停止付款或按比例支付                │
│  □ 复工检查：冬歇后复工检查、模板支撑体系检查费用归属           │
│  □ 反恐要求：新疆地区特殊安保措施费用是否单独列项               │
│  □ 水资源费：新疆地区施工用水取水许可及费用约定                 │
│  □ 文物保护：新疆文物丰富区域施工遇文物的处理程序               │
│  □ 草场/林地补偿：野外施工占用草场补偿费用约定                  │
│  □ 地方材料：戈壁料、风积沙等地方材料的质量标准与计价           │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

**实操建议：**
1. **签约前尽调：** 新疆各地州（乌鲁木齐/昌吉/伊犁/喀什/阿克苏/库尔勒/哈密等地）气候条件、人工费水平、材料供应情况差异大，应逐一评估
2. **价格留余量：** 新疆项目不可预见因素多，建议报价时增加5%-10%的不可预见费
3. **约定本地仲裁：** 争议解决优先选择乌鲁木齐仲裁委员会或项目所在地仲裁机构，减少异地诉讼成本
4. **工期留余地：** 北疆项目工期计算务必考虑冬歇期，合同工期应在实际施工天数基础上减去3-4个月
