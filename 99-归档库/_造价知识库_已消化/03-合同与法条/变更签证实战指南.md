# 变更签证实战指南

> 适用场景：建设工程施工过程中的工程变更与现场签证管理
> 版本：v1.0 | 更新日期：2026年6月
> 适用依据：《建设工程工程量清单计价规范》（GB50500-2013）、新疆2024版定额

---

## 一、变更的5种类型及处理程序

### 类型1：设计变更

**定义：** 设计单位对原施工图设计文件的修改、补充和完善。

**常见原因：**
- 设计图纸与现场实际不符（新疆常见：地质条件与勘察报告不符）
- 设计错误、遗漏
- 发包人要求调整功能或标准
- 优化设计以降低造价或方便施工

**处理程序：**
```
发起（设计单位/发包人/承包人）→ 设计变更通知单 → 总监审核 → 发包人审批 → 下发施工单位 → 施工实施 → 签证确认
```

**关键节点：**
- 设计变更通知单须在施工前下发（避免返工）
- 收到变更后7天内承包人应提交变更价款报告

### 类型2：施工变更

**定义：** 因施工条件变化或发包人要求调整施工方案引起的变更。

**常见原因：**
- 施工场地条件变化（新疆常见：开挖后发现地质条件与勘察不符）
- 施工顺序、施工方法调整
- 发包人要求的施工标准提高
- 现场发现地下障碍物、管线

**处理程序：**
```
发现变化 → 承包人提出变更申请 → 监理/发包人确认 → 变更方案论证 → 审批 → 实施 → 签证确认
```

### 类型3：工程量清单偏差

**定义：** 招标工程量清单与施工图纸实际工程量之间的差异。

**常见原因：**
- 清单编制漏项、错项
- 工程量计算偏差（新疆定额的计量规则与国标清单差异）
- 项目特征描述不准确

**处理程序：**
```
发现偏差 → 承包人复核并提交差异报告 → 发包人/造价咨询复核 → 确认 → 签证
```

**注意：** 根据GB50500-2013，工程量偏差超过15%时，综合单价应进行调整。

### 类型4：现场签证

**定义：** 承包人与发包人（监理）对施工过程中涉及合同价款之外的责任事件所做的签认证明。

**适用范围：**
- 合同外零星用工、零星机械
- 突发事件的抢修、抢险
- 发包人临时要求的合同外工作
- 隐蔽工程现场确认
- 施工条件变化导致的额外工作

**处理程序：**
```
事件发生 → 现场记录（拍照/测量）→ 24小时内通知监理 → 三方现场确认 → 签证单签署 → 结算
```

### 类型5：工期变更

**定义：** 因非承包人原因导致的施工工期调整。

**常见原因：**
- 发包人原因（场地移交延迟、图纸提供延迟、甲供材延迟）
- 不可抗力（新疆：极端低温、大风、暴雨、地震）
- 冬歇期延长或提前
- 政策因素（重大活动停工、环保要求停工）

**处理程序：**
```
事件发生 → 7天内提交工期延期申请及证据 → 监理审核 → 发包人确认 → 工期签证
```

### 2.2 填写规范要点

**核心原则：** 一事一签、依据充分、数据准确、签认完整

**事由填写规范（5W1H法）：**
- **When（时间）：** 具体到年/月/日/时
- **Where（地点）：** 具体部位（如：2号楼3层A-C/1-3轴）
- **What（内容）：** 做什么、怎么做
- **Why（原因）：** 为什么做（设计变更单编号、会议纪要编号、现场条件）
- **Who（参与方）：** 哪些人参与确认
- **How（如何计价）：** 明确计价方式

**工程量填写规范：**
- 必须有详细计算公式，附计算简图
- 隐蔽工程签证应在覆盖前完成测量确认
- 土石方工程应有原始地面标高和开挖后标高（三方联测）
- 混凝土工程应有模板展开面积或混凝土体积

**金额填写规范：**
- 明确采用哪期信息价、哪本定额
- 人工工日单价按合同约定还是按实际签证
- 材料价格附采购发票或市场询价记录
- 机械台班费按定额台班还是实际租赁价

### 2.3 典型错误填写示范

**错误示范1（事由不清）：**
> "因现场条件变化，增加部分工作。"

**问题：** 什么条件变化？增加什么工作？多少量？为什么变化？全都没说清楚。

**正确写法：**
> "因2号楼基坑开挖至-4.5m时发现地质条件与勘察报告不符，原设计持力层为卵石层，实际为粉质粘土层（承载力不足），经2026年5月20日四方会议（纪要编号：2026-020）决定，基坑超挖1.2m并换填C20素混凝土。换填范围：2号楼基础范围（30m×20m），换填深度1.2m，换填方量720m³。具体计算见附件。"

**错误示范2（量与价不分）：**
> "签证机械台班费5000元。"

**问题：** 什么机械？用了多少台班？单价怎么来的？没有过程数据。

**正确写法：**
> "使用PC220挖掘机1台，工作8小时（1个台班），台班单价1350元/台班（按施工当期乌鲁木齐信息价），合计1350元。"

## 三、变更估价三种方法（含案例）

### 方法1：适用已有单价

**条件：** 变更工作内容与合同清单中已有项目相同或类似

**原则：** 直接采用合同已有的综合单价

**案例：**
```
原合同清单：C30混凝土浇筑（柱），综合单价850元/m³
变更内容：增加3根柱子C30混凝土浇筑，工程量15m³
估价：15m³ × 850元/m³ = 12,750元
```

### 方法2：参照类似单价换算

**条件：** 变更工作内容与合同清单项目类似但材料/规格/厚度不同

**原则：** 在原单价基础上仅调整差异部分

**案例：**
```
原合同清单：C30混凝土路面（200mm厚），综合单价180元/m²
变更内容：改为C30混凝土路面（250mm厚）
估价调整：
  原单价中混凝土含量：0.2m³/m² × (450元/m³) = 90元/m²
  新单价中混凝土含量：0.25m³/m² × (450元/m³) = 112.5元/m²
  调整后单价：180 - 90 + 112.5 = 202.5元/m²
注意：模板、人工、机械等费用不变
```

### 方法3：重新组价

**条件：** 变更工作内容与合同清单项目完全不同

**原则：** 采用合同约定或施工期的定额和信息价重新组价

**案例：**
```
变更内容：增加GRC装饰线条安装（非合同清单项目）
组价过程：
  人工费：0.8工日/m × 人工单价180元/工日 = 144元/m
  材料费：GRC线条 85元/m + 胶粘剂15元/m + 连接件20元/m = 120元/m
  机械费：2元/m
  管理费：(144+120+2)×8% = 21.28元/m
  利润：(144+120+2)×5% = 13.30元/m
  综合单价(不含税)：144+120+2+21.28+13.30 = 300.58元/m
  税金：300.58×9% = 27.05元/m
  含税综合单价：300.58+27.05 = 327.63元/m

注意：管理费和利润费率按合同约定费率执行
```

### 估价方法选择优先级

```
第一优先：合同中有适用单价 → 直接采用
第二优先：合同中有类似单价 → 参照换算
第三优先：合同中无适用/类似 → 重新组价（采用合同约定费率）
第四优先：合同无限价依据 → 按施工期信息价+定额组价
```

## 四、签证时效管理

### 4.1 关键时间节点一览

| 节点 | 时限 | 依据 | 逾期后果 |
|------|------|------|----------|
| 变更实施前通知 | 发现变更后7天内 | 合同通用条款 | 可能被拒签 |
| 变更价款申请 | 收到变更通知后14天内 | GB50500-2013 | 视为放弃权利（合同有约定时） |
| 发包人审核变更价款 | 收到申请后14天内 | GB50500-2013 | 逾期视为认可 |
| 现场签证确认 | 事件发生后7天内 | 合同通用条款 | 事实难以认定 |
| 工期延期申请 | 延期事件发生后14天内 | GF-2017-0201 | 视为延期事件不影响工期 |
| 索赔意向通知 | 索赔事件发生后28天内 | GF-2017-0201 | 失去索赔权利 |
| 索赔报告提交 | 索赔意向通知后28天内 | GF-2017-0201 | 视为放弃索赔 |
| 最终结清申请 | 竣工验收后28-56天内 | GF-2017-0201 | 失去结算权利 |

### 4.2 新疆项目特殊时效关注

**冬歇期时效管理：**
- 冬歇前（11月初）：必须完成当年所有变更签认，避免跨年扯皮
- 冬歇期间：虽然现场停工，但签证申报工作不能停，利用冬歇期整理全年签证资料
- 复工后（3月）：复工后14天内应对上年未完签证进行集中确认

**实操建议：**
```
建立"签证台账"管理表（Excel）：
┌──────┬────────┬────────┬────────┬─────────┬─────────┬─────────┐
│ 序号 │ 签证号 │ 申报日期│ 金额(元)│ 审核状态│ 签认日期│ 备注    │
├──────┼────────┼────────┼────────┼─────────┼─────────┼─────────┤
│ 1    | BS-001  │ 6/15   │ 25,000 │ 已签认  │ 6/28    │         │
│ 2    │ BS-002  │ 7/1    │ 18,500 │ 审核中  │ -       │ 催办中  │
│ 3    │ BS-003  │ 7/15   │ 36,000 │ 已提交  │ -       │ 待催办  │
└──────┴────────┴────────┴────────┴─────────┴─────────┴─────────┘
每周更新一次，对超过14天未签认的发起催办；超过28天未签认的升级处理。
```

## 五、常见被拒原因及应对策略

### 5.1 被拒原因TOP10

| 排名 | 被拒原因 | 比例 | 应对策略 |
|------|----------|------|----------|
| 1 | 签证依据不充分（缺少设计变更单/会议纪要/现场照片） | 35% | 建立签证依据台账，事前一并准备 |
| 2 | 超时申报（超过合同约定的时限） | 20% | 专人跟踪时限，设置日历提醒 |
| 3 | 工程量计算错误或依据不足 | 15% | 三方联测、附计算书和图示 |
| 4 | 属于合同范围内的工作（包干范围争议） | 10% | 合同中明确包干范围边界 |
| 5 | 签证金额过大，发包人审批层级高 | 8% | 分拆为多个小签证（注意不违反合同） |
| 6 | 申报资料不完整（缺扫描件/照片/简图） | 5% | 建立签证资料清单，逐项核对 |
| 7 | 变更未经事前审批即施工 | 3% | 严格事前审批流程，紧急情况事后3天内补办 |
| 8 | 签证单签字不全 | 2% | 提交前逐项检查签字栏 |
| 9 | 计价方式与合同约定不符 | 1% | 计价方法严格按合同执行 |
| 10 | 与同期其他签证内容重复 | 1% | 建立签证台账避免重复 |

### 5.2 应对策略详解

**策略1：提前确认法**
> 变更实施前，先提交变更申请并获取"原则上同意"的书面回复或会议纪要，再行施工。避免"先干后签"导致的被拒。

**策略2：证据链闭合法**
> 每个签证形成完整证据链：变更依据（设计单/会议纪要）→ 实施过程记录（图片/日志）→ 完成确认（验收单/测量记录）→ 签证单

**策略3：分级处理法**
> - 10000元以下：现场签证单（快速签认）
> - 10000-50000元：变更联系单+预算书
> - 50000元以上：正式变更审批流程+组价文件

**策略4：专人负责制**
> 项目部设专人负责签证管理，确保：
> - 24小时内完成事件记录
> - 7天内完成签证申报
> - 14天内完成跟踪催办
> - 28天内完成签认闭环

## 六、新疆项目签证特点

### 6.1 冬季停工签证

**背景：** 新疆北疆地区每年11月-次年3月为冬歇期，但新疆冬季施工标准（日平均气温连续5天低于5℃）比常规更早进入冬施。

**签证要点：**
1. 记录冬季停工起止时间（以实际气温数据为准，附气象局证明）
2. 冬季停工前做好越冬维护（混凝土保温覆盖、机械设备防冻）
3. 越冬维护费用按实签证（保温材料、看护人工、取暖费）
4. 复工前检查费用签证（模板支撑体系检查、脚手架检查、机械设备检修）

**冬施签证模板：**
```
签证事由：2025-2026年冬季停工越冬维护
依据：气温连续5天低于5℃（附气象记录），2025年11月10日起进入冬施/冬歇
签证内容：
1. 越冬维护时间：2025年11月15日 - 2026年3月15日（共120天）
2. 看护人员：2人 × 120天 × 200元/天 = 48,000元
3. 取暖费：办公区/生活区用电 800kWh/月 × 4个月 × 0.8元/kWh = 2,560元
4. 混凝土保温覆盖材料：保温被500m² × 15元/m² = 7,500元
5. 越冬维护人工：4人×15天（覆盖+整理）×250元/天 = 15,000元
合计：48,000 + 2,560 + 7,500 + 15,000 = 73,060元
```

### 6.2 高原施工降效签证

**适用地区：** 帕米尔高原（塔什库尔干县海拔3200-4000m）、昆仑山北麓（和田南部山区）、天山高海拔地区

**签证要点：**
1. 海拔高度确认（须有GPS测量数据或官方海拔数据）
2. 高原降效系数取值（参考新疆2024定额附录）
3. 人工降效：一般海拔3000m以上降效系数1.15-1.25
4. 机械降效：内燃机功率下降（海拔每升高1000m功率下降8-12%）
5. 额外医疗保障费用

**高原降效签证模板：**
```
签证事由：塔什库尔干县某项目高原施工降效签证
依据：项目所在地海拔3500m，参照新疆2024定额高原降效系数
签证内容：
1. 合同清单人工费合计：5,680,000元
2. 高原降效系数：1.20（海拔3500m，查定额附表）
3. 高原人工降效增加费：5,680,000 × (1.20-1.00) = 1,136,000元
4. 合同清单机械费合计：3,200,000元
5. 高原机械降效系数：1.15（海拔3500m）
6. 高原机械降效增加费：3,200,000 × (1.15-1.00) = 480,000元
合计增加费：1,136,000 + 480,000 = 1,616,000元
```

### 6.3 风沙防护签证

**适用地区：** 南疆喀什、和田、阿克苏、吐鲁番、哈密及沙漠沿线地区

**签证内容：**
- 施工道路/场地防尘覆盖费用
- 混凝土浇筑期间防风沙棚搭设费用
- 已完工程（外墙抹灰、门窗）沙尘污染清理费用
- 材料/设备的防风沙覆盖费用
- 因沙尘暴（能见度<50m）导致的停工损失

### 6.4 超长运距签证

**适用情况：** 项目地处偏远（如且末、若羌、民丰等），材料运距超过合同约定范围

**签证内容：**
- 材料实际运距确认（GPS轨迹记录或运输单据）
- 超运距运费单价（合同约定或市场调研）
- 运输损耗、材料损耗增加
- 因运距增加的采购周期对工期的影响

## 七、签证实战案例

### 案例1：基础超深签证

**背景：** 乌鲁木齐某住宅项目，设计基础埋深-4.5m，开挖至-4.5m时发现地基承载力不足。

```
签证编号：BS-2026-015
工程名称：乌鲁木齐某住宅项目1#楼
签证类型：设计变更+现场签证
事由：1#楼基坑开挖至-4.5m时，经承载力检测（报告编号：XXXX），
     地基承载力为120kPa，不满足设计要求180kPa。
     经2026.5.20四方会议（纪要编号：HY-026）决定：
     ① 基底超挖0.8m至-5.3m
     ② 换填C15毛石混凝土至设计底标高

工程量：
  超挖土方：25m × 18m × 0.8m = 360m³
  换填混凝土：360m³ × 1.02（损耗率）= 367.2m³

计价：
  超挖土方（坚土）：360m³ × 28.5元/m³ = 10,260元
  C15毛石混凝土换填：367.2m³ × 685元/m³ = 251,532元
  共计：261,792元

附件：承载力检测报告、会议纪要、开挖后照片3张、标高测量记录
```

### 案例2：冬季施工增加费签证

**背景：** 昌吉某项目业主要求冬季继续施工（不冬歇）。

```
签证编号：BS-2026-032
工程名称：昌吉某商业综合体
签证类型：施工变更（冬施）
事由：合同原计划2025年11月15日冬歇，现业主要求继续施工至12月30日。
     依据：2025年11月5日工程例会纪要（HL-048）

冬施增加费计算：
1. 混凝土冬施外加剂：
   使用量：350m³ × 15kg/m³ = 5,250kg
   -10℃防冻剂：5,250kg × 3.5元/kg = 18,375元
   早强剂：5,250kg × 2.5元/kg = 13,125元
   合计：31,500元

2. 混凝土保温养护：
   保温被租赁：800m² × 45天 × 1.5元/m²·天 = 54,000元
   彩条布：500m² × 3.5元/m² = 1,750元
   暖风机租赁：6台 × 45天 × 180元/台·天 = 48,600元
   柴油费：6台 × 45天 × 15升/天 × 7.5元/升 = 30,375元
   合计：134,725元

3. 冬施人工降效增加：
   降效系数1.15，冬施期间完成产值850,000元
   人工降效增加：850,000 × (1.15-1.00) × 30%（人工占比）= 38,250元

4. 冬施管理费增加：
   取暖费：8,000元/月 × 1.5月 = 12,000元
   冬施方案编制及技术措施费：5,000元

冬施增加费合计：31,500 + 134,725 + 38,250 + 12,000 + 5,000 = 221,475元
```

### 案例3：材料调差签证

**背景：** 喀什某项目施工期间钢筋价格上涨超合同约定范围。

```
签证编号：BS-2026-048
工程名称：喀什某保障房项目
签证类型：现场签证（材料调差）
事由：2026年3月-6月施工期间，钢筋（HRB400）价格持续上涨，
     从合同基准价3,850元/t（2025年12月信息价）上涨至4,620元/t
     （2026年4月信息价），涨幅20%，超过合同约定的±5%风险范围。
     依据：合同专用条款12.1条。

调差计算：
1. 风险范围：±5%
   基准价（信息价2025.12）：3,850元/t
   风险下限：3,850 × 0.95 = 3,657.5元/t
   风险上限：3,850 × 1.05 = 4,042.5元/t

2. 施工期钢筋平均信息价（2026.4）：4,620元/t
   涨幅 = (4,620 - 3,850) / 3,850 = 20%
   超出5%部分 = 20% - 5% = 15%

3. 调差量：2026年3月-6月使用钢筋量：286t
   调差单价：3,850 × 15% = 577.5元/t
   调差金额：286t × 577.5元/t = 165,165元
   税金：165,165 × 1.09 = 180,030元

附件：钢筋进场台账、信息价截图、采购发票复印件
```

### 案例5：隐蔽工程签证

**背景：** 伊犁某河道治理工程，基础处理深度超过设计。

```
签证编号：BS-2026-072
工程名称：伊犁某河道治理工程（K2+300-K2+600段）
签证类型：现场签证（隐蔽工程）
事由：K2+300-K2+600段基础开挖后，发现基底为淤泥质粉土，
     需清除换填。施工前三方联测原始标高，完成后三方联测验收。

联测数据：
  原始地面标高：平均725.30m
  设计清淤底标高：724.10m
  实际清淤底标高（清除淤泥至硬底）：723.60m
  超深：0.50m
  清淤宽度：河底宽8m，边坡1:1.5，两侧超挖0.5m
  清淤长度：300m

工程量计算：
  断面面积：[(8+0.5×2) + (8+0.5×2 + 2×1:1.5×0.5)] / 2 × 0.5 = 5.06m²
  清淤总量：5.06m² × 300m = 1,518m³
  设计量：设计清淤至724.10，断面面积4.32m² × 300m = 1,296m³
  签证增量：1,518 - 1,296 = 222m³

计价：222m³ × 42元/m³（合同清单外运单价）= 9,324元

附件：三方联测记录、标高测量原始数据、施工前后照片
```

## 九、常见问题速查表

| 问题 | 解决方案 |
|------|----------|
| 发包人拖签不签字 | ① 每周发催办函留存证据 ② 使用EMS邮寄签证单 ③ 合同期满28天未签认视为认可 |
| 监理口头同意但拒签 | ① 要求出具书面意见 ② 保留微信/短信记录 ③ 强调现场影像记录 |

| 跨年签证未闭环 | ① 利用冬歇期整理 ② 复工前集中清理 ③ 必要时折让部分金额换取签认 |
| 变更与签证界限不清 | 一般划分：设计出图变更走变更程序，现场临时调整走签证程序 |

> **最后提示：** 签证/变更管理是施工项目经营效益的关键环节。根据行业统计，项目利润的30%-60%来自签证变更。新疆地区条件特殊，签证变更管理的精细化程度直接决定项目盈亏。建议每个项目配备专职签证管理负责人，建立"事前有策划、事中有跟踪、事后有闭环"的全流程管理机制。
