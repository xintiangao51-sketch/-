## 工具一：土方工程算量模板

### 1.1 方格网法土方计算

**用途：** 场地平整、基坑开挖的土方量计算。适用于场地地形起伏较大、需要精确计算挖填方量的场景。

**核心公式：**

```
方格网法原理：V = (h1 + h2 + h3 + h4) / 4 × A

其中：
h1-h4 = 方格四个角点的施工高度（设计标高-自然标高）
A = 方格面积（通常20m×20m或10m×10m）

填方区高度为负(-)，挖方区高度为正(+)
```

**Excel模板结构：**

| 方格编号 | 自然标高 | 设计标高 | 施工高度h1 | h2 | h3 | h4 | 方格面积 | 挖方量(m³) | 填方量(m³) |
|---|---|---|---|---|---|---|---|---|---|
| A-1 | 852.35 | 851.00 | +1.35 | +1.20 | +0.95 | +1.10 | 400 | =(D3+E3+F3+G3)/4*H3 | 0 |
| A-2 | 851.20 | 851.00 | +0.20 | -0.15 | -0.30 | +0.10 | 400 | 需用零点法 | 需用零点法 |

**关键公式：**

```
挖方量（全挖方格）= SUM(施工高度)/4 × 面积
填方量（全填方格）= ABS(SUM(施工高度))/4 × 面积

零线计算（部分挖部分填）：
零点距角点距离 = 角点施工高度 / (角点施工高度 + 相邻角点施工高度绝对值) × 方格边长
```

**操作方法：**
1. 导入场地测量数据（自然标高）
2. 输入设计标高（根据图纸）
3. 计算施工高度 = 自然标高 - 设计标高
4. 标记正负号（正=挖，负=填）
5. 判断方格类型（全挖/全填/混合）
6. 混合方格需用零点法分割计算

**注意要点：**
- 方格大小根据地形复杂程度选择，地形变化大时用10m×10m
- 零点位置计算精度影响总方量精度
- 新疆项目注意：冻土深度影响开挖边坡系数
- 地下水位以下土方需单独计算排水费用
- 结果需附加松散系数（一般土方1.25-1.30）

### 1.2 断面法土方计算

**用途：** 道路、管沟、河道等线性工程的土方量计算。

**核心公式：**

```
V = (A1 + A2) / 2 × L

其中：
A1、A2 = 相邻两断面的截面积
L = 两断面间距

道路工程常用平均断面法，精度更高用棱台法：
V = (A1 + A2 + √(A1×A2)) / 3 × L
```

**Excel模板结构：**

| 桩号 | 断面积(m²) | 间距(m) | 平均面积法(m³) | 棱台法(m³) |
|---|---|---|---|---|
| K0+000 | 32.50 | — | — | — |
| K0+020 | 38.20 | 20 | =(B3+B4)/2*C4 | =(B3+B4+SQRT(B3*B4))/3*C4 |
| K0+040 | 42.10 | 20 | ... | ... |

**操作方法：**
1. 按桩号输入各断面面积
2. 计算相邻断面间距
3. 分别用平均面积法和棱台法计算
4. 平原地区可用平均面积法，山区用棱台法

**注意要点：**
- 断面间距不宜过大，地形变化大时加密至10m
- 填方和挖方分别计算
- 路基土方需考虑路槽加深量
- 新疆戈壁地区土石比例对造价影响大，需注明土石类别

## 工具二：钢筋翻样快速估算

### 2.1 钢筋工程量计算

**用途：** 快速估算各类构件的钢筋用量，支撑工程量提量、材料采购、成本测算。

**核心公式：**

```
单根钢筋长度 = 净长 + 弯钩增加值 + 锚固长度 + 搭接长度（如有）
单构件钢筋重量 = Σ(单根长度 × 根数 × 理论重量)
理论重量(kg/m) = d² × 0.00617（d为钢筋直径mm）

示例：φ25钢筋理论重量 = 25² × 0.00617 = 3.856 kg/m
```

**Excel模板结构：**

| 构件 | 直径 | 单根长度(m) | 根数 | 总长(m) | 理论重量(kg/m) | 小计(kg) |
|---|---|---|---|---|---|---|
| 框架柱KZ1主筋 | φ25 | 3.85 | 12 | 46.20 | 3.86 | 178.33 |
| 框架柱KZ1箍筋 | φ10 | 1.85 | 48 | 88.80 | 0.62 | 55.06 |
| 框架梁KL1主筋 | φ22 | 6.50 | 8 | 52.00 | 2.98 | 154.96 |
| 框架梁KL1箍筋 | φ8 | 2.10 | 36 | 75.60 | 0.40 | 30.24 |

**关键公式：**

```
理论重量(kg/m)：=ROUND(B2^2*0.00617,2)
总长(m)：=C2*D2
小计(kg)：=E2*F2
```

**常用钢筋理论重量速算：**

| 直径(mm) | 理论重量(kg/m) | 每吨长度(m/t) | 常见用途 |
|---|---|---|---:|---:|---|
| 6 | 0.222 | 4,505 | 箍筋、分布筋 |
| 8 | 0.395 | 2,532 | 箍筋、板筋 |
| 10 | 0.617 | 1,621 | 箍筋、板筋、分布筋 |
| 12 | 0.888 | 1,126 | 梁柱箍筋、板筋 |
| 14 | 1.210 | 826 | 梁筋、墙筋 |
| 16 | 1.580 | 633 | 梁筋、柱筋 |
| 18 | 2.000 | 500 | 梁筋、柱筋 |
| 20 | 2.470 | 405 | 梁筋、柱筋、基础筋 |
| 22 | 2.980 | 336 | 柱筋、基础筋 |
| 25 | 3.860 | 259 | 柱筋、基础筋、大梁筋 |
| 28 | 4.840 | 207 | 大梁筋、基础筋 |
| 32 | 6.310 | 158 | 基础筋、大跨度梁 |

### 2.2 钢筋常规含量参考

| 结构类型 | 钢筋含量(kg/m²) | 备注 |
|---|---|---|
| 多层砖混住宅 | 25-35 | 新疆地区常用 |
| 多层框架住宅 | 40-55 | — |
| 小高层(12-18层) | 55-70 | — |
| 高层(18-30层) | 65-85 | — |
| 框架厂房 | 35-50 | — |
| 地下车库 | 100-140 | 含底板、顶板 |
| 人防工程 | 140-200 | — |

**操作方法：**
1. 从施工图中提取各构件配筋信息
2. 按构件类型分类计算
3. 锚固长度、搭接长度按规范取值
4. 汇总各构件钢筋量
5. 按直径规格汇总，便于材料采购

**注意要点：**
- 锚固长度按GB50010-2010(2015版)取值，注意抗震等级
- 新疆地区抗震设防烈度多为7-8度，锚固长度按抗震等级执行
- 搭接接头面积百分率影响搭接长度系数（1.2/1.4/1.6）
- 直径≥22mm的钢筋宜用机械连接，按接头数量另行计算
- 砌体拉结筋按构造设置，容易被漏算

## 工具三：混凝土工程量自动计算

**用途：** 快速计算梁、板、柱、基础等构件的混凝土用量。

**核心公式：**

```
混凝土体积 = 构件截面面积 × 构件长度（或高度）
梁：V = b × h × L（b=梁宽，h=梁高，L=梁长）
板：V = b × L × h（b=板宽，L=板长，h=板厚）
柱：V = b × h × H（b=柱宽，h=柱高，H=柱高）
独立基础：V = a × b × h（a=长，b=宽，h=高）
```

**Excel模板结构：**

| 层号 | 构件类型 | 构件编号 | 截面尺寸(宽×高)mm | 长度(m) | 数量 | 单个体积(m³) | 小计(m³) |
|---|---|---|---|---|---|---|---|
| 1F | 框架柱 | KZ1 | 600×600 | 4.50 | 8 | =B2*C2/1e6*D2 | =F2*E2 |
| 1F | 框架梁 | KL1 | 300×600 | 6.00 | 4 | =B3*C3/1e6*D3 | =F3*E3 |
| 1F | 楼板 | B1 | 100mm厚 | 36.00m² | 1 | =36*0.10 | ... |

**操作方法：**
1. 从结构图中读取构件尺寸
2. 按层和构件类型分别录入
3. 梁柱扣减节点区重复部分（手工调整扣减值约3-5%）
4. 楼梯混凝土按水平投影面积×0.2688（定额系数）估算

**注意要点：**
- 梁柱节点区混凝土体积易被重复计算，一般扣减2-3%
- 后浇带宽度通常800-1000mm，后期浇筑，单独统计
- 不同强度等级的混凝土分开统计（C30/C35/C40等）
- 新疆地区注意区分抗冻混凝土等级（F150/F200/F250等）
- 基础垫层混凝土C15或C20，厚度100mm，容易被漏算

## 工具四：综合单价分析模板

**用途：** 对清单项目进行人材机拆解分析，形成完整的综合单价组成表。用于投标报价、成本分析、变更组价。

**Excel模板结构：**

| 清单编码 | 项目名称 | 单位 | 数量 |
|---|---|---|---|
| 010502001001 | 矩形柱C30 | m³ | 1.00 |

| 费用组成 | 定额编号 | 名称 | 单位 | 定额量 | 单价(元) | 合价(元) |
|---|---|---|---|---|---|---|
| **人工费** | A1-1 | 混凝土浇筑人工 | 工日 | 1.85 | 148.00 | 273.80 |
| **材料费** | — | 商品混凝土C30 | m³ | 1.015 | 385.00 | 390.78 |
| | — | 塑料薄膜 | m² | 2.20 | 2.50 | 5.50 |
| | — | 水 | m³ | 0.15 | 3.85 | 0.58 |
| **机械费** | — | 混凝土输送泵 | 台班 | 0.05 | 850.00 | 42.50 |
| **直接费小计** | | | | | | **713.16** |
| 管理费(28%) | | | | | | 199.68 |
| 利润(18%) | | | | | | 128.37 |
| **综合单价** | | | | | | **1,041.21** |

**核心公式：**

```
人工费 = Σ(各工种工日 × 人工单价)
材料费 = Σ(材料消耗量 × 材料单价)
机械费 = Σ(机械台班量 × 台班单价)
直接费小计 = 人工费 + 材料费 + 机械费
管理费 = 直接费小计 × 管理费率
利润 = 直接费小计 × 利润率
综合单价 = 直接费小计 + 管理费 + 利润
```

**操作方法：**
1. 确定清单项目对应的定额子目
2. 分解人材机消耗量（从定额中获取）
3. 填入当期人工、材料、机械单价
4. 自动计算各项费用和综合单价
5. 与市场价对比检查合理性

**注意要点：**
- 商品混凝土已含泵送费的，不再另算机械费
- 辅材费（铁钉、铁丝等）一般按主材费的3-5%估算
- 新疆地区注意人工费调增的取费规则
- 管理费利润率按工程类别和企业资质确定
- 综合单价分析应与实际成本对账，验证报价合理性

## 工具五：进度款审核对量表

**用途：** 对施工方提交的进度款申报数据进行审核，核对已完成工程量，计算应付金额。

**Excel模板结构：**

| 清单编号 | 项目名称 | 单位 | 合同单价(元) | 清单总量 | 累计完成量 | 累计应付(万元) | 累计已付(万元) | 本期申报量 | 本期应计量 | 本期应付(万元) |
|---|---|---|---|---|---|---|---|---|---|---|
| 0101 | 土方开挖 | m³ | 25.50 | 15,000 | 12,000 | 30.60 | 25.50 | 2,500 | 2,200 | 5.61 |
| 0102 | 基础混凝土C30 | m³ | 680.00 | 2,500 | 2,100 | 142.80 | 119.00 | 400 | 380 | 25.84 |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

**关键公式：**

```
累计应付 = 累计审核完成量 × 合同单价
本期应付 = 本期审核完成量 × 合同单价
累计完成率 = 累计审核完成量 / 清单总量
应付合计 = Σ(本期应付) × (1 - 质保金比例) × (1 - 下浮率)
```

**操作方法：**
1. 接收施工方申报的进度款申请表和工程量计算书
2. 赴现场核对实际完成工程量（关键步骤）
3. 与施工方确认审核后的工程量
4. 填入审核对量表
5. 按合同约定计算应付金额
6. 扣除已付款、甲供材、质保金、罚款等

**注意要点：**
- 必须现场核实工程量，不能仅凭申报单
- 隐蔽工程应在覆盖前完成计量并签认
- 进度款累计支付不超过合同价的85%（一般合同约定）
- 甲供材按实际领用量逐月扣回
- 新疆冬季停工项目的进度款审核应在停工前完成

## 工具六：结算审核对比表

**用途：** 对施工方提交的结算申报与审核结果进行逐项对比，记录差异原因，用于结算谈判和终审。

**Excel模板结构：**

| 序号 | 清单编号 | 项目名称 | 单位 | 申报量 | 申报单价(元) | 申报金额(元) | 审核量 | 审核单价(元) | 审核金额(元) | 差异金额(元) | 差异原因 |
|---|---|---|---|---|---|---|---|---|---|---|---|
| 1 | 0101 | 土方开挖 | m³ | 16,500 | 28.50 | 470,250 | 15,200 | 25.50 | 387,600 | -82,650 | 量差1300m³，价差3元 |
| 2 | 0102 | 基础C30 | m³ | 2,650 | 720.00 | 1,908,000 | 2,580 | 680.00 | 1,754,400 | -153,600 | 量差70m³，价差40元 |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

**关键差异分析公式：**

```
量差 = (申报量 - 审核量) × 审核单价
价差 = (申报单价 - 审核单价) × 申报量
综合差异 = 申报金额 - 审核金额
差异率 = 差异金额 / 申报金额 × 100%
```

**操作方法：**
1. 导入施工方结算申报数据
2. 录入审核后的量和单价
3. 自动计算差异金额和差异率
4. 标记差异原因分类（量差/价差/多项/漏项）
5. 生成汇总统计表

**差异原因分类：**

| 原因代码 | 原因分类 | 说明 |
|---|---|---|---|
| Q-01 | 工程量差异 | 图纸量vs申报量不一致 |
| Q-02 | 计算规则差异 | 双方计算规则理解不同 |
| P-01 | 单价差异 | 合同单价vs申报单价 |
| P-02 | 材料价差异 | 信息价取用标准不同 |
| S-01 | 变更签证差异 | 签证单工程量/单价争议 |
| S-02 | 漏项 | 施工方漏报项目 |
| D-01 | 重复项 | 不同部位重复计算 |
| O-01 | 其他 | 定额套用、取费差异等 |

**注意要点：**
- 差异原因逐项记录，作为后续对量谈判依据
- 大额差异项（单笔>5万元）重点标注
- 出具结算审核报告时，附差异明细表
- 新疆项目注意：定额换算是常见争议点
- 冬季施工费、高原降效等新疆特色项目单独对比
