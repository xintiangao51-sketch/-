# 工程量计算规则汇总

---

## 一、计算规则总说明

### 1.1 计算依据
- 《建设工程工程量清单计价规范》GB50500-2013
- 《房屋建筑与装饰工程工程量计算规范》GB50854-2013
- 《新疆维吾尔自治区房屋建筑与装饰工程消耗量定额》2024版
- 施工图纸及相关技术资料

### 1.2 基本原则
1. 工程量计算必须依据施工图纸和设计说明
2. 计算规则必须与定额工程量计算规则一致
3. 计算单位必须与定额单位一致
4. 工程量计算必须准确，不得重复计算或漏算
5. 计算过程清晰可追溯

## 二、土石方工程

### 2.1 一般规定
- 土石方体积按天然密实体积计算
- 土方开挖按设计图示尺寸（含工作面和放坡）以体积计算
- 回填土按不同部位分别计算

### 2.2 土方开挖

| 项目 | 计算规则 | 计算公式 |
|---|---|---|
| 沟槽土方 | 断面面积 × 长度 | V = S断面 × L |
| 基坑土方 | 底面积 × 深度（加放坡） | V = (a+2c+KH) × (b+2c+KH) × H + 4/3K²H³ |
| 一般土方 | 开挖体积 | 按实际形状计算 |

**常用放坡系数：**

| 土壤类别 | 放坡起点深度（m） | 人工开挖 | 机械开挖（坑上） |
|---|---|---|---|
| 一、二类土 | 1.20 | 1:0.5 | 1:0.75 |
| 三类土 | 1.50 | 1:0.33 | 1:0.67 |
| 四类土 | 2.00 | 1:0.25 | 1:0.33 |

**工作面宽度（每边）：**

| 基础类型 | 工作面宽度（mm） |
|---|---|
| 砖基础 | 200 |
| 浆砌毛石、条石基础 | 150 |
| 混凝土基础垫层（支模板） | 300 |
| 混凝土基础（支模板） | 300 |
| 基础垂直面做防水层 | 1000（防水层外） |

### 2.3 土方回填

| 项目 | 计算规则 | 备注 |
|---|---|---|
| 场地回填 | 回填面积 × 平均厚度 | 平整场地 |
| 室内回填 | 主墙间净面积 × 回填厚度 | 地面以下 |
| 基础回填 | 挖方体积 - 基础及垫层体积 | 地下部分 |

### 2.4 土方运输
- 按天然密实体积计算
- 运距按实际运输距离计算
- 余土外运 = 挖方量 - 回填量（如需购土则为负）

## 三、地基处理与边坡支护工程

### 3.1 垫层
- 按设计图示尺寸以体积计算
- 公式：V = 长度 × 宽度 × 厚度

### 3.2 桩基

| 项目 | 计算规则 | 单位 |
|---|---|---|
| 预制钢筋混凝土桩 | 按设计图示桩长（包括桩尖）× 截面面积 | m³ |
| 钢管桩 | 按设计图示尺寸以质量计算 | t |
| 灌注桩 | 按设计图示桩长（包括桩尖）× 截面面积 | m³ |
| 截（凿）桩头 | 按设计图示桩截面 × 桩头长度 | m³ |

### 3.3 基坑与边坡支护

| 项目 | 计算规则 | 单位 |
|---|---|---|
| 地下连续墙 | 按设计图示墙长 × 厚度 × 槽深 | m³ |
| 土钉支护 | 按设计图示尺寸以面积计算 | ㎡ |
| 锚杆支护 | 按设计图示以长度计算 | m |
| 喷射混凝土 | 按设计图示尺寸以面积计算 | ㎡ |

## 四、砌筑工程

### 4.1 一般规定
- 按设计图示尺寸以体积计算
- 扣除门窗洞口、过人洞、空圈、嵌入墙内的钢筋混凝土柱、梁、圈梁、挑梁、过梁及凹进墙内的壁龛、管槽、暖气槽、消火栓箱所占体积
- 不扣除梁头、板头、檩头、垫木、木楞头、沿缘木、木砖、门窗走头、砖墙内加固钢筋、木筋、铁件、钢管及单个面积≤0.3㎡的孔洞所占体积
- 凸出墙面的腰线、挑檐、压顶、窗台线、虎头砖、门窗套的体积不增加

### 4.2 砖墙计算

**墙体厚度：**
- 标准砖 240×115×53mm
- 半砖墙 115mm（实际120mm）
- 一砖墙 240mm
- 一砖半墙 365mm（实际370mm）
- 两砖墙 490mm

**墙体长度：**
- 外墙：外墙按外墙中心线长
- 内墙：内墙按内墙净长线长

**墙体高度：**

| 墙体类型 | 计算规则 |
|---|---|
| 外墙 | 平屋面算至钢筋混凝土板底 |
| 内墙 | 位于屋架下弦者，算至屋架下弦底 |
| 女儿墙 | 从屋面板上表面算至女儿墙顶面（如有压顶算至压顶下） |

**计算公式：**
```
墙体体积 = (墙体长度 × 墙体高度 - 门窗洞口面积) × 墙体厚度 - 嵌入构件体积
```

### 4.3 砖基础
- 按设计图示尺寸以体积计算
- 包括附墙垛基础宽出部分体积
- 扣除地梁（圈梁）、构造柱所占体积

**基础与墙身划分：**
- 使用同一种材料时，以设计室内地面为界（有地下室者，以地下室室内地面为界）
- 使用不同材料时，位于设计室内地面±300mm以内时，以不同材料为分界线；超过±300mm时，以设计室内地面为分界线

### 4.4 砌块墙
- 按设计图示尺寸以体积计算
- 扣除门窗洞口、单个面积>0.3㎡的孔洞体积
- 砌块墙内的钢筋混凝土过梁、圈梁等另行计算
- 加气混凝土砌块墙按设计尺寸以体积计算，不扣除砌块内孔洞体积

## 五、混凝土及钢筋混凝土工程

### 5.1 一般规定
- 按设计图示尺寸以体积计算
- 不扣除构件内钢筋、预埋铁件所占体积
- 伸入承台的桩头不扣除

### 5.2 基础工程

| 基础类型 | 计算规则 |
|---|---|
| 带形基础 | 断面面积 × 长度（外墙按中心线，内墙按净长线） |
| 独立基础 | 按设计图示尺寸体积计算（含放脚） |
| 满堂基础 | 筏板面积 × 厚度 + 梁体积（梁式） |
| 桩承台 | 按设计图示尺寸体积计算 |

### 5.3 柱工程

**柱体积计算：**
```
柱体积 = 柱截面积 × 柱高
```

**柱高计算：**

| 柱类型 | 柱高计算规则 |
|---|---|
| 有梁板的柱高 | 自柱基上表面（或楼板上表面）至上一层楼板上表面 |
| 无梁板的柱高 | 自柱基上表面（或楼板上表面）至柱帽下表面 |
| 框架柱的柱高 | 自柱基上表面至柱顶高度 |
| 构造柱 | 自基础上表面至柱顶高度（含马牙槎） |

**构造柱马牙槎：**
- 每边马牙槎宽度按60mm计算
- 马牙槎体积 = 槎宽 × 柱高 × 墙厚 × 槎面数/2
- 两面墙交接：2个槎面
- 三面墙交接：3个槎面
- 四面墙交接：4个槎面

### 5.4 梁工程

**梁体积计算：**
```
梁体积 = 梁截面积 × 梁长
```

**梁长计算：**
- 梁与柱交接时，梁长算至柱侧面
- 主梁与次梁交接时，次梁长算至主梁侧面
- 梁与墙交接时，梁长算至墙侧面

**梁类型：**
- 基础梁、矩形梁、异形梁、圈梁、过梁、弧形梁

### 5.5 板工程

| 板类型 | 计算规则 |
|---|---|
| 有梁板 | 按梁、板体积之和计算（主、次梁与板） |
| 无梁板 | 按板和柱帽体积之和计算 |
| 平板 | 按设计图示尺寸体积计算，无主次梁 |
| 拱板 | 按设计图示尺寸体积计算 |

**板厚度：**
- 有梁板：按板的净厚度计算，梁另行计算（已含在有梁板中）
- 伸入墙内的板头并入板体积内

### 5.6 墙工程

**剪力墙体积计算：**
```
墙体积 = (墙长 × 墙高 - 门窗洞口面积) × 墙厚
```

**墙长：** 按设计图示中心线长计算
**墙高：**
- 基础底板（或楼板）上表面算至上一层楼板上表面
- 有框架梁时，算至梁底

### 5.7 楼梯
- 按设计图示尺寸以水平投影面积计算
- 包括休息平台、平台梁、斜梁和楼梯的连接梁
- 不扣除宽度≤500mm的楼梯井
- 伸入墙内部分不计算

**水平投影面积：**
```
S = 楼梯间净宽 × 楼梯段长
```

### 5.8 其他构件

| 构件 | 计算规则 |
|---|---|
| 散水 | 按设计图示尺寸以面积计算 | ㎡ |
| 坡道 | 按设计图示尺寸以面积计算 | ㎡ |
| 台阶 | 按设计图示尺寸以水平投影面积计算 | ㎡ |
| 压顶 | 按设计图示尺寸以体积计算 | m³ |
| 雨篷 | 按设计图示尺寸以墙外部分体积计算 | m³ |
| 挑檐 | 按设计图示尺寸以体积计算 | m³ |

## 六、钢筋工程

### 6.1 一般规定
- 按设计图示钢筋长度 × 单位理论质量以吨（t）计算
- 钢筋理论质量 = 钢筋截面积 × 长度 × 密度（7850kg/m³）
- 简化公式：W = 0.00617 × d² （d为直径，单位mm）

### 6.2 钢筋长度计算

**直筋长度：**
```
直筋长度 = 构件长度 - 保护层厚度×2 + 弯钩长度×2 + 搭接长度
```

**弯起钢筋长度：**
```
弯起钢筋长度 = 构件长度 - 保护层厚度×2 + 弯起增加长度×2 + 弯钩长度×2 + 搭接长度
```

### 6.3 保护层厚度

| 环境类别 | 构件类型 | 保护层厚度（mm） |
|---|---|---|
| 一类环境 | 板、墙、壳 | 15 |
| 一类环境 | 梁、柱、杆 | 20 |
| 二a类环境 | 板、墙、壳 | 20 |
| 二a类环境 | 梁、柱、杆 | 25 |
| 二b类环境 | 板、墙、壳 | 25 |
| 二b类环境 | 梁、柱、杆 | 35 |
| 三类环境 | 板、墙、壳 | 30 |
| 三类环境 | 梁、柱、杆 | 40 |

> 新疆多为二a类或二b类环境（寒冷地区）

### 6.4 钢筋弯钩长度

| 弯钩类型 | 弯钩长度（按直径倍数） | 说明 |
|---|---|---|
| 180°弯钩 | 6.25d | HPB300光圆钢筋 |
| 135°弯钩 | 4.9d（推算值） | 箍筋、拉筋 |
| 90°弯钩 | 3.5d + 弯折调整 | 上部钢筋弯折 |

### 6.5 绑扎搭接长度

| 钢筋类型 | C30混凝土 | C35混凝土 | C40混凝土及以上 |
|---|---|---|---|
| HPB300 | 30d | 27d | 25d |
| HRB335 | 38d | 34d | 32d |
| HRB400 | 42d | 38d | 35d |

> 受拉钢筋搭接长度为锚固长度La×修正系数（1.2~1.6），具体按规范

### 6.6 箍筋计算

**箍筋长度：**
```
矩形箍筋长度 = (构件宽 + 构件高) × 2 - 8×保护层 + 2×弯钩长度
```

**箍筋根数：**
```
加密区根数 = (加密区长 / 加密区间距) + 1
非加密区根数 = (非加密区长 / 非加密区间距) - 1
总根数 = 加密区根数×2 + 非加密区根数
```

### 6.7 常用钢筋理论质量

| 直径（mm） | 理论质量（kg/m） | 直径（mm） | 理论质量（kg/m） |
|---|---|---|---|
| 6.5 | 0.260 | 18 | 1.998 |
| 8 | 0.395 | 20 | 2.466 |
| 10 | 0.617 | 22 | 2.984 |
| 12 | 0.888 | 25 | 3.850 |
| 14 | 1.208 | 28 | 4.834 |
| 16 | 1.578 | 32 | 6.313 |

### 7.2 计算规则

| 项目 | 计算规则 | 单位 |

## 八、屋面及防水工程

### 8.1 屋面防水
- 按设计图示尺寸以面积计算
- 斜屋顶（不包括平屋顶找坡）按斜面积计算
- 平屋顶按水平投影面积计算
- 屋面女儿墙、伸缩缝和天窗等处的弯起部分，并入屋面工程量内
- 弯起高度按设计规定，无规定时按250mm计算

### 8.2 防水卷材/涂料
- 按设计图示尺寸以面积计算
- 搭接及附加层用量不计入工程量，在综合单价中考虑

**屋面防水面积：**
```
S = 屋面水平面积 + 女儿墙弯起面积 + 天窗弯起面积 + 突出物弯起面积
```

### 8.3 地下防水
- 基础底板防水：按底板面积计算
- 外墙外防水：按外墙外侧垂直面积计算
- 外墙内防水：按外墙内侧垂直面积计算
- 后浇带防水：按后浇带长度 × 宽度计算

### 8.4 变形缝
- 按设计图示以长度计算
- 单位：m
- 包括屋面变形缝、墙身变形缝、基础变形缝

## 九、保温隔热工程

### 9.1 一般规定
- 按设计图示尺寸以面积计算，扣除门窗洞口面积
- 单位：㎡
- 厚度按设计要求

### 9.2 各部位计算规则

| 部位 | 计算规则 |
|---|---|
| 屋面保温 | 按设计图示尺寸以面积计算，不扣除房上烟囱、风帽底座、风道等 |
| 外墙外保温 | 按设计图示尺寸以面积计算，扣除门窗洞口，增加门窗洞口侧壁 |
| 外墙内保温 | 按设计图示尺寸以面积计算，扣除门窗洞口 |
| 天棚保温 | 按设计图示尺寸以面积计算，不扣除柱、垛、管道所占面积 |
| 地面保温 | 按设计图示尺寸以面积计算，扣除凸出地面的构筑物、设备基础 |

## 十、楼地面装饰工程

### 10.1 整体面层
- 按设计图示尺寸以面积计算
- 扣除凸出地面构筑物、设备基础、室内铁道、地沟等所占面积
- 不扣除间壁墙及≤0.3㎡的柱、垛、附墙烟囱及孔洞所占面积
- 门洞、空圈、暖气包槽、壁龛的开口部分不增加面积

### 10.2 块料面层
- 按设计图示尺寸以面积计算
- 门洞、空圈、暖气包槽、壁龛的开口部分并入相应的工程量内

### 10.3 楼梯装饰
- 按设计图示尺寸以楼梯（包括踏步、休息平台及≤500mm的楼梯井）水平投影面积计算
- 楼梯与楼地面相连时，算至梯口梁内侧边沿
- 无梯口梁者，算至最上一层踏步边沿加300mm

### 10.4 踢脚线
- 按设计图示长度乘以高度以面积计算
- 或按延长米计算

## 十一、墙柱面装饰工程

### 11.1 墙面抹灰
- 按设计图示尺寸以面积计算
- 扣除墙裙、门窗洞口及单个>0.3㎡的孔洞面积
- 不扣除踢脚线、挂镜线和墙与构件交接处的面积
- 门窗洞口和孔洞的侧壁及顶面不增加面积

**内墙面抹灰面积：**
```
S = (内墙净长 × 内墙净高) - 门窗洞口面积 - 墙裙面积
```

### 11.2 外墙抹灰
- 按设计图示尺寸以面积计算
- 扣除门窗洞口、外墙裙和单个>0.3㎡的孔洞面积
- 门窗洞口侧壁及顶面不增加面积

### 11.3 块料墙面
- 按设计图示尺寸以面积计算
- 门窗洞口侧壁并入墙面工程量内
- 按实贴面积计算

### 11.4 墙饰面
- 按设计图示墙净长 × 净高以面积计算
- 扣除门窗洞口及单个>0.3㎡的孔洞面积

### 11.5 柱（梁）饰面
- 按设计图示饰面外围尺寸以面积计算
- 柱帽、柱墩并入相应柱饰面工程量内

## 十二、天棚工程

### 12.1 天棚抹灰
- 按设计图示尺寸以水平投影面积计算
- 不扣除间壁墙、垛、柱、附墙烟囱、检查口和管道所占的面积
- 带梁天棚，梁两侧抹灰面积并入天棚面积内

### 12.2 吊顶天棚
- 按设计图示尺寸以水平投影面积计算
- 不扣除间壁墙、检查口、附墙烟囱、柱、垛和管道所占面积
- 扣除单个>0.3㎡的孔洞、独立柱及与天棚相连的窗帘盒所占的面积

## 十三、脚手架工程

### 13.1 综合脚手架
- 按建筑面积计算
- 适用于能够按"建筑面积计算规则"计算建筑面积的建筑工程脚手架

### 13.2 单项脚手架

| 项目 | 计算规则 |
|---|---|
| 外脚手架 | 按外墙外边线长度 × 高度 以面积计算 |
| 里脚手架 | 按内墙净长 × 内墙高度 以面积计算 |
| 满堂脚手架 | 按室内净面积计算 |
| 挑脚手架 | 按搭设长度 × 搭设层数 以延长米计算 |
| 悬空脚手架 | 按搭设水平投影面积计算 |

### 13.3 脚手架高度
- 外脚手架高度：设计室外地坪至檐口滴水高度
- 有女儿墙者，高度算至女儿墙压顶
- 平屋顶算至檐口板顶

## 十四、模板工程

### 14.1 一般规定
- 按模板与现浇混凝土构件的接触面积计算
- 单位：㎡

### 14.2 各构件模板计算

| 构件 | 计算规则 |
|---|---|
| 基础模板 | 按基础与模板接触面积计算 |
| 柱模板 | 按柱周长 × 柱高 计算 |
| 梁模板 | 按梁两侧面 + 梁底 面积计算 |
| 板模板 | 按板底面积计算 |
| 墙模板 | 按墙两侧面积计算 |
| 楼梯模板 | 按楼梯水平投影面积计算 |
| 后浇带模板 | 按后浇带接触面积计算 |

### 14.3 柱模板
```
S = (a + b) × 2 × 柱高
```
- a、b为柱截面长、宽
- 柱高按通高计算，扣板厚（有梁板）

### 14.4 梁模板
```
S = (梁高 × 2 + 梁宽) × 梁长 - 与柱、梁交接面积
```
- 梁底 + 两侧面
- 扣除与柱、主梁交接面积

## 十五、垂直运输及超高

### 15.1 垂直运输
- 按建筑面积计算
- 或按施工工期日历天数计算
- 单位：㎡ 或 天

### 15.2 超高施工增加
- 建筑物檐高超过20m计算超高增加费
- 按超高部分建筑面积计算
- 檐高 = 设计室外地坪至檐口滴水高度

**超高降效系数：**
- 20m以上至30m：10%
- 30m以上至40m：15%
- 40m以上至50m：20%
- 50m以上：每增加10m增加5%
