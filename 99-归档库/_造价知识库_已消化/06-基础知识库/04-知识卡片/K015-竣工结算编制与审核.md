# K015-竣工结算编制与审核

## 一、结算编制依据

1. 中标通知书、投标文件、施工合同及补充协议
2. 完整施工图纸、竣工图纸及设计变更通知单
3. 经确认的工程签证单、工程联系单、会议纪要
4. 隐蔽工程验收记录、材料进场验收记录
5. 甲供材料清单及价格确认单
6. 施工组织设计、专项施工方案
7. 完整的工程量计算书（底稿需保存备查）

## 二、资料清单（必收项）

| 类别 | 资料名称 | 备注 |
|------|---------|------|
| 合同类 | 施工合同、补充协议、中标通知书 | 核对合同范围 |
| 图纸类 | 施工图、竣工图、图纸会审纪要 | 确认变更轨迹 |
| 变更类 | 设计变更单、技术核定单 | 签字盖章齐全 |
| 签证类 | 现场签证单、工程联系单 | 时效内完成签认 |
| 材料类 | 材料认价单、甲供材料清单 | 价格依据完整 |
| 验收类 | 隐蔽工程验收、分部分项验收 | 质量合格证明 |

## 三、结算编制程序

1. **基础整理**：汇总所有合同内工程量清单，逐项核对完成情况
2. **变更汇总**：将所有设计变更、签证进行编号、分类、汇总
3. **调价计算**：合同内调价（信息价波动调差）+ 变更增减费用
4. **索赔计算**：工期索赔、费用索赔（如遇物价大幅上涨）
5. **编制结算书**：按清单格式编制完整的结算书
6. **内部审核**：三级审核——编制人自审→项目负责人复核→公司经营部终审

## 四、审核对量技巧

**发包人审核关键点：**
1. **工程量复核**：重点审查土方量（放坡系数、工作面宽度）、钢筋量（搭接和锚固长度）、混凝土量（不扣减的孔洞是否多扣）。
2. **定额套用**：检查高套定额（普通砼套成防水砼）、重复套用（工作内容已包含仍单独列项）。
3. **取费合理性**：工程类别判定是否准确（一类/二类/三类）、费率是否合规。

**承包人对审技巧：**
- 提前准备完整的计算底稿，标注每个数据的来源
- 对争议较大的项目，准备充分的现场照片和施工记录作为佐证
- 先易后难：先统一无争议部分，缩小争议范围后再集中谈分歧

## 五、新疆结算审核时间规定

- 竣工结算报告提交后，发包人应在28天内审核完毕（500万元以下项目）
- 500万-5000万元项目：42天内审核完毕
- 5000万元以上项目：56天内审核完毕
- **逾期不审视为认可**——这是承包人的重要维权依据

## 六、常见争议焦点

1. **土方量争议**：特别在新疆戈壁地区，实际开挖与图纸不符（地质变化）
2. **材料价差调整**：合同约定是否明确调整范围和幅度
3. **冬施费用计取**：实际发生天数与计划是否一致
4. **高原降效费用**：有无充分的计算依据和签证支撑

> 结算决胜在过程——过程资料越完善，结算越顺利。新疆项目尤其注意冬季施工记录和特殊地质条件下的施工作业证据。
