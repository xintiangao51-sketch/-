### 6.2 计取方式

```
高原降效增加费 = (定额人工费 × 人工降效系数 + 定额机械费 × 机械降效系数) × (1 + 管理费费率 + 利润率)

说明：
1. 海拔<2500m不计取高原降效
2. 以工程所在地海拔为准，以测量报告为依据
3. 降效系数可叠加冬季施工降效，但合计不超过0.50
4. 高原降效部分计取管理费、利润、规费、税金
```

## 九、费率计算示例

### 示例一：房屋建筑工程费率计算

```
某乌鲁木齐住宅项目，直接费（人工+材料+机械）= 5,000万元

管理费 = 5,000 × 28% = 1,400万元
利润   = 5,000 × 18% =   900万元
规费   = 人工费部分(假设1,200万) × 33.4% = 400.8万元
税金   = (5,000+1,400+900+400.8) × 9% = 693.07万元
安全文明施工费 = 5,000 × 3.5% = 175万元
                     
合计 = 5,000+1,400+900+400.8+693.07+175 = 8,568.87万元
```

### 示例二：人工费调差计算

```
某昌吉项目，定额人工费为1,000万元
昌吉二类工市场价为142元/工日
定额二类工基价为112元/工日

人工调增额 = 1,000 × (142-112)/112 = 267.86万元
人工调增部分税金 = 267.86 × 9% = 24.11万元

需注意：人工调增部分不计管理费、利润、规费
```
