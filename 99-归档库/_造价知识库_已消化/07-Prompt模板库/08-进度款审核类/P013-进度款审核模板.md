# P013-进度款审核模板

## 模板信息

| 字段 | 内容 |
|------|------|
| 模板编号 | P013 |
| 模板名称 | 进度款审核模板 |
| 分类 | 08-进度款审核类 |
| 版本 | V1.0 |
| 适用场景 | 施工过程中对施工单位提交的进度款申请进行审核，包括已完成工程量确认、进度款计算、质保金扣留、变更签证同步计量 |
| 前置条件 | 已有施工合同、经批准的施工进度计划、已审核的工程量清单、上期累计已付进度款数据 |

## 质量检查清单

- [ ] 已完成工程量现场核实确认（有现场工程师签字）
- [ ] 变更签证已获审批，手续完备
- [ ] 与施工进度计划对照，完成量在合理范围内
- [ ] 未发生重复计量
- [ ] 预付款抵扣符合合同约定方式和比例
- [ ] 质保金扣留累计未超过合同约定的上限金额
- [ ] 其他扣款有合同依据或协议
- [ ] 累计支付未超过合同总价
- [ ] 支付台账连续、无断期
- [ ] 审批流程已按制度要求执行（项目负责人/监理/审计签字齐全）
