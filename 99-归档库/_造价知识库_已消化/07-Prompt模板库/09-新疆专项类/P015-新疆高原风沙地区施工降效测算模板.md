## 质量检查清单

- [ ] 项目海拔数据和气象参数来源可靠（气象站/设计资料）
- [ ] 高原判定基准（2500m）正确
- [ ] 降效系数取值有当地定额或文件依据
- [ ] 人工降效和机械降效分别计算，未混淆
- [ ] 风沙防护方案与现场实际匹配
- [ ] 运输附加费有详细里程和路况分析
- [ ] 人员健康保障方案符合劳动保护法规
- [ ] 停工损失考虑正确（非施工方原因）
- [ ] 各项费用已与建设单位沟通或合同有约定
- [ ] 费用测算结果有书面报告供审批备案
