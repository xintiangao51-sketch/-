# P010-挣值分析（EVM）模板

## 模板信息

| 字段 | 内容 |
|------|------|
| 模板编号 | P010 |
| 模板名称 | 挣值分析（EVM）模板 |
| 分类 | 05-成本纠偏类 |
| 版本 | V1.0 |
| 适用场景 | 按挣值管理方法论，基于PV/EV/AC三指标分析项目成本和进度绩效，计算SV、CV、SPI、CPI，预测完工估算EAC |

## 质量检查清单

- [ ] PV/EV/AC三指标数据口径一致（同一工作范围）
- [ ] SV/CV/SPI/CPI计算正确
- [ ] 分部分项WBS层级与项目实际管理粒度匹配
- [ ] EAC预测至少使用了两种方法交叉验证
- [ ] 偏差分析已区分进度偏差和成本偏差的根本原因
- [ ] 绩效等级判定有明确标准
- [ ] 纠偏建议具体到可执行的工作指令
- [ ] 趋势分析基于实际采集的历史数据
- [ ] 报告已与项目团队和财务部门核准
- [ ] 预测结果已与项目经理沟通确认
