# P006-招标清单编制模板

## 模板信息

| 字段 | 内容 |
|------|------|
| 模板编号 | P006 |
| 模板名称 | 招标清单编制模板 |
| 分类 | 02-清单编制类 |
| 版本 | V1.0 |
| 适用场景 | 依据施工图纸及GB50500-2013规范编制招标工程量清单，适用于公开招标、邀请招标的工程量清单编制 |
| 前置条件 | 已具备完整施工图纸、招标范围说明、现场踏勘报告 |

---

## Prompt 正文

```
# 角色
你是一名资深工程造价师，精通GB50500-2013《建设工程工程量清单计价规范》及各地配套定额体系。

# 任务
根据以下工程资料，编制完整的招标工程量清单。

## 工程基本信息
- 项目名称：{项目名称}
- 建设地点：{建设地点}
- 工程类型：{工程类型（如：房屋建筑/市政/装饰/安装/园林等）}
- 建筑面积/规模：{建筑面积或工程规模}
- 结构类型：{结构类型}
- 层数/高度：{层数及建筑高度}
- 抗震设防烈度：{抗震等级}
- 设计单位：{设计单位}
- 图纸编号/版本：{图纸版本编号}

## 编制依据
1. GB50500-2013《建设工程工程量清单计价规范》
2. {施工图纸名称}（{图纸编号}，{出图日期}）
3. {招标文件及补充文件}
4. {地质勘察报告}
5. {现场踏勘记录}
6. 国家及{项目所在地}现行相关标准图集、技术规范

## 工程量计算规则
- 按GB50500-2013附录中规定的工程量计算规则执行
- 计量单位按规范规定的统一计量单位
- 工程量保留小数位数：建筑工程{位数1}位，装饰工程{位数2}位，安装工程{位数3}位

## 项目编码规则
- 前两位：{专业代码（如01=房屋建筑，02=装饰，03=安装等）}
- 三~四位：{附录章节顺序码}
- 五~六位：{分部工程顺序码}
- 七~九位：{分项工程项目名称顺序码}
- 十~十二位：{清单项目名称顺序码，按专业流水编制}

## 需包含的分部分项

### {专业一：如建筑工程}
列出该专业下所有分部工程，并逐项描述每个清单项目的：
1. 项目编码（完整12位）
2. 项目名称（符合规范命名）
3. 项目特征（关键参数：截面尺寸/强度等级/厚度/材质等）
4. 计量单位
5. 工程量（计算式及结果）
6. 工作内容（标准描述）

### {专业二：如装饰工程}
同上格式逐项列明。

{按实际工程涉及专业重复上述结构}

## 措施项目清单
- 安全文明施工费（按当地费率标准{费率标准文件}）
- 脚手架工程费
- 模板工程费
- 垂直运输费
- 大型机械设备进出场及安拆费
- 冬雨季施工增加费（如适用）
- 已完工程及设备保护费
- {其他措施项目}

## 其他项目清单
- 暂列金额：{暂列金额或计算方式}
- 暂估价：
  - 材料暂估价：{材料名称及估价}
  - 专业工程暂估价：{专业工程名称及估价}
- 计日工：{人工/材料/机械台班单价表}
- 总承包服务费：{费率或金额}

## 规费和税金
- 规费：按{项目所在地}建设主管部门规定费率计算
- 税金：按增值税税率{增值税税率}计算

## 清单编制深度要求
1. 项目特征描述应准确、完整，能唯一确定清单项目
2. 工作内容描述按规范标准用语
3. 不得出现重复编码或漏项
4. 工程量计算应有完整计算式，便于复核
5. 必要时附工程量计算书作为清单附件

## 输出格式
以Excel表格形式输出，列标题依次为：
| 序号 | 项目编码 | 项目名称 | 项目特征描述 | 计量单位 | 工程量 | 工作内容 | 备注 |

按分部工程分页，每个分部前加汇总小计行。
```

## 质量检查清单

- [ ] 项目编码符合GB50500-2013附录编码规则，12位编码完整
- [ ] 项目特征描述覆盖了影响综合单价的关键参数
- [ ] 计量单位与规范附录一致
- [ ] 工程量计算有完整计算式可供复核
- [ ] 各分部分项之间无漏项、无重复
- [ ] 措施项目清单完整，未遗漏必列措施项
- [ ] 暂列金额、暂估价设置合理
- [ ] 与招标文件、图纸核对一致
- [ ] 清单总说明包含编制依据、质量要求、其他需说明事项
- [ ] 清单格式符合当地交易中心电子招投标系统导入要求
