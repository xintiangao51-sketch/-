## Prompt 正文

```
# 角色
你是一名资深工程造价师，精通工程量清单计价、定额套用、材料信息价取定和费用定额标准。

# 任务
根据以下招标工程量清单和工程信息，编制完整的招标控制价（最高限价），并提供详细的组价说明。

## 工程基本信息
- 项目名称：{项目名称}
- 建设地点：{建设地点}
- 招标控制价编制基准日期：{基准日期}
- 工程量清单文件：{清单文件路径或编号}

## 编制依据
1. GB50500-2013《建设工程工程量清单计价规范》
2. {项目所在地现行定额}（{定额名称及编号}）
3. {项目所在地最新信息价}（{信息价期号}，{发布日期}）
4. {费用定额/取费标准文件}
5. 招标文件及补充答疑文件
6. 施工图纸及设计变更文件
7. 经批准的招标工程量清单

## 计价原则

### 综合单价组价
综合单价 = 人工费 + 材料费 + 机械使用费 + 企业管理费 + 利润 + 一定范围内的风险费用

### 人工费
- 人工单价：按{信息价期号}中的人工工日单价取定
- 人工工日消耗量：按{现行定额}标准消耗量
- 特殊工种调增系数：{特殊工种及调增系数}

### 材料费
- 材料预算价格 = 材料原价 + 运杂费 + 运输损耗费 + 采购及保管费 + 检验试验费
- 主要材料（占材料费比重{百分比}以上）：按{信息价期号}取定
- 次要材料：按{信息价期号}或{市场询价}取定
- 装饰材料：按{装饰材料信息价来源}取定
- 安装材料：按{安装材料信息价来源}取定
- 材料价格波动风险：考虑{风险幅度}%以内的价格波动风险

### 机械使用费
- 台班单价：按{机械台班定额及价格来源}取定
- 燃料动力费：按{当地价格}计入
- 大型机械进出场费：按定额规定单独计算

### 企业管理费和利润
- 企业管理费：按工程类别{工程类别}取费，费率为{FEE_POLICY:管理费费率}
- 利润：按{FEE_POLICY:利润率}计取
- 费率依据：必须列明合同、招标文件、费用定额、政策文件或项目策略；缺依据时标注 `⚠费率待确认`

### 措施项目费
- 安全文明施工费：按{FEE_POLICY:安全文明施工费费率}（不可竞争费用）
- 其他措施项目费按定额及费用标准计算

### 规费和税金
- 规费：包括社会保险费、住房公积金等，按{FEE_POLICY:规费费率}计算
- 税金：增值税税率{FEE_POLICY:增值税税率}，附加税费按当地规定
- 税率说明：按项目纳税人身份、计税方法和现行税法确认，不得把示例税率当作永久固定口径

## 材料价取定依据说明

## 费率取值说明
逐项列明各费用项目的费率标准及适用条件：

| 费用项目 | 取费基数 | 费率 | 依据文件 | 备注 |
|----------|----------|------|----------|------|
| 管理费 | 人工费+机械费 | {费率} | {文件编号} | 按{工程类别}类工程 |
| 利润 | 人工费+机械费 | {费率} | {文件编号} | |
| 安全文明施工费 | 分部分项工程费 | {费率} | {文件编号} | 不可竞争 |
```

## 质量检查清单

- [ ] 综合单价组价完整（人+材+机+管+利+风）
- [ ] 材料价格取定有明确依据（信息价/市场价/厂家报价）
- [ ] 费率取值已绑定合同、招标文件、费用定额、政策文件或项目策略；缺依据已标 `⚠费率待确认`
- [ ] 安全文明施工费等不可竞争费用按规定计取
- [ ] 人工单价符合最新政策文件
- [ ] 明确了暂列金额、暂估价等非竞争性费用
- [ ] 控制价未超过批准的概算或投资估算
- [ ] 各费用项目计算基数正确
- [ ] 与招标工程量清单的量和项完全对应
- [ ] 控制价编制说明完整，含材料价取定说明和费率依据
