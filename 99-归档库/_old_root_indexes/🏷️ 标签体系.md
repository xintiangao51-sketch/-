## 状态标签

| 标签 | 说明 |
|------|------|
| `#待完善` | 内容不完整，需补充 |
| `#待核实` | 数据/信息待核实 |
| `#精华` | 高质量核心内容 |
| `#归档` | 已归档，不再更新 |
| `#索引` | 索引/导航文件 |
| `#auto-generated` | 自动生成内容 |
| `#待填充` | 空目录/占位，待补充 |

## 标签使用规范

1. **每篇笔记至少打1个一级标签 + 1个二级标签**
2. **涉及具体项目的加项目标签**
3. **专业方向明确的加三级标签**
4. **状态标签按实际情况添加**
5. **标签放在 YAML frontmatter 的 tags 字段里**

### 示例

```yaml
---
tags: [造价, 造价/定额, 市政, 项目/特克斯, 精华]
---
```
