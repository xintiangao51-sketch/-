## 🚀 手动运行

```bash
cd D:/知识库/tools

# 信息价采集
python crawl_xj_info_price.py

# 全网知识采集 + 基础知识卡片生成
python crawl_policy.py

# 笔记 → 知识卡片
python auto_kc_generator.py

# 价格趋势更新
python material_price_updater.py

# 知识库健康度检查
python obsidian_optimizer.py
```
