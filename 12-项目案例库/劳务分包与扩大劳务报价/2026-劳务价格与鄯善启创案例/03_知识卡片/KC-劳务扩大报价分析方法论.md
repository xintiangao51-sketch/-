# KC-劳务扩大报价分析方法论

> 来源文件：`劳务扩大报价分析提示词.md`  
> 适用：新疆工业厂房劳务清包工/劳务扩大报价审查。

## 1. 分析主线

| 模块 | 核心动作 | 红线阈值 |
|---|---|---|
| 工程量核查 | 原始清单 vs 用户摘取量，按编码/名称/单位/工程量逐项比对 | 偏差率>±3%标⚠，>±10%标🔴 |
| 单价溯源 | 判断定额基价、投标价、下浮后单价、市场劳务价、混合取价 | 单价偏差>±15%重点关注 |
| 平米指标 | 土建/安装/合计按建筑面积闭合 | 工业厂房劳务占比通常18%~28% |
| 漏项检查 | 对照土建+安装标准工序全表 | 宁可多列，不可漏列 |
| 利润空间 | 中标价-主材-劳务-管理-规费税金 | 劳务占比<18%警惕报价偏低，>28%警惕甲方压价 |

## 2. 新疆本地化参数

| 参数 | 取值/规则 |
|---|---|
| 劳务上浮系数 | 内地参考价 × 1.15~1.30 |
| 冬歇期 | 11月中旬~次年3月中旬，约4个月 |
| 冬歇期降效 | 工期压缩/赶工时人工效率下降5%~10% |
| 普通工日工资 | 300~380元/工日 |
| 技术工日工资 | 400~550元/工日 |
| 特殊工日工资 | 500~650元/工日（焊工、架子工等） |
| 本项目下浮率口径 | 13%（用于鄯善启创案例） |

## 3. 工业厂房劳务扩大参考区间

| 厂房类型 | 结构 | 土建劳务元/m² | 安装劳务元/m² | 合计元/m² |
|---|---|---:|---:|---:|
| 单层厂房 | 钢排架 | 280~380 | 60~100 | 340~480 |
| 单层厂房 | 框架 | 320~420 | 70~110 | 390~530 |
| 多层厂房 | 框架 | 350~480 | 80~130 | 430~610 |
| 重型厂房 | 钢砼排架+吊车梁 | 380~520 | 90~140 | 470~660 |

## 4. 标准输出结构

1. 工程量核查结论：比对表、漏项清单、归类问题。
2. 单价分析结论：报价性质、单价溯源、下浮影响。
3. 劳务扩大报价准确性：平米指标、人工单价、漏项缺项、利润空间。
4. 综合结论与调整建议：必调项、建议调项、漏项补价、调整前后对比。

## 5. 执行约束

- 金额保留2位，工程量保留3位。
- 每个单价必须标注来源：本地知识库/新疆定额/联网市场价/历史项目/用户提供。
- 劳务清包工、劳务扩大、专业分包必须分清口径。
- 安装边界默认仅含水暖电人工+辅材+小型机具，不含设备及设备安装费；特殊设备调试另列。
