## 一、抖音评论话术（针对视频评论区）

### 通用评论话术
```
老师傅说得对。我就是专门帮施工队算报价的，见过太多漏项亏钱的。
有需要的兄弟，私信发我"报价"，我把《现场报价缺项检查表》发你，
以后报价前对着查一遍，能少亏不少。
```

### ID-001：报价26万被压至16万
```
兄弟，你这26万压到16万太狠了。回本了吗？
我看报价里有些措施费和损耗可能没算进去。
发我"报价"，给你一份报价自查表，以后报价前对着查一遍。
```

### ID-003：小项目亏损率41.6%
```
41.6%的亏损率，这是把措施费、机械转场、材料损耗全漏了吧？
我专门帮施工队算报价的，发我"报价"，给你发份《报价缺项检查表》，
以后报价前对着这10条查一遍，能少亏很多。
```

### ID-008：简易报价亏损率高
```
简易报价最容易漏的就是措施费和材料损耗，新疆运距远损耗更大。
我这边有个报价自查表，10个必填项填完就知道漏没漏。
发我"报价"，免费发你。
```

### ID-026：甲方先干后补不认账
```
这种最坑。干之前没留证据，换个人就不认了。
我有个签证证据清单，25项一一核对，该留什么证据、什么时候留，全列清楚了。
发我"签证"，免费发你。
```

## 二、贴吧回复话术（针对帖子主题）

### ID-011：审计扣除规费
```
审计扣规费这事不少见。关键看合同怎么约定的，规费是硬性费用不能随便扣。
我帮施工队处理过不少这类结算纠纷，有需要的话可以帮你看看结算单。
发我"结算"，免费发你一份《结算资料检查清单》。
```

### ID-015：口头约定反被索赔18万
```
口头约定在工程上等于没有约定。18万买个教训太贵了。
以后签合同前最好找人看一下关键条款。我这边可以帮你快速审查合同风险点。
发我"合同"，免费发你一份《合同风险自检表》。
```

### ID-039：阿克苏结算核减155万
```
155万的核减，你那个结算书肯定有漏洞被审计抓住了。
这种大额结算争议，建议你把结算书发我，我帮你看看哪些项目还能争回来。
```
*（注意：这条适合私聊，不适合直接回复）*

## 三、微信群话术（发到施工群/工程群）

### 进群打招呼（不直接推服务）
```
刚进群，大家好。我是专门帮施工队做报价和签证资料的。
在新疆做了好几年造价了，大家有报价拿不准、签证不会写的可以问我，
免费的，就当交个朋友。
```

### 群内日常互动（有人在群里问问题时）
```
A：这个活200一平方能做吗？
→ 回复：得看什么活、什么做法、材料谁出。200的价在新疆某些地方可能刚好保本，但措施费和运费要单算。

A：签证单怎么写？甲方让加个东西不给加价。
→ 回复：签证单核心就三条：1.什么事 2.谁让干的 3.干了多少。证据一定要留齐，微信记录也算。

A：结算审计来了，说我们不合规要扣钱。
→ 回复：审计扣钱要有依据。让他们出书面意见，一项一项对，合同里有的谁也不能扣。
```

### 群内分享（每周1次，不刷屏）
```
分享一个好东西——我整理的《现场报价缺项检查表》。
报价前对着这10个问题查一遍，基本能避免漏项亏钱。
需要的群友发我"报价"，我私发给你。
```

## 四、企业微信/个人微信1对1私聊话术

```

我这边可以帮您快速复核一下报价有没有漏项、材料价合不合理。
（我自己就是新疆做造价的，定额和当地市场价都熟）
方便的话可以聊聊？免费的，就是交个朋友。
```
