# 新疆造价信息自动采集脚本 · 使用说明

> **脚本路径**：`07-AI工具与技能/daily_crawl.py`
> **状态**：✅ 2026-06-24 实测可运行，5 分钟内抓取 130 条记录
> **维护**：Codex（鼎梁筑）

## 一、脚本能力

### 数据源（4 个，2026-06-24 实测可达）

| 数据源 | 路径 | 实测状态 |
|--------|------|---------|
| 自治区住建厅 | `/xjzjt/c113382/zwgk_list{_2,3,4,5}.shtml` | ✅ 72 条/分页 |
| 自治区交通运输厅 | `/xjjtysj/zjxx/nav2_list.shtml` | ✅ 稳定 |

| 兵团 | http://www.xjbt.gov.cn | ⚠ 占位 |

### 智能分类（8 类）

- 造价管理办法 / 国标 / 智能建造 / 装配式
- 定额 / 勘误 / 人工单价 / 材料信息价

### 输出

1. **日报**：`01-造价核心/造价信息日报_YYYY-MM-DD.md`（按数据源 + 关键词双视图）
2. **原始数据**：`01-造价核心/采集日志/raw_YYYYMMDD.json`（结构化 130+ 条记录）
3. **控制台日志**：实时输出采集进度

## 二、使用方法

### 基本用法

```bash
# 1. 全量采集（推荐每日 08:00 跑一次）
python D:\知识库\07-AI工具与技能\daily_crawl.py

# 2. 快速模式（只跑住建厅 + xjzj.com，5 分钟内完成）
python D:\知识库\07-AI工具与技能\daily_crawl.py --quick

# 3. 只看自某日以来的新发布
python D:\知识库\07-AI工具与技能\daily_crawl.py --since 2026-06-20

# 4. 只采集，不生成日报（调试用）
python D:\知识库\07-AI工具与技能\daily_crawl.py --no-report
```

### Windows 任务计划（每日 08:00）

```powershell
# 创建任务
schtasks /create /tn "新疆造价信息采集" /tr "python D:\知识库\07-AI工具与技能\daily_crawl.py --quick" /sc daily /st 08:00 /f

# 查看任务
schtasks /query /tn "新疆造价信息采集"

# 删除任务
schtasks /delete /tn "新疆造价信息采集" /f
```

### Linux/Cron（每日 08:00）

```bash
# 编辑 crontab
crontab -e

# 添加
0 8 * * * cd /path/to/vault && python 07-AI工具与技能/daily_crawl.py --quick >> 01-造价核心/采集日志/cron.log 2>&1
```

## 四、脚本结构

```
daily_crawl.py
├── 配置（数据源、关键词、输出路径）
├── 工具函数（fetch、classify、log）
├── 住建厅爬虫（crawl_zjt）→ 多分页
├── 交通厅爬虫（crawl_jtyst）→ 单页
├── xjzj.com 爬虫（crawl_xjzj）→ 首页索引
├── 日报生成（generate_report）
└── 主函数（main）
```

## 六、故障排查

| 现象 | 原因 | 解决 |
|------|------|------|
| 住建厅 0 条 | URL 路径变更 | 检查 [zjt.xinjiang.gov.cn](https://zjt.xinjiang.gov.cn) 首页，更新 `list_paths` |
