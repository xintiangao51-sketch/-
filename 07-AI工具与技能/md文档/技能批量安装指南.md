## 三、安装路径说明

### 安装后技能的物理存储位置

Claude Code 安装技能后，文件会存放在：

```
C:\Users\20752\AppData\Local\Claude-3p\local-agent-mode-sessions\
  skills-plugin\{uuid}\
    {uuid}\skills\
      <技能名称>\
        SKILL.md
        ...
```

> **注意：** 这个路径是可变的（`{uuid}` 部分每次创建 session 都不一样），所以不适合硬编码共享。

## 四、共享方案

### 方案 A：保存为 Cowork 技能（推荐 · 持久化）

Cowork 技能是**持久化存在**的，不受 session 影响。安装到 Cowork 后，任何 Agent 都可以通过 `/skills` 菜单调用。

**操作方法：** 在我的 Cowork 提示中执行 `/save-skill` 命令，将安装好的技能 SKILL.md 内容保存为 Cowork 的持久技能。

> 当前 session 中已经可用的造价技能列表：

> - `anthropic-skills:construction-budget-specialist`
> - `anthropic-skills:construction-finance-controller`
> - `anthropic-skills:construction-legal-contract-reviewer`
> - `anthropic-skills:construction-civil-engineer-cn`
> - `anthropic-skills:construction-site-manager`
> - `anthropic-skills:construction-quantity-surveyor`
> - `anthropic-skills:cost-data-analyze`

> - `anthropic-skills:knowledge-base-rag-obsidian`

这些技能我已经可以直接在当前 Cowork 环境中调用。要让它们对 Codex 和 Hermes Agent 也可用，需要做以下操作。

### 方案 B：共享给 Hermes Agent

**Hermes Agent 的 CLAUDE.md 中引用技能路径。**

1. 在 Codex 里执行安装后，找到技能目录 `~/.claude/skills/`
2. 在 Hermes Agent 的 `CLAUDE.md` 策略配置中，添加：

```yaml
skills_path: "C:\Users\20752\.claude\skills\"
```

或者直接把需要的技能 SKILL.md 内容复制到 Hermes Agent 可读取的目录。

### 方案 C：共享给 Codex（同一套环境）

**Codex 和 Claude Code 在同一个机器上共享同一个配置目录。**

```bash
# 在任一 Codex 实例中，技能目录是共享的
# 以下文件包含了安装后所有技能的索引
~/.claude/skills/
```

安装一次，所有 Codex 实例都能用。**不需要重复安装。**

## 五、总结：最优方案

```
┌─────────────────────────────────────────────┐
│  你的目标：在 Codex / Hermes / Cowork        │
│  三端都能调取同一套造价技能                    │
├─────────────────────────────────────────────┤
│                                             │
│  步骤 1: 在 Claude Code 执行上述安装命令       │
│    → 技能文件写入 ~/.claude/skills/           │
│                                             │
│  步骤 2: 验证安装成功                         │
│    输入 /skills 查看列表                      │
│                                             │
│  步骤 3: Cowork 端已经可以直接用               │
│    调用前缀：anthropic-skills:<技能名>         │
│                                             │
│  步骤 4: Hermes Agent 端                      │
│    在其 CLAUDE.md 里添加 skills_path 指向      │
│    C:\Users\20752\.claude\skills\            │
│    或者直接复制需要的 SKILL.md 到 Hermes 目录   │
│                                             │
└─────────────────────────────────────────────┘
```

**一句话答案：** 在 Claude Code 装一次，技能文件落盘到 `~/.claude/skills/`，Cowork 已经能直接用（当前 session 已有全部 anthropic-skills 前缀的技能），Hermes Agent 指向同一路径即可复用，不需要装三遍。
