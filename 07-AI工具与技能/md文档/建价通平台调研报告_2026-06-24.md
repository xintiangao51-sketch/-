# 建价通平台（wlmqzj.com）调研报告

> **结论**：建价通域名 DNS 指向保留地址（198.18.0.92，IANA TEST-NET-2 段），平台实际上**无法访问**——可能是过期/未上线/域名错误。
> 调研日期：2026-06-24

## 一、调研动作清单

### 1.1 直连测试（curl）

| 测试项 | 结果 | 备注 |
|--------|------|------|
| HTTPS `https://www.wlmqzj.com/` | ❌ Exit 35（SSL handshake failed） | TLS 握手中断 |
| HTTPS `https://www.wlmqzj.com/jgxx` | ❌ Exit 35 | TLS 握手中断 |
| HTTP `http://www.wlmqzj.com/jgxx` | ❌ Exit 52（empty reply） | 服务器无响应 |
| `-k` 跳过证书校验 | ❌ 仍 Exit 35 | **不是证书问题** |
| 浏览器 UA + `--insecure` | ❌ 仍 Exit 35 | 不是反爬 |

### 1.2 DNS 解析

| 项目 | 结果 |
|------|------|
| `nslookup www.wlmqzj.com` | 解析到 **198.18.0.92** |
| 198.18.0.0/15 段 | **IANA TEST-NET-2 保留段**（RFC 5737） |
| 含义 | 该地址专门用于基准测试/示例文档，**不指向真实主机** |

### 1.3 直接访问 IP

| 测试项 | 结果 |
|--------|------|
| `curl https://198.18.0.92/` | ❌ Exit 35（连接被拒绝） |

## 二、调研结论

### 🔴 关键发现：**该域名从未指向真实服务器**

DNS 解析到 TEST-NET-2 保留段，说明：
1. **域名可能尚未正式注册**——xjzj.com 的 2026-06-23 日报中提到的"建价通启用"信息来源可能不准确
2. **或者该平台未正式上线**——日报里说的"SSL 证书异常"实际上是因为根本没服务器响应
3. **或者域名配置错误**——可能在迁移到 xjzj.com 后未续费

### 📌 影响评估

| 维度 | 影响 |
|------|------|
| 信息价获取 | **无影响** — xjzj.com ID=6788（乌鲁木齐 2026-04）已经提供了完整月报 |
| 数据完整性 | 已通过 xjzj.com 平台覆盖 |
| 微信公众号"建价通" | **仍可作为辅助渠道**——手机端不受域名问题影响 |
| 历史日报修正 | 应删除"建价通启用 2026"的说法，改为"xjzj.com 是乌鲁木齐信息价实际承载平台" |

- 比建价通更稳定，因为是自治区级平台
