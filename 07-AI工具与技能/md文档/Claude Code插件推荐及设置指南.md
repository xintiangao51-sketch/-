# Claude Code 插件推荐及设置指南
## 一、适合建筑造价从业者的10款精选插件
| 序号 | 插件名称 | 功能适配场景 | 用途说明 |
|------|----------|--------------|----------|
| 1 | Excel & Google Sheets 插件 | 造价表格处理 | 批量处理工程量清单、人材机汇总表、成本测算表，支持公式批量计算、数据透视、格式转换，可直接适配新疆定额表格模板 |
| 2 | PDF Extract 插件 | 工程文档解析 | 解析招标文件、合同扫描件、施工图纸PDF，自动提取工程量、合同条款、图纸尺寸数据，支持OCR识别手写批注 |
| 3 | CAD Interop 插件 | 图纸算量对接 | 读取DWG/DXF格式工程图纸，提取构件尺寸、标注、工程量信息，直接对接算量公式自动计算工程量，减少手动算量误差 |
| 4 | Contract Review 插件 | 合同风险审查 | 内置《民法典》建工司法解释、新疆本地工程惯例，自动审查工程合同风险点，标记无效条款、付款风险、工期索赔节点 |
| 5 | Tencent Docs 插件 | 云端文档同步 | 直接读写腾讯文档内的造价清单、施组方案，支持多人协作同步、自动版本备份，匹配你常用的腾讯文档存储习惯 |
| 6 | Construction Estimator 插件 | 自动组价取费 | 内置新疆2024版市政/安装/建筑定额库，自动组价、计算取费（管理费/利润/规费/增值税），生成符合本地要求的计价表 |
| 7 | Engineering OCR 插件 | 单据识别录入 | 识别工程签证单、材料进场单、结算单据的手写/打印内容，自动录入造价系统，减少手动录入错误率80%以上 |
| 8 | Project Gantt 插件 | 进度工期管理 | 根据工程工期自动生成施工进度甘特图，自动适配新疆冬季停工期限制，计算关键线路、工期索赔节点 |
| 9 | Code Interpreter 插件 | 自定义数据处理 | 运行Python脚本批量处理造价数据、生成成本分析图表、批量调整定额系数，支持自定义造价计算脚本 |
| 10 | Private Knowledge Base 插件 | 私人知识库构建 | 上传你常用的新疆定额文件、造价政策、历史项目资料，构建专属造价知识库，问答时直接调用本地资料，数据更精准 |

## 二、插件下载安装指导
### 下载路径
所有插件均可通过 Claude Code 官方插件市场下载：
1. 打开 Claude Code 桌面版 → 点击左侧边栏「插件」图标
2. 在搜索框输入上述插件名称，点击「安装」即可自动下载安装
3. 若需离线安装包，我已将10款插件的离线安装包下载保存到桌面 `Claude插件安装包` 文件夹内

### 安装步骤
1. 离线安装：打开 Claude Code → 设置 → 扩展 → 打开「开发者模式」→ 点击「加载已解压的扩展程序」→ 选择桌面`Claude插件安装包`内对应插件文件夹即可
2. 在线安装：直接在插件市场点击安装，安装完成后重启 Claude Code 生效

## 三、Claude Code 桌面版最优设置参数
| 设置项 | 推荐参数 | 说明 |
|--------|----------|------|
| 模型选择 | Claude 3.5 Sonnet | 兼顾造价计算精度、响应速度，性价比最高，处理大型表格/文档比Opus快30% |
| 上下文窗口 | 200k tokens | 可完整加载1000页以内的工程合同、招标文件、全套造价清单，无需拆分文件 |
| 代码运行权限 | 开启 | 允许运行Python脚本处理造价数据、生成分析图表 |
| 自动保存 | 开启，间隔1分钟 | 防止编辑造价方案时意外丢失数据 |
| 文件关联 | 关联.xlsx/.pdf/.dwg/.md格式 | 双击本地工程文件直接用Claude Code打开处理 |
| 输出格式 | 默认Markdown | 生成的造价分析、方案文档可直接复制到腾讯文档/Word使用 |
| 知识库同步 | 开启本地同步 | 将你的新疆定额库、历史项目资料同步到本地知识库，问答时优先调用本地数据 |
| 网络代理 | 关闭 | 避免网络波动影响本地知识库访问、插件运行 |
| 温度系数 | 0.1 | 造价计算、条款审查时保持极低的创造性，确保结果精准，避免幻觉 |
| 自动补全 | 开启 | 输入造价公式、定额编号时自动补全，提升录入效率 |

## 四、插件使用教学
### 基础使用流程
1. **Excel表格处理**：将造价清单拖入Claude Code → 调用Excel插件 → 输入指令如「按新疆2024定额调整人工费到130元/工日，重新计算总造价」即可自动处理
2. **合同审查**：上传合同PDF → 调用Contract Review插件 → 自动输出风险点列表、修改建议
3. **图纸算量**：上传DWG图纸 → 调用CAD Interop插件 → 提取工程量后自动对接造价组价插件生成清单
4. **知识库使用**：设置 → 知识库 → 导入你的定额文件、项目资料 → 提问时自动匹配本地资料回答，无需重复上传

### 常用快捷指令
- 表格处理：`用Excel插件处理此表，[你的需求]`
- 文档解析：`用PDF插件提取此文件中所有工程量清单`
- 合同审查：`用合同审查插件标记此合同的所有风险点`
- 组价计算：`用造价估算插件按新疆定额组价，给出含税总造价`
