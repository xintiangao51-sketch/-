## 6. 共享架构卡片

```mermaid
flowchart LR
  A[Codex 主工作区] --> B[技能实体目录]
  A --> C[MCP 服务实体]
  A --> D[知识库沉淀]
  B --> E[Codex Junction]
  B --> F[Claude Code Junction 或路径复用]
  B --> G[Hermes 中文端 Junction 或路径复用]
  C --> F
  C --> G
  D --> E
  D --> F
  D --> G
```

共享原则：Codex 作为主安装源，技能实体不重复复制；Claude Code 和 Hermes Agent 优先通过 Junction、配置路径或只读引用复用。MCP 服务优先复用同一套 Node、Python、npm-global、mcp-servers 资产；各 Agent 只维护自己的配置入口和密钥环境变量。
