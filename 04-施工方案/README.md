## 🏔️ 新疆特殊施工条件

| 条件 | 适用时期/地区 | 费用影响 |
|:-----|:------------|:---------|
| 冬季施工 | 11月~次年3月（北疆5~6个月） | 人工降效 ×1.06~1.25 |
| 高海拔 | >1500m（喀什/塔县等） | 人工/机械高原降效 |
| 风沙 | 沙漠边缘地区 | 防护+运输附加费 |
| 抗震设防 | 全疆7~8度区 | 结构增量 |

> 详细系数见 [[00-工作台/常用参数-新疆2024|常用参数速查]]
