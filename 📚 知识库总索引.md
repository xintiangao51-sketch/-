## 🔍 动态查询

### 全部文件按目录统计

```dataview
TABLE length(rows) as "文件数
FROM ""
WHERE file.folder != ".obsidian"
GROUP BY split(file.folder, "/")[0]
SORT split(file.folder, "/")[0]
```

### 最近30天修改的文件

```dataview
TABLE file.mtime as "修改时间", file.folder as "目录"
FROM ""
WHERE file.mtime >= date(today) - dur(30 days) AND file.folder != ".obsidian"
SORT file.mtime DESC
LIMIT 30
```

### 按标签检索

```dataview
TABLE file.folder as "目录", file.mtime as "修改时间"
FROM #造价
SORT file.mtime DESC
```
