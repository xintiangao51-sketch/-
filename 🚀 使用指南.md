# 🚀 知识库使用指南

> 5分钟掌握知识库核心功能

---

## 1. 导航

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 快速打开 | `Ctrl+O` | 输入文件名直达 |
| 全局搜索 | `Ctrl+Shift+F` | 搜索文件内容 |
| 关系图谱 | `Ctrl+G` | 可视化链接关系 |
| 返回上一页 | `Alt+←` | 浏览器式后退 |
| 前进 | `Alt+→` | 浏览器式前进 |

## 2. 核心功能

### 双向链接
```markdown
在任何笔记中写 [[文件名]]，就建立了链接。
被链接的文件会自动显示"反向链接"面板。
```

### 标签
```markdown
在笔记的 YAML frontmatter 中添加：
---
tags: [造价, 造价/定额, 市政]
---
或在正文中写 #造价 都可以。
```

### 搜索
- `Ctrl+O` — 按文件名搜索
- `Ctrl+Shift+F` — 按内容搜索
- 支持正则表达式

## 3. 文件夹结构

每个主文件夹都有中文「总览」索引文件，包含：
- 子目录导航
- 文件清单（Dataview 自动生成）
- 反向链接
- 标签分布

## 4. 新建笔记规范

### 必须做的
1. **选择正确的文件夹** — 放到对应的分类目录下
2. **添加标签** — 至少1个一级标签 + 1个二级标签
3. **添加链接** — 关联到相关笔记

### 推荐的
- 使用模板（Templater 插件）
- 添加创建日期
- 添加摘要/说明

## 5. 常用操作

### 从模板新建
1. `Ctrl+P` → 输入 "Templater"
2. 选择模板
3. 填写内容

### 查看某标签下所有文件
1. 点击笔记中的标签
2. 或在搜索框输入 `tag:#造价`

### 查看关系图谱
1. `Ctrl+G` 打开全局图谱
2. 点击节点查看该文件的链接关系
3. 颜色分组已配置（造价=橙色，合同=绿色，施工=黄色）

## 6. Dataview 查询

知识库中大量使用 Dataview 插件自动生成索引。

### 常用查询示例

**查看所有造价相关文件：**
```dataview
TABLE file.folder as "目录"
FROM #造价
```

**查看最近修改的文件：**
```dataview
TABLE file.mtime as "修改时间"
FROM ""
SORT file.mtime DESC
LIMIT 10
```

**按标签统计：**
```dataview
TABLE length(rows) as "数量"
FROM ""
FLATTEN file.tags as tag
GROUP BY tag
SORT length(rows) DESC
```
