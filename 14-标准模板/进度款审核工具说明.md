# 进度款审核工具说明 —— 进度款审核全流程工具

> 版本：v1.0 | 更新日期：2026-06-19 | 适用：新疆地区建设工程进度款审核

---

## 概述

进度款审核是工程造价管理中的核心工作之一。本文件提供进度款审核的全流程工具、模板和检查清单，帮助造价工程师规范、高效地完成进度款审核工作。

---

## 一、进度款审核流程说明图

### 1.1 审核流程图

```
┌─────────────────────────────────────────────────────────────────────┐
│                       进度款审核全流程图                               │
└─────────────────────────────────────────────────────────────────────┘

每月25日
    │
    ▼
施工方提交 → ① 进度款支付申请表
             ② 已完工程量计算书（附计算过程）
             ③ 材料进场报验单（附进场清单）
             ④ 隐蔽工程验收记录（已签认）
             ⑤ 检验批/分项工程质量验收记录
    │
    ▼
初审阶段 → ① 形式审查：资料是否齐全、签字盖章是否完整
             ② 时效审查：是否在合同约定的期限内提交
             ③ 格式审查：是否按合同约定格式填写
    │
    ▼
现场计量 → ① 到场核实已完成工程量
             ② 对可计量的项目逐项量测
             ③ 对隐蔽工程复核已签认记录
             ④ 对未完成工程和缺陷工程标注扣减
             ⑤ 现场拍照留存证据
    │
    ▼
审核计算 → ① 逐项审核申报量与实际完成量
             ② 按合同单价计算应付金额
             ③ 扣减甲供材费用（按实际领用量）
             ④ 扣减预付款（按合同约定比例扣回）
             ⑤ 扣减质保金（按合同约定比例）
             ⑥ 扣减罚款/违约金（如有）
             ⑦ 汇总本期应付金额
    │
    ▼
内部复核 → ① 审核人与复核人交叉复核
             ② 重点审核大额项目和新增项目
             ③ 复核扣款项目的准确性
    │
    ▼
双方确认 → ① 与施工方核对审核结果
             ② 对争议项协商解决
             ③ 签署工程计量审核单
             ④ 签署进度款支付证书
    │
    ▼
审批支付 → ① 内部审批流程
             ② 财务复核
             ③ 业主审批
             ④ 支付进度款
    │
    ▼
资料归档 → ① 本期审核全套资料归档
             ② 更新进度款支付台账
             ③ 准备下期审核
```

### 1.2 各环节时间节点控制

| 环节 | 参考时间 | 责任方 | 依据 |
|---|---|---|---|
| 施工方提交 | 每月25日前 | 施工方 | 合同约定 |
| 初审 | 收到后1天内 | 造价工程师 | — |
| 现场计量 | 初审后2天内 | 造价工程师+监理 | — |
| 审核计算 | 现场计量后3天内 | 造价工程师 | — |
| 内部复核 | 审核后1天内 | 造价负责人 | — |
| 双方确认 | 复核后2天内 | 造价师+施工方 | — |
| 审批支付 | 确认后5天内 | 业主 | 合同约定 |
| **总周期** | **约14天** | — | — |

> **新疆特殊提示：** 南疆三地州（喀什、和田、克州）因地理距离远、交通不便，建议适当延长现场计量环节的时间，预留1-2天路途时间。

**审核汇总：**

| 项目 | 金额(元) |
|---|---|
| 本期审核应付（A） | ________ |
| 扣减：预付款（B = A × 预付款扣回比例） | ________ |
| 扣减：甲供材（C） | ________ |
| 扣减：质保金（D = A × 质保金比例） | ________ |
| 扣减：罚款/违约金（E） | ________ |
| **本期实际应付 = A - B - C - D - E** | ________ |

### 2.2 核心公式

```
本期审核量 = 现场实际核实的完成量（非施工方申报量）
本期应付 = 本期审核量 × 合同单价
累计已完成金额 = Σ(各期审核应付之和)
累计支付率 = 累计已完成金额 / 合同总价 × 100%
完成比例 = 本期审核量 / 清单总量 × 100%
本期实际应付 = 本期应付 - 预付款扣回 - 甲供材扣回 - 质保金 - 罚款
```

### 2.3 使用说明

1. 按清单逐项审核，确保每一项目都有对应的审核记录
2. 本期申报量与本期审核量存在差异时，须在"差异说明"栏注明原因
3. 对施工方多报部分，附现场照片和测量记录作为审核依据
4. 采取清单总量控制的累计审核量不得超过清单总量
5. 季节性施工项目（如冬季施工）单独列项审核

## 五、甲供材扣回计算表

### 5.1 用途说明

对甲方供应材料进行分月扣回管理，确保甲供材按合同约定及时扣回。

### 5.2 模板结构

| 材料名称 | 规格型号 | 单位 | 合同价(元) | 上期累计领用量 | 本期领用量 | 累计领用量 | 本期扣回(元) | 累计扣回(元) | 合同总量 | 领用比例 |
|---|---|---|---|---|---|---|---|---|---|---|
| 钢筋 | HRB400 φ16-25 | t | 3,500 | 180 | 25 | 205 | 87,500 | 717,500 | 650 | 31.54% |
| 水泥 | PO42.5 | t | 420 | 520 | 68 | 588 | 28,560 | 246,960 | 2,800 | 21.00% |
| 商品混凝土 | C30 | m³ | 360 | 1,850 | 220 | 2,070 | 79,200 | 745,200 | 8,600 | 24.07% |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

### 5.3 核心公式

```
本期扣回 = 本期领用量 × 合同单价
累计扣回 = Σ(各期扣回之和)
领用比例 = 累计领用量 / 合同总量 × 100%
超领预警：当领用比例 > 完成比例 + 损耗率 时预警
```

### 5.4 甲供材管理要点

| 要点 | 说明 |
|---|---|
| 领用程序 | 施工方填写领用单→工长签字→材料员签字→造价员备案 |
| 损耗率控制 | 钢筋3%、混凝土1.5%、水泥2%，超耗由施工方承担 |
| 超耗处理 | 超领部分按合同价的1.2倍在进度款中扣回 |
| 核销核对 | 每月与施工方核对甲供材领用台账，双方签字确认 |
| 竣工结算 | 竣工时进行甲供材总核销，多退少补 |
| 新疆特殊 | 南疆项目因运距远，建议钢筋、水泥提前备料1-2个月 |

## 八、进度款支付台账模板

### 8.1 模板结构

| 期次 | 本期申报(万元) | 本期审核(万元) | 审核增减(万元) | 甲供材扣回(万元) | 预付款扣回(万元) | 质保金(万元) | 扣款(万元) | 本期支付(万元) | 累计支付(万元) | 累计支付率 | 备注 |
|---|---|---|---|---|---|---|---|---|---|---|---|
| 第1期 | 320.00 | 285.00 | -35.00 | 42.00 | — | 8.55 | — | 234.45 | 234.45 | 9.38% | 预付款未开始扣 |
| 第2期 | 450.00 | 410.00 | -40.00 | 58.00 | — | 12.30 | 2.00 | 337.70 | 572.15 | 22.88% | — |
| 第3期 | 520.00 | 480.00 | -40.00 | 65.00 | 48.00 | 14.40 | — | 352.60 | 924.75 | 36.99% | 开始扣预付款 |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| **合计** | | **2,500.00** | | | | | | | | |

### 8.2 核心公式

```
审核增减 = 本期审核 - 本期申报
本期支付 = 本期审核 - 甲供材扣回 - 预付款扣回 - 质保金 - 扣款
累计支付率 = 累计支付 / 合同总价 × 100%
累计支付 ≤ 合同总价 × (1 - 质保金比例)
```

### 8.3 台账管理要点

| 要点 | 说明 |
|---|---|
| 连续编号 | 按期次连续编号，不漏不缺 |
| 数据闭环 | 台账数据与审核表、财务报表数据一致 |
| 定期对账 | 每月与施工方、财务对账一次，确保各方数据一致 |
| 预警设置 | 当累计支付率达到85%时预警，提醒注意超额支付风险 |
| 归档保存 | 台账为重要财务凭证，保存期限不少于10年 |
