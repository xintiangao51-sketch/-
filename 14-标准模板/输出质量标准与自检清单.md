## 一句话摘要
造价成果输出前10项自检+零信任标注规则+合同审查6条款框架+三级风险标注。

## 标准造价任务工作流

```
1. 判断任务类型
   └→ 清单/成本测算/图纸提量/审图/变更/合同审查/投标分析

2. 明确项目上下文
   └→ 项目名称/日期/地区/专业/计价依据/费率依据

3. 读取或解析资料
   └→ Excel/Word/PDF/CAD/知识库

4. 形成结构化结果
   └→ 结论先行 → 明细 → 风险 → 待确认 → 可执行动作

5. 输出前执行10项自检 ✓
```

## 10项输出前自检清单

| # | 检查项 | 通过标准 |
|---|--------|----------|
| 1 | **费率引用** | 全部费率引自FEE_POLICY.md，未写死数字 |
| 2 | **定额版本** | 明确标注所引定额版本号和施行日期 |
| 3 | **地区参数** | 人工单价/冬施/海拔系数对应当前地区 |
| 4 | **四维拆解** | 成本已拆为人工/辅材/主材/机械 |
| 5 | **偏差标注** | 任一维度偏离±5%已标注颜色 |
| 6 | **风险标注** | 高/中风险项已用红/黄标注 |
| 7 | **待确认标记** | 缺定额/缺市价/缺图纸→已标记⚠ |
| 8 | **公式验证** | 管理费/利润/规费/税金计算基数正确 |
| 9 | **合规检查** | 未越六不红线 |
| 10 | **格式标准** | 数字精度正确(m³=2位,t=3位,元=2位) |

## 零信任造价标注规则

当以下任一情况发生时，必须输出标注而非猜测：

| 情况 | 标注格式 | 说明 |
|------|----------|------|
| 缺定额子目 | ⚠待查：【定额名称+编号】 | 新疆2024定额中未找到匹配项 |
| 缺市场价 | ⚠待询：【材料名+规格+地区】 | 信息价/市场价未覆盖 |
| 缺图纸信息 | ⚠待补：【图纸编号+缺失内容】 | 图纸不详/未提供 |
| 术语不确定 | ⚠待确认：【术语+可能含义】 | 多义词/非标准术语 |

## 合同审查6条款框架（P016）

| 条款 | 审查要点 | 风险级别 |
|------|----------|:---:|
| **价格条款** | 综合单价/可调价/固定总价→判断合理性 | 🔴 |
| **变更条款** | 变更估价三原则是否写入+调价公式 | 🔴 |
| **索赔条款** | 28天时效+同期延误分配+窝工标准 | 🟡 |
| **结算条款** | 计量周期+支付比例+质保金(3%) | 🔴 |
| **保修条款** | 保修期起算+保修范围+费用承担 | 🟡 |
| **违约条款** | 违约金上限+免责条款+争议解决 | 🟡 |

### 三级修改建议
- **必须修改**：可能导致经济损失或合同无效的条款
- **建议修改**：可能产生争议但非致命的条款
- **提示关注**：格式/期限/程序性优化建议

## 定额匹配四级策略

| 级别 | 方法 | 适用场景 |
|:---:|------|----------|
| L1 | 精确匹配（编码直查） | 定额编号已知 |
| L2 | 类型+参数匹配 | 关键词+尺寸容差 |
| L3 | 模糊匹配（rapidfuzz） | 文本相似度<80% |
| L4 | 用户手动选择 | 前三级均失败 |

## 工程量计算核心规则
- 梁：V = 宽×高×长 (m³)，模板 = (宽+2×高)×长
- 板：V = 面积×厚度，模板 = 底模+侧模
- 柱：V = 截面×层高，模板 = 周长×层高
- 钢筋：按图纸逐根计算 + 钢筋表汇总（t，3位小数）
- 精度：m³=2位，t=3位，m²=2位，m=2位，元=2位
