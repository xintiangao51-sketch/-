**费率按专业 × 计税方式双轴查询**：企业管理费区间表（GB 50500-2024附件1）是当前最权威依据。利润/规费/费率表见下文。

## 费用结构总览（按 GB 50500-2024 计算程序）

```
一、分部分项工程费
   = ∑(工程量 × 综合单价)
   其中综合单价 = 人工费 + 材料费 + 机械费 + 企业管理费 + 利润

二、措施项目费
   = (一)单价措施 + (二)总价措施（含安全文明施工费）

三、其他项目费 = 暂列金额 + 暂估价 + 计日工 + 总承包服务费

四、规费（原单独列项，2024版并入管理费或措施费）

五、增值税 = (一+二+三+四-可抵扣) × 9%

六、建设工程造价 = 一+二+三+四+五
```

## 五、增值税

| 计税方式 | 税率 | 适用条件 |
|---------|:---:|---------|
| **一般计税** | **9%** | 一般纳税人 |
| **简易征收** | **3%** | 小规模纳税人 / 合同明确约定 |

增值税计税基数 = (分部分项 + 措施项目 + 其他项目 - 可抵扣进项税额)

## 七、实操要点

1. **管理费**：按 GB 50500-2024 附件1 区间取值，**最高投标限价按上限**
2. **利润**：按 2020版费用定额，合同可约定下浮
3. **规费**：2026-07-01 起不再单独列项，注意新旧合同切换
4. **人工调增**：政策性人工调增部分 **只计税金，不计管理费/利润/规费**
5. **每个费率**必须标注来源 + 核实日期
