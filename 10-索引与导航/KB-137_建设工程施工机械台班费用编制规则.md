# KB-137: 建设工程施工机械台班费用编制规则

> **来源**：住建部标准定额研究所 · 《建设工程施工机械台班费用编制规则》（2015）
> **原文**：`D:\高治国资料\知识库原始资料\09-参考规范\国标清单\`（GB50856 同级）
> **关联**：[[KC-286_新疆人材机价格信息查询路径]] §三 机械台班除税价表

## ⚡ 一句话结论

**施工机械台班单价 = 7 项费用构成，按八小时工作制计算。** 2015 版国标至今沿用，是编制招标控制价和投标报价时机械费计取的底层依据。

---

## 一、机械台班单价构成（7 项费用）

```
台班单价 = 折旧费 + 检修费 + 维护费 + 安拆费及场外运费 + 人工费 + 燃料动力费 + 其他费
```

### 1.1 折旧费

> 施工机械在规定的耐用总台班内，陆续收回其原值的费用。

```
折旧费 = 预算价格 × (1 - 残值率) / 耐用总台班
```

**预算价格**（国产机械）：
```
预算价格 = 施工机械原值 + 相关手续费和一次运杂费 + 车辆购置税
```

**预算价格**（进口机械）：
```
预算价格 = 到岸价格 + 关税 + 增值税 + 消费税 + 相关手续费 + 国内一次运杂费 + 银行财务费用 + 车辆购置税
```

**残值率**：按编制期国家有关规定取定

### 1.2 检修费

> 施工机械在规定的耐用总台班内，按规定的大修间隔进行必要的修理，以恢复其正常功能所需费用。

```
检修费 = 一次检修费 × 检修次数 / 耐用总台班
```

### 1.3 维护费

```
维护费 = 各级维护一次费用 × 各级维护次数 + 临时故障排除费 + 替换设备台班摊销费
```

**简化算法**：当各项数值难以取定时：
```
维护费 = 检修费 × K（K = 维护费系数）
```

### 1.4 安拆费及场外运费

**三种类型**：

| 类型 | 说明 | 处理方式 |
|------|------|---------|
| **不需计算** | ①不需安拆的机械 ②自行移动机械 ③固定在车间的机械 | 不计入台班单价 |
| **计入台班单价** | 安拆简单、移动需要起重及运输机械的轻型施工机械 | 计入台班单价 |
| **单独计算** | ①安拆复杂、移动需要起重及运输机械的重型施工机械 ②利用辅助设施移动的机械 | 单独计算 |

```
安拆费及场外运费 = 一次安拆费及场外运费 × 年平均安拆次数 / 年工作台班
```

- 一次场外运费包含：运输、装卸、辅助材料、回程等
- **运输距离按平均 30km 计算**
- 自升式塔式起重机、施工电梯：安拆费的超高起点及增加费由各地区、部门取定

### 1.5 人工费

> 机上司机（司炉）和其他操作人员的人工费。

```
人工费 = 人工消耗量 × (1 + 年制度工作日 / 年工作台班) × 人工单价
```

### 1.6 燃料动力费

```
燃料动力费 = ∑(燃料动力消耗量 × 燃料动力单价)
```

### 1.7 其他费

```
其他费 = (年车船税 + 年保险费 + 年检测费) / 年工作台班
```

## 三、机械停滞费与租赁费

### 3.1 停滞费

> 施工机械非自身原因停滞期间所发生的费用。

```
施工机械停滞费 = 折旧费 + 人工费 + 其他费
```

### 3.2 租赁费

> 由各地区、部门参照租赁市场价格并结合本地区、部门实际确定。

---

## 四、实操要点

1. **台班按 8 小时工作制计算**——不足 8 小时按比例折算
2. **国产 vs 进口**——规格相同的以国产为准取定原值
3. **进口机械预算价格** = 到岸价 + 关税 + 增值税 + 消费税 + 运杂费 + 银行财务费 + 购置税
4. **车辆购置税**：国产 =（机械原值 + 手续费 + 运杂费 - 增值税）× 税率
5. **安拆费运输距离** = 平均 30km
6. **材料除税价表**用于机械台班中的燃料动力费计算（见 [[KC-286]]）
