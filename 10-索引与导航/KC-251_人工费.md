# KC-251: 人工费

> [KC] | 来源：新疆2020版费用定额 + D:\定额库\31、新疆定额全套
> 核实日期：2026-06-25 | 版本 v2.0 新增人工费构成与计价规则

---

## ⚡ 一句话结论

**人工费 = ∑ (人工工日数量 × 人工工日单价)**，是建安费三大要素之一。政策性调整部分只计税金。

---

## 人工费构成（按 GB/T 50500-2024）

```
人工费
├── 计时工资 / 计件工资
├── 奖金（超额/质量/安全）
├── 津贴补贴（流动施工/高温/高空/高原）
├── 加班加点工资
└── 特殊情况工资（工伤/产假/婚丧假）
```

## 取费方式

| 专业 | 计费基数 | 说明 |
|------|---------|------|
| 建筑与装饰工程 | 人工费 + 机械费 | 管理费/利润均以此为基数 |
| 安装工程 | 人工费 | 安装专业的管理费/利润按纯人工计取 |
| 市政道路工程 | 人工费 + 机械费 | |
| 排水养护 | 人工费 | |

---

## 人工费调整原则

1. **政策性人工费调整**由住建厅发布，所有在建项目均应执行
2. **人工费调整幅度** ≠ 直接乘以文件百分比——要区分"定额人工"与"市场人工"
3. **人工调增部分**只计税金，不计管理费/利润/规费（新疆2020版费用定额规则）
4. **投标报价中的人工费**应参考信息价发布的人工单价，不低于法定最低工资
