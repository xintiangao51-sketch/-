## 🔴 新疆落地文件 3 个附件（重要）

### 附件 1：企业管理费费率区间表
> [[08-参考规范/GB-T50500-2024新疆落地_配套调整.pdf]]

**一般计税**（节选）：

| 专业 | 计费基数 | 区间费率% |
|------|----------|:---:|
| 建筑与装饰工程 | 人工费+机械费 | **10.77~15.08** |
| 安装工程 | 人工费 | **14.24~16.87** |
| 市政道路工程 | 人工费+机械费 | **25.67~32.09** |
| 市政桥涵工程 | 人工费+机械费 | **17.9~22.78** |
| 市政隧道工程 | 人工费+机械费 | **9.22~11.39** |
| 排水设施养护与维修 | 人工费 | **14.30~17.60** |
| 园林绿化（综合） | 人工费/机械费 | **10.54~24.11** |
| 抗震加固 | 人工费 | **13.81~19.04** |

**简易计税** 区间费率略低 0.5~0.8 个百分点。

📌 **关键规则**：
- 编制设计概算、**最高投标限价时应按上限计取**
- 投标人根据项目复杂难易程度**自主报价**，亦可参考区间费率自主确定

### 附件 2：费用计算程序
> [[08-参考规范/GB-T50500-2024新疆落地_房建安装补充.pdf]]

**最高投标限价费用计算程序**：
```
一、分部分项工程费 = ∑ 工程数量 × 综合单价
   其中：人工费 / 材料费 / 机械费 分别累计
二、措施项目费 = (一)单价措施 + (二)总价措施
   (一)单价措施：按"工程数量×综合单价"计算
   (二)总价措施：按实际发生项之和计算
       → 安全文明施工费 / 检验试验费 / 定位复测 / 已完工程及设备保护
       → 夜间施工 / 二次搬运 / 冬雨季施工 / 施工因素增加（市政）/ 竣工档案编制
       → 特殊地区施工增加 / 其他总价措施
三、其他项目费 = 暂列金额 + 暂估价 + 计日工 + 总承包服务费
四、增值税 = (一+二+三-18) × 税率
五、建设工程造价 = 一+二+三+四
```

1. **新项目**（2025-09-01 后启动）→ 默认按 GB/T 50500-2024 执行

4. **造价软件/数据交换** → 按 DB65/T 8013-2024 编码体系
