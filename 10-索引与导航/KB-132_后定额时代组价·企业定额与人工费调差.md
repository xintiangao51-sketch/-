# KB-132: 后定额时代组价·企业定额与人工费调差

> 来源：《走出造价困境——后定额时代如何组价套定额》（孙嘉诚，机械工业出版社 2022）
> 核心定位：后定额时代**施工企业如何组价套定额**——材料消耗量测算、人工费调差方案、企业定额编制
> 核实日期：2026-06-25

---

## ⚡ 一句话结论

**后定额时代的核心矛盾是"政府定额不能完全适配企业实际"，出路有两条：**
1. **人工费调差**——利用合同条款+民法典+调价文件做合同价差调整
2. **企业定额编制**——以全统消耗量定额为基础，编制自有企业定额自主报价

---

## 一、材料消耗量测算公式

### 1.1 净用量 + 损耗量

```
材料消耗量 = 净用量 × (1 + 损耗率)
损耗率   = 损耗量 / 净用量 × 100%
```

### 1.2 砖墙用量计算（标准砖）

```
净用量 = 2 × 砌体厚度系数 / [砌体厚 × (标准砖长 + 灰缝厚) × (标准砖厚 + 灰缝厚)]
```

| 墙厚 | 厚度系数 |
|------|:------:|
| 半砖墙 | 0.115 m |
| 一砖半墙 | 0.365 m |
| 两砖墙 | 0.49 m |
| 三砖墙 | 0.74 m |

**灰缝厚度**：8~12mm，取中间值 0.01m

**示例**（3 砖厚墙净用量）：
```
净用量 = 2 × 3 / [0.74 × (0.24 + 0.01) × (0.053 + 0.01)] = 515 块/m³
```

### 1.3 损耗率

| 材料 | 损耗率 |
|------|:----:|
| 实砖墙 | 2% |
| 砂浆 | 1% |

**示例**（3 砖厚墙实际用量）：
```
砖实际用量 = 515 × (1 + 0.02) = 525 块/m³
砂浆用量   = 0.247 × (1 + 0.01) = 0.249 m³/m³
```

### 1.4 周转性材料摊销量公式

```
摊销量 = 周转使用量 - 回收量 × 回收系数

其中：
  周转使用量 = 一次使用量 × [1 + (周转次数 - 1) × 损耗率] / 周转次数
  回收量     = 一次使用量 × (1 - 损耗率) / 周转次数
  回收系数   = 回收折价率(50%) / (1 + 间接费率)
```

**示例**（混凝土独立基础模板）：
```
一次使用量 = 2.1 m² × 0.083 m³/m² = 0.1743 m³
周转次数   = 6 次
损耗率     = 16.6%

周转使用量 = [0.1743 + 0.1743 × (6-1) × 16.6%] / 6 = 0.053 m³
回收量     = 0.1743 × (1 - 16.6%) / 6 = 0.024 m³
摊销量     = (0.053 - 0.024) × 50% / (1 + 1.82%) = 0.012 m³
```

## 二、人工费调差（4 象限法）

### 2.1 决策树

```
合同 + 调差约定
  │
  ├── 合同有效 + 约定可调 → 按合同约定调
  │      法律依据：《民法典》§509 第 1 款
  │
  ├── 合同无效 + 约定可调 → 参照合同约定调
  │      法律依据：《民法典》§793 第 1 款（验收合格可参照）
  │      + 建标〔2013〕44 号（人工费属工程价款）
  │
  ├── 合同有效 + 约定不可调 → 不调
  │      法律依据：《民法典》§509 第 1 款
  │
  └── 合同无效 + 约定不可调 → 参照约定不调
         法律依据：《民法典》§793 第 1 款
```

### 2.2 调差四象限速查

| 合同状态 | 约定可调 | 约定不可调 |
|---------|---------|----------|
| **合同有效** | ✅ 按合同调 | ❌ 不调 |
| **合同无效** | ✅ 参照调 | ❌ 参照不调 |

**关键提示**：调差文件出台后，**必须**先查合同约定再查效力，**不能想当然**。

### 2.3 调差三原则

1. **合同约定优先**——调差是合同行为，不是行政行为
2. **"无效"非"不存在"**——合同无效但验收合格，价款约定仍参照
3. **风险共担**——不可抗力按合同约定比例分担，无约定按公平原则

## 三、企业定额编制方案

### 3.1 编制思路

| 维度 | 传统政府定额 | 企业定额 |
|------|------------|---------|
| 消耗量 | 一般不调整 | 可灵活调整 |
| 单价 | 按实际调整 | 按实际调整 |
| 适用范围 | 整个行业 | 仅本企业 |
| 编制依据 | 全统消耗量定额 | 全统+企业实际 |
| 调整频率 | 5-10 年 | 动态调整 |

### 3.2 编制步骤

```
第 1 步：基本框架搭建
  └─ 参照全统/地区定额，搭建 17 章结构（17 类分项工程）

第 2 步：计算说明 + 计算规则
  └─ 分析全统定额说明 + 地区调整，建立自有说明规则

第 3 步：消耗量动态表格
  └─ 编制企业消耗量动态管理表（含量/单价实时调整）

第 4 步：综合单价库
  └─ 基于历史项目结算数据 + 市场询价，建立综合单价库

第 5 步：审核发布
  └─ 企业内部审核 + 试运行 + 正式发布
```

### 3.3 应用指南三原则

- **简易计算原则**：能简化的步骤简化
- **系数换算原则**：用系数表达弹性差异
- **补充原则**：政府定额未覆盖的，企业定额补充

### 3.4 适用边界

| 工程类型 | 适用 | 不适用 |
|---------|:---:|:-----:|
| 房屋建筑与装饰（含安装）| ✅ | — |
| 市政工程（围墙/规划线外）| ✅ | — |
| 房屋修缮 | ✅ | — |
| 公路工程 | — | ❌（交通部独立体系）|
| 水利工程 | — | ❌（水利部独立体系）|

### 3.5 房屋建筑与装饰 17 章分类（企业定额参照）

| 章 | 类别 |
|:--:|------|
| 1 | 土石方工程 |
| 2 | 地基处理及边坡支护工程 |
| 3 | 桩基工程 |
| 4 | 砌筑工程 |
| 5 | 混凝土及钢筋混凝土工程 |
| 6 | 金属结构工程 |
| 7 | 木结构工程 |
| 8 | 门窗工程 |
| 9 | 屋面及防水工程 |
| 10 | 保温、隔热、防腐工程 |
| 11 | 楼地面装饰工程 |
| 12 | 墙、柱面装饰与隔断、幕墙工程 |
| 13 | 天棚工程 |
| 14 | 油漆、涂料、裱糊工程 |
| 15 | 其他装饰工程 |
| 16 | 拆除工程 |
| 17 | 措施项目 |

### 3.6 风险控制

- 企业定额应**定期修订**（建议每年一次）
- 投标报价使用企业定额时，**编制说明中应明确**"采用企业定额"
- 与政府定额**校验比对**——避免偏差过大被质疑
- 重大工程建议**双轨制**：政府定额作底，企业定额作报价

## 四、土建与市政取费边界

> 经常关注的是土建/市政取费界限区分

| 情况 | 取费系数 |
|------|---------|
| 有围墙 | 围墙外=市政，围墙内=房建（含安装）|
| 无围墙（按规划线）| 规划线外=市政，内=房建 |
| 管道 | 接入市政管网第一个井外=市政，井内=房建 |
| 场区内小市政 | **执行土建系数** |

**场区小市政特别注意**：通常归土建范围，不可误套市政费率。

## 五、机械费构成

### 5.1 价值 2000 元规则

| 价值 | 归类 |
|------|------|
| **< 2000 元** | 工具用具使用费（**属企业管理费**，不计机械费）|
| **≥ 2000 元** | 施工机械使用费（**属机械费**）|

> 大部分地区定额中机械费为**除税单价**。

### 5.2 自有机械台班单价构成（7 项）

```
台班折旧费
+ 台班大修费
+ 台班经常修理费
+ 台班安拆费及场外运费
+ 台班人工费
+ 台班燃料动力费
+ 台班养路费及车船使用税
```

### 5.3 租赁机械的 3 大价格因素

1. **双方协定的租金**（核心）
2. **约定的机械进出场费用补贴**（安拆+场外运输）
3. **约定的机械操作及应计入总承包方的日常维修费用**

### 5.4 机械费价格计入注意事项

- ✅ **以工程机械租赁/供应合同**作为价格基础 + 结合企业利润报价
- ✅ **塔式起重机租赁价通常含司机工资+维修费**——人工费/维修费不单列
- ✅ **小型机械（振动棒/对焊机/圆盘锯）**通常由分包方自带，**已含在劳务分包合同价中**——不再单列

## 六、组价中常见陷阱

| 陷阱 | 后果 | 规避 |
|------|------|------|
| 用全统定额套地州项目 | 错套费率 | 用本地估价表 |
| 措施项目不列项 | 漏计 | 模板/脚手架/安全文明单列 |
| 周转材料漏算 | 漏成本 | 按公式测算摊销量 |
| 人工费不调差 | 实际成本超支 | 签合同时约定调差条款 |
| 风险范围不明确 | 后期争议 | 合同明确风险范围与计算方法 |
| 全费用综合单价混用 | 漏规费/税金 | 区分 GB 50500-2013 vs 2024 |

---

## 七、实操要点

1. **材料消耗量**——按净用量+损耗率+摊销量公式计算
2. **人工费调差**——先查合同约定，再判合同效力
3. **企业定额**——参照全统+企业实际，定期修订
4. **取费边界**——特别注意场区小市政归土建
5. **风险费用**——综合单价已含，不要重复计
