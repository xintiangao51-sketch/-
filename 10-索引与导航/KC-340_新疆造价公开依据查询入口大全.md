# KC-340: 新疆造价公开依据查询入口大全

> [KC] | 来源：各官方渠道 | 核实日期：2026-06-23

## 关键词
查询入口, 住建厅, 造价信息网, 工程建设云, 标准信息平台

## ⚡ 一句话结论
六大官方入口覆盖新疆造价工作全部公开查询需求——**按优先级矩阵逐级检索**。

## 六大入口矩阵

| 用途 | 入口 | 链接 |
|------|------|------|
| **政策/定额一手发布** | 自治区住建厅公示公告 | [zjt.xinjiang.gov.cn](https://zjt.xinjiang.gov.cn/xjzjt/c113280/jumpkey.shtml) |
| **价格/信息价/定额** | 新疆建设工程造价信息网 | [xjzj.com](https://www.xjzj.com/) |
| **地州月度信息价** | 造价信息网 · 价格检索 | [xjzj.com 检索](https://www.xjzj.com/Home/CheckView?content=) |
| **企业信用/法规** | 新疆工程建设云 | [jsy.xjjs.gov.cn](https://jsy.xjjs.gov.cn/articleList?&itemType=cms_xyzcfg) |
| **招标/行业通知** | 新疆建设工程综合信息网 | [ztbxj.com](https://www.ztbxj.com/index.php?s=/index/article/index/id/22) |
| **标准元数据** | 全国标准信息公��服务平台 | [std.samr.gov.cn](https://std.samr.gov.cn/db/search/stdDBDetailed?id=1A09BC403A9F2957E06397BE0A0AAA53) |
| **公路材料价格** | 自治区交通运输厅 | [jtyst.xinjiang.gov.cn](https://jtyst.xinjiang.gov.cn/xjjtysj/zjxx/nav2_list.shtml) |

## 使用优先级
```
🔴 第一优先 → zjt.xinjiang.gov.cn  (政策/标准/定额原文)
               xjzj.com              (价格/信息价)

🟡 第二优先 → 项目所在地州住建局网站 (本地估价表/人工单价)
               jsy.xjjs.gov.cn       (企业资质/信用/法规)

🟢 第三优先 → ztbxj.com             (招标公告/行业通知)
               std.samr.gov.cn        (标准全文验证)
```

## ⚠ 防死链技巧
自治区住建厅网站经常栏目重组 → 链接失效时从首页搜索关键词。
