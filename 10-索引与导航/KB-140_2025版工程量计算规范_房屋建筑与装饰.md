## ⚡ 一句话结论

**2025 版计算规范在 2013 版基础上更新了工程量计算规则**，同时调整了部分项目的计量单位和项目特征。与 GB/T 50500-2024（2026-07-01 施行）配套使用。

## 二、工程计量核心规则

### 2.1 编制依据

1. 经审定通过的施工设计图纸及其说明
2. 经审定通过的施工组织设计或施工方案
3. 经审定通过的其他有关技术经济文件

### 2.2 计量精度

| 单位 | 保留位数 | 示例 |
|:----:|:-------:|------|
| t | 3 位小数 | 12.345 t |
| m / m² / m³ / kg | 2 位小数 | 123.45 m³ |
| 个/件/根/组/系统 | **整数** | 15 个 |

### 2.3 补充项目

```
编码规则：01 + B + 三位数字（如 018001）
同一招标工程不得重码
补充项目需附：名称、项目特征、计量单位、计算规则、工作内容
```

## 三、清单编制核心规则

### 3.1 项目编码（十二位）

```
01 01 01 001 001
│  │  │  │  └─ 十至十二位：清单项目名称顺序码
│  │  │  └──── 九至九位：分项工程项目名称顺序码
│  │  └─────── 七至八位：分部工程顺序码
│  └────────── 五至六位：附录分类顺序码
└───────────── 一至四位：专业工程代码（01=房屋建筑与装饰）
```

### 3.2 项目特征描述

按附录中规定的项目特征，结合拟建工程实际予以描述。详见 [[KB-138_工程量清单项目特征描述指南]]

### 3.3 特殊规则

- **现浇混凝土**：模板工程可含在综合单价中，也可在措施项目中单列——招标人根据实际情况选用
- **预制混凝土构件**：按现场制作编制，模板已含在工作内容中；若采用成品构件，成品价计入综合单价
- **金属结构构件**：按成品编制，成品价计入综合单价
- **门窗**：按成品编制，含安装（橱窗除外）

## 五、与 GB 50500-2013 的主要变化

| 变化维度 | 2013 版 | 2025 版 |
|---------|--------|--------|
| **模板工程** | 一般单列在措施项目 | 沿袭模板可在综合单价或措施项目中，但更明确需招标人选用 |
| **预制构件** | — | 明确说明含模板不另列 |
| **金属结构** | — | 明确按成品编制 |
| **门窗** | 按成品编制 | 沿袭成品编制（橱窗除外）|
| **编码体系** | 十二位 | 沿袭十二位编码体系 |
| **计量精度** | 保留三位/两位/整数 | 沿袭 |
| **补充项目** | 01B001 起 | 沿袭 018001 起 |

> 与 GB/T 50500-2024（清单计价规范）配套使用时，注意**规费不再单独列项**等重大变化。
