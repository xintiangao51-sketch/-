# KB-133: 土建审计·黑与白

> 来源：《走出造价困境——土建审计的黑与白》（机械工业出版社）
> 核心定位：**站在审计方/被审计方双角度**——审计如何找问题，造价师如何避坑
> 核实日期：2026-06-25 | 评分 v1.1 基于 PDF 重读

---

## ⚡ 一句话结论

**审计是"算量"+"组价"+"合同"三方面的合规性检查**。常见争议点：①工程量计算规则的口径差；②定额套用错位；③特征描述漏项；④变更签证程序瑕疵。**造价师懂审计才能避免结算时被砍。**

## 一、土方工程审计要点（23 个点）

### 1.1 招投标阶段（8 个点）

| 审计点 | 关键审查项 |
|--------|----------|
| **土壤类别** | 需有地勘报告，特征描述中应写"详见地勘报告" |
| **土方体积** | 按天然密实体积（自然方）计算，不是虚方 |
| **土方折算系数** | 虚方/实方有折算系数（松散系数 1.3/0.87 等）|
| **大开挖 vs 基坑土方** | 大开挖执行"挖一般土方"，集水坑/下柱墩执行"挖基坑土方" |
| **场地踏勘** | 必须踏勘，看是否具备存土条件（影响外运/内倒）|
| **土方放坡/工作面** | 综合单价应含放坡，否则按"垫层底面积×挖土深度"算 |
| **基坑清槽** | 基础底部预留 20-30cm 人工清槽，不要漏项 |
| **外购土方** | 应办理签证，明确"换填范围/材料种类/换填厚度" |

### 1.2 实施阶段（7 个点）

| 审计点 | 关键 |
|--------|------|
| **原始地面标高** | 必须有甲方提供的原始标高图/方格网图 |
| **挖淤泥/流沙** | 需业主/监理见证，办理签证（一般土与淤泥价差大）|
| **集水坑/下柱墩** | 明确做法核定单，单独列项（人工/机械挖土单价差大）|
| **回填松填/夯填** | 肥槽=夯填，地库顶板/房心/绿化=松填（**松夯价差近 3 倍**）|
| **土方坡道** | 专项分包不再计；总包范围内的按合同处理 |
| **渣土消纳** | 部分地区需办理消纳证，结算时可能扣减 |
| **交叉作业补挖** | 总包收场地时需测量坑底高程，办理见证签字 |

### 1.3 结算阶段（8 个点）

| 审计点 | 关键 |
|--------|------|
| **资料准备** | ①地勘报告 ②方案 ③原始/开挖后高程 ④现场施工方案 |
| **土方软件** | 南方 CASS9.1 配 CAD2010/2016 为常用组合 |
| **专业分包补挖** | 收场时三方面测量+见证，避免争议 |
| **软弱地基** | 砂石换填签证要含三要素（范围/材料/厚度）|
| **塌方** | 明确责任（设计/施工/不可抗力）|
| **外购土运距** | 需签认运输距离，避免结算争议 |
| **渣土消纳证** | 结算时可能要求提供，否则扣减 |
| **外运土** | 实际是虚方，但要按实方计算回填体积 |

## 二、平整场地审计要点（3 个点）

### 2.1 计算规则

| 标准 | 说明 |
|------|------|
| 18 清单 | 按设计图示尺寸以建筑物首层建筑面积计算 |
| 部分地区 | 按外墙外边线每边各加 2m 计算 |
| 北京 2012 | 地下室单层面积 > 首层面积时，按地下室最大单层面积 |

### 2.2 三大审计点

| 审计点 | 说明 |
|--------|------|
| **单独地下车库** | 有顶板无首层，按车库面积计算平整场地 |
| **机械大开挖** | 视地区定额而定（争议大），河北定额站：按定额站文件解释 |
| **三通一平 vs 平整场地** | 三通一平=整个场区（含楼座/生活/加工区），平整场地=建筑物底面积 |

## 三、其他专业审计要点

### 3.1 砌体工程（8 个点）

- 砌体厚度计算（半砖/一砖/一砖半/两砖/三砖墙）
- 灰缝厚度（标准 10mm）
- 砌体加固筋（构造柱/圈梁/过梁）
- 门窗洞口扣除（>0.3m²）
- 砌体加筋（拉结筋/抗震筋）

### 3.2 混凝土及钢筋混凝土（20 个点）

- 现浇 vs 预制分别列项
- 模板工程量按接触面积
- 钢筋按理论重量 + 损耗
- 预埋件/预留洞
- 后浇带单独列项
- 泵送费（信息价已含泵送，现场无泵送可扣减）

### 3.3 屋面及防水（11 个点）

- 屋面防水卷材/涂膜分别列项
- 防水上反高度（地下室顶板/屋面女儿墙）
- 屋面设备基础（>0.3m² 扣除）
- 屋面排水（落水管/水簸箕）
- 排汽孔做法
- 女儿墙防水收头

### 3.4 措施项目（32 个点）

- 模板（按接触面积/含量）
- 脚手架（外墙/里脚手/满堂红）
- 垂直运输（按建筑面积）
- 安全文明施工费（不可竞争）
- 临时设施
- 二次搬运
- 冬雨季施工增加费
- 夜间施工增加费

### 3.5 其他专业

- 木结构（2 个点）
- 金属结构（6 个点）
- 门窗工程（9 个点）
- 保温/隔热/防腐（8 个点）
- 楼地面装饰（8 个点）
- 墙柱面装饰（10 个点）
- 天棚工程（2 个点）

## 四、审计的"黑与白"

### 4.1 灰色地带（常见争议点）

- **大开挖 vs 基坑土方**——口径不统一，看定额站解释
- **机械大开挖是否计平整场地**——地区差异
- **松填 vs 夯填**——**3 倍价差**，需图纸明确
- **基坑清槽**——定额是否含独立子目
- **措施项目**——按项/按量/按% 各地区不同
- **材料调差**——风险范围 5% 阈值外的计算方法

### 4.2 审计师常用"砍价"招数

1. **工程量压缩**——按图计量+系数核减
2. **定额套用降级**——高套定额改低套
3. **特征描述漏项**——找清单中未描述清楚的子目
4. **材料价**——按信息价而非市场价
5. **规费/税金**——按最低标准
6. **变更签证**——程序瑕疵扣减
7. **结算资料**——不齐全扣减

### 4.3 造价师如何反制

1. **特征描述**——清单阶段就写清楚，"精/全/清"
2. **暂列金额**——预留风险（≤ 20%）
3. **合同条款**——明确调差/风险范围/计量方式
4. **签证**——及时办理，4 要素齐全（量/价/时/因）
5. **隐蔽验收**——所有隐蔽必须有影像+签证
6. **资料归档**——按 14 类资料（清单/合同/变更/签证/会议纪要/竣工图/验收等）保存
7. **争议处理**——28 天索赔通知+84 天最终索赔+造价鉴定

## 五、审计流程图

```
图纸会审
  ↓
清单/招标控制价审核
  ↓
投标报价核对（不平衡报价审查）
  ↓
合同签订审核
  ↓
施工过程监督（变更/签证/计量）
  ↓
竣工资料审核（结算资料完整性）
  ↓
结算审核（量/价/项/规/税 五审）
  ↓
争议处理（鉴定/仲裁/诉讼）
```

## 六、关键提醒

### 6.1 造价师防砍价 5 大要点

1. **清单特征**——写细、写全、写清
2. **暂列金额**——<20% 但要留足
3. **签证及时**——24 小时现场签证，7 天价款报告
4. **隐蔽验收**——必须影像+签证
5. **资料齐全**——结算前 14 类资料备齐

### 6.2 审计师找问题 5 大方向

1. **工程量**——按图+按计算规则核对
2. **定额套用**——是否高套/错套
3. **材料价**——信息价 vs 市场价
4. **费率**——按最新文件
5. **变更签证**——程序合规性
