# 📊 项目台账仪表盘

> 自动聚合 09-项目资料 下所有项目 | 更新：刷新页面即可

## 在建项目

`dataview
TABLE 
  project_name as "项目名称", 
  contract_amount as "合同金额(万)", 
  region as "地区", 
  status as "状态",
  date as "日期"
FROM "09-项目资料"
WHERE status = "在建" OR status = "#在建"
SORT date DESC
`

## 全部项目

`dataview
TABLE 
  project_name as "项目名称",
  contract_amount as "合同金额(万)", 
  region as "地区",
  status as "状态"
FROM "09-项目资料"
WHERE file.name != this.file.name
SORT file.ctime DESC
`

## 按地区分布

`dataview
TABLE length(rows) as "项目数"
FROM "09-项目资料"
WHERE region
GROUP BY region
SORT length(rows) DESC
`

## 最近更新的项目

`dataview
TABLE 
  file.mtime as "最后修改",
  status as "状态"
FROM "09-项目资料"
WHERE file.name != this.file.name
SORT file.mtime DESC
LIMIT 10
`
