# KC-481: 新疆造价公开依据清单维护节奏

> [KC 维护卡] | 维护人：Codex（鼎梁筑） | 生效日期：2026-06-23

## 关键词
维护, 更新, 月度, 季度, CHANGELOG, 定额发布

## ⚡ 一句话结论
**任何新版定额发布 → 必须同步修订 KC-480 全系列 + AGENTS.md §五 + CHANGELOG.md。**

## 维护节奏

| 频率 | 任务 | 涉及卡片 |
|------|------|----------|
| **每周一** | 检查住建厅公示公告 + 造价信息网首页通知 | KC-104/106/107 |
| **每月第一周** | 更新价格信息索引 + 验证链接有效性 | KC-286/105 |
| **每季度初** | 采集乌鲁木齐季度人工单价 + 本地信息价 | KC-250/286 |
| **即发即更** | 新版定额发布 → 全系列联动修订 | KC-480 全系列 |

## 联动更新顺序
```
1. KC-104 或 KC-106 (定额本体)
2. KC-480 (总索引)
3. KC-227 (费率联动)
4. KC-341 (五步核实法-新增路径)
5. AGENTS.md §五
6. CHANGELOG.md
```
