# 🏷️ 标签体系

> 创建于 2026-06-21 | P3 知识沉淀核心基建
> 本页定义全库标签分类体系、命名规范与使用规则

---

## 📂 分类体系

按 **领域 / 属性 / 状态 / 项目** 四维标签空间组织：

### 领域标签（`领域/` 命名空间）

| 标签 | 含义 | 覆盖范围 |
|:-----|:-----|:---------|
| `造价` | 造价计价核心 | 定额、清单、组价、取费、工程量 |
| `施工` | 施工技术与管理 | 施工方案、施组、安全、进度 |
| `合同` | 合同与法务 | 合同模板、FIDIC、法规、索赔 |
| `投标` | 投标报价 | 投标文件、报价策略、招标分析 |
| `图纸` | 工程图纸与BIM | 图纸、图集、DWG、PDF图纸 |
| `材料` | 材料与设备 | 材料价格、设备选型、信息价 |

### 领域子标签（`领域/子域/` 命名空间）

| 标签 | 含义 |
|:-----|:-----|
| `造价/定额` | 定额套用、定额子目、消耗量 |
| `造价/工程量` | 工程量计算规则、BOQ |
| `造价/取费` | 管理费/利润/规费/税金 |
| `造价/清单` | 工程量清单编制 |
| `造价/组价` | 综合单价组价 |
| `造价/变更` | 变更签证、索赔 |
| `施工/安全` | 安全管理、红线管理 |
| `施工/施组` | 施工组织设计 |
| `施工/方案` | 专项施工方案 |
| `合同/法务` | 法律法规、司法解释 |
| `合同/模板` | 合同范本与模板 |
| `投标/策略` | 投标策略与报价技巧 |
| `投标/分析` | 招标文件分析、评标分析 |

### 属性标签（`属性/` 命名空间）

| 标签 | 含义 | 使用场景 |
|:-----|:-----|:---------|
| `精华` | 高质量/高价值笔记 | ⭐ 核心知识点、经验总结 |
| `索引` | 目录/导航页 | MOC页、总览页、README |
| `模板` | 可复用模板 | 笔记模板、台账模板、清单模板 |
| `规范` | 标准/规范/法规 | 国标、行标、地标、法规原文 |
| `待核实` | 数据/结论需核实 | 费率、参数、人工价待确认 |

### 状态标签（`状态/` 命名空间）

| 标签 | 含义 |
|:-----|:-----|
| `在建` | 项目正在施工中 |
| `竣工` | 项目已竣工/验收 |
| `投标中` | 正在投标阶段 |
| `停工` | 项目暂停/搁置 |
| `结算` | 结算/审计阶段 |

### 项目标签（`项目/` 命名空间）

| 标签 | 全称 | 状态 |
|:-----|:-----|:----:|
| `项目/哈密国源` | 哈密国源综合服务中心 | 🔴 在建 |
| `项目/特克斯` | 特克斯阳光谷 | 🟡 投标中 |
| `项目/鄯善启创` | 鄯善启创 | 🟡 在建 |
| `项目/科特迪瓦` | 科特迪瓦海外项目 | 🟡 资料处理 |
| `项目/阳安二线` | 阳安二线 | 🟡 变更索赔 |
| `项目/阿勒泰` | 阿勒泰电缆沟 | 🟢 验收中 |
| `项目/道班房` | 道班房 | 🟡 摸底中 |

## 📐 标签使用规则

1. **每笔记最多 5 个标签**，优先使用层级标签（`领域/子域`）
2. **标签顺序**：领域 → 属性 → 状态 → 项目
3. **项目标签**仅用于 `09-项目资料/` 下的笔记
4. **精华标签**由人工标记，代表"值得反复阅读"
5. **待核实标签**应在核实后移除，保持时效性
6. **标签小写中文**，不含特殊字符（`#` 仅在 YAML 中作为标签语法前缀）

### 标签示例

```yaml
# 一个造价笔记的正确标签
tags: [造价/定额, 造价/组价, 精华, 项目/哈密国源]

# 一个项目总览的正确标签
tags: [索引, 项目/特克斯, 在建]
```

## 📊 标签覆盖查询（Dataview）

### 标签使用分布

```dataview
TABLE length(rows) as "笔记数"
FROM ""
WHERE file.tags
FLATTEN file.tags as tag
WHERE !startswith(tag, "项目/")
GROUP BY tag
SORT length(rows) DESC
LIMIT 30
```

### 项目标签覆盖

```dataview
TABLE length(rows) as "笔记数"
FROM "09-项目资料"
WHERE file.tags
FLATTEN file.tags as tag
WHERE startswith(tag, "项目/")
GROUP BY tag
SORT length(rows) DESC
```

### 待核实笔记清单

```dataview
TABLE file.folder as "目录", file.mtime as "最后修改"
FROM ""
WHERE contains(tags, "待核实")
SORT file.mtime DESC
```

### 精华笔记清单

```dataview
TABLE file.folder as "目录", file.mtime as "最后修改"
FROM ""
WHERE contains(tags, "精华")
SORT file.mtime DESC
LIMIT 50
```
