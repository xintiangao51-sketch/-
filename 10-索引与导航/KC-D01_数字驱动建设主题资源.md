# KC-D01: 数字驱动建设 · 主题资源整合

> 主题：数字驱动建设（数字住建 + 智能建造 + BIM + 装配式）
> 整合人：Codex（鼎梁筑） | 整理日期：2026-06-24
> 用途：项目策划/方案比选/投标/合同审查时，**数字驱动建设**相关政策的查询起点

## ⚡ 一句话结论
新疆已形成"**国标底座 + 自治区 BIM 全生命周期 + 智能建造定额 + 装配式补充定额 + 数字住建规划**"五位一体的数字驱动建设政策体系。**全国首批智能建造定额**已在新疆落地（2026-05-07 发布）。**2026-07-01 起**，新疆 ≥1 亿元政府投资公共建筑必须应用 BIM 全生命周期。

## 二、🔴 核心政策清单（已抓全文）

### 2.1 住建部"数字住建"建设整体布局规划
- **发布日期**：2024 年 2 月 7 日
- **来源**：[xjzj.com ID=6429](https://www.xjzj.com/Home/PoliciesDetail/6429)
- **全文**：[[06-参考文档/DDC数字驱动建设/cards/_index_DDC数字驱动建设|正文 9761 字]]
- **整体框架**："2+2+N+3"
  - 第一个 2：数字基础设施 + 数据资源体系
  - 第二个 2：信息安全保障体系 + 政策标准保障体系
  - N：数字住房/数字工程/数字城市/数字村镇
  - 3：大系统共治/大数据慧治/大服务惠民
- **关键目标**：
  - 2027 年底：部省市三级平台高效联通，"一网统管"
  - 2035 年底：数字基础设施全面夯实，"数字住建"体系高效运行
- **关键核心技术**：城市级海量数据处理、多源传感信息融合感知、BIM 三维图形引擎、**建筑机器人应用**、CIM 平台

### 2.2 自治区 BIM 技术全生命周期应用通知 ✅ 已抓全文
- **发布日期**：2026 年 5 月 8 日（**2026-07-01 施行**）
- **发布机构**：自治区住建厅 + 自治区数字化发展局
- **来源**：[zjt.xinjiang.gov.cn](https://zjt.xinjiang.gov.cn/xjzjt/c113382/202605/1c73f0afa8ad4725984ece400f6465cb.shtml)
- **全文**：[[06-参考文档/DDC数字驱动建设/cards/_index_DDC数字驱动建设|全文 Markdown]]

**强制应用范围**（2026-07-01 起）：

| 触发条件 | 阈值 |
|----------|------|
| 项目类型 | 自治区政府投资项目 |
| 投资额 | **≥ 1 亿元** |
| 建筑面积 | **≥ 2 万 m²** |
| 工程类型 | 大型公共建筑 / 工程总承包 / 装配式建筑 |

**各方主体责任**：

| 主体 | 责任要点 |
|------|----------|
| **建设单位** | 牵头落实全生命周期应用；保障经费投入 |
| **设计单位** | 全专业模型与设计文件**同步交付**；**BIM 正向设计建模** |
| **造价机构** | 基于 **BIM 模型精准算量**；全过程成本管控 |
| **施工图审查机构** | 实施 **BIM 智能审图** |
| **招标代理机构** | 招标文件明确 **BIM 实施标准与评标要求** |
| **施工单位** | 优化施工方案；同步更新竣工模型；**数字化交付** |
| **运维单位** | 数据管护；**设施智能监控**、运维高效协同 |

## 七、实操要点

1. **方案比选**：智能建造定额子目存在 → 应优先考虑建筑机器人方案
2. **投标策略**：新疆项目 BIM 应用是**强制项**（≥1 亿元政府投资项目）
3. **合同审查**：装配式构件定价应引用 [[10-索引与导航/知识库总索引#KC-106]] 的 2022-12 试行定额

5. **2026-07-01 强施行**：新疆所有 ≥1 亿元政府投资项目必须在设计/造价/施工/运维全生命周期应用 BIM
