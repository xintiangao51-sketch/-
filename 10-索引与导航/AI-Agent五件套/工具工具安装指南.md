# 工具安装指南 · Supermemory + Dory + MCP

## 一、Supermemory（主推记忆引擎）

### 简介
Supermemory 是开源记忆 API（22K+ GitHub Stars），LongMemEval 81.6% 准确率。
作为鼎梁筑记忆体系的外部持久化引擎。

### 安装方式（任选一）

```bash
# 方式一：一键安装（推荐）
curl -fsSL https://supermemory.ai/install | bash

# 方式二：npx 本地运行
npx supermemory local
# 启动后在 http://localhost:6767 访问

# 方式三：Python 包
pip install supermemory-agent-mcp

# 方式四：Claude Desktop MCP（推荐桌面端用户）
npx -y install-mcp@latest https://mcp.supermemory.ai/mcp --client claude --oauth=yes
```

### 后台运行（Windows）
```powershell
# 创建计划任务每天启动
schtasks /create /tn "Supermemory" /tr "cmd /c npx supermemory local" /sc onlogon /ru %USERNAME%
```

### 验证安装
```bash
curl http://localhost:6767/health
# 预期返回：{"status":"ok"}
```

## 二、Dory（本地离线记忆备选）

### 简介
Dory 是本地优先的图谱记忆方案（84% LongMemEval），单 SQLite 文件运行。

### 安装
```bash
pip install dory-memory
```

### 快速使用
```python
from dory import DoryMemory

mem = DoryMemory()
mem.observe("昕田是新疆工程造价工程师，从业21年")
mem.observe("昕田擅长2020版新疆定额套用")

result = mem.query("昕田的专业背景是什么？")
print(result)
# → 新疆工程造价工程师，从业21年
```

### 优劣势对比

| 对比项 | Supermemory | Dory |
|--------|-------------|------|
| 部署方式 | 本地服务/云端 | 单SQLite文件 |
| 图谱类型 | RAG+Profile | 4类型（情节/语义/程序/工作） |
| 联网要求 | 可选 | 完全离线 |
| 安装大小 | ~200MB | ~5MB |
| 适用场景 | 多Agent共享记忆 | 单Agent本地记忆 |

## 三、MCP 服务器配置

### Claude Desktop MCP 配置
在 `claude_desktop_config.json` 中添加：

```json
{
  "mcpServers": {
    "supermemory": {
      "command": "npx",
      "args": [
        "-y",
        "@supermemory/mcp-server"
      ]
    }
  }
}
```

### 文件路径
Windows: `%APPDATA%\Claude\claude_desktop_config.json`
macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
