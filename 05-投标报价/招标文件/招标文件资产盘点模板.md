# 招标文件资产盘点模板

> 拿到招标文件后第一轮盘点，快速建立文件坐标和优先级。

## 盘点步骤

1. **目录递归扫描**：记录所有文件名、后缀、大小、修改时间
2. **格式分类**：按后缀分组（.pdf/.docx/.xlsx/.dwg/.zip/专用格式）
3. **可解析性判断**：
   - PDF/Word/Excel：可直接文本抽取
   - DWG：需 CAD 工具或图纸目录读取
   - .SHZF/.CEZB 等：国能e招专用格式，需专用工具打开
   - .zip：解压后盘点
4. **内容映射**：
   - 招标正文 → 项目概况/投标时间/资格要求/评标办法/合同条款
   - 工程量清单 → 专业构成/清单格式/工程量量级
   - 图纸 → 建筑/结构/暖通/电气/给排水/总图
   - 制度协议 → 安全管理/农民工工资/外委承包商
5. **优先级排序**：
   - 🔴 立即读取：投标截止时间、资格要求、报价口径、最高限价
   - 🟡 重要但不紧急：清单工程量校核、图纸与清单一致性
   - 🔵 后续补充：制度协议全文、图纸细节

## 输出物

- 文件资产台账（.md 或 .xlsx）
- 关键信息速查表（项目概况/时间/资质/报价要求）
- 清单专业构成概览
- 待深入问题清单
