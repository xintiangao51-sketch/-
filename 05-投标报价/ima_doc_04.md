# 施工组织设计与投标标准模板

> 老高造价工作台 | 适用：新疆地区建设工程施工投标

---

## 一、施工组织设计标准框架

### 1.1 编制依据
| 类别 | 文件 | 备注 |
|------|------|------|
| 招标文件 | 招标图纸、工程量清单、合同条款 | 以招标文件为准 |
| 规范标准 | 国标+新疆地标+行业标准 | 列出具体编号 |
| 企业标准 | 企业施工工艺标准、安全手册 | — |
| 现场踏勘 | 现场地形、交通、水电、地材 | 附照片 |

### 1.2 工程概况
- 项目位置、规模、结构形式
- 主要工程量汇总
- 地质水文条件（⚠新疆重点关注：地震设防烈度、冻土深度、地下水位）

### 1.3 施工总体部署
- 施工区段划分
- 施工顺序与流向
- 总体进度计划（横道图/网络图）
- ⚠ 新疆冬歇期：北疆11月-3月，南疆12月-2月

### 1.4 关键施工方案（每个方案需2-3项比选）

| 工艺 | 方案A | 方案B | 方案C | 推荐 |
|------|-------|-------|-------|------|
| 基坑支护 | 钢板桩 | 放坡+喷锚 | 灌注桩排桩 | 按深度选 |
| 模板体系 | 木模 | 钢模 | 铝模 | 按周转次数 |
| 混凝土浇筑 | 泵送 | 塔吊+料斗 | 自卸 | 按浇筑量 |
| 钢筋连接 | 绑扎 | 焊接 | 套筒 | 按直径选 |

## 二、投标报价策略

### 2.1 不平衡报价策略
| 策略 | 操作 | 风险 |
|------|------|------|
| 早收款 | 前期项目高报 | 🔵低风险 |
| 工程量增减 | 预计增量项目高报 | 🟡需核实变更趋势 |
| 图纸不明 | 模糊项低报 | 🔴可能亏损 |

### 2.2 新疆地区投标注意事项
| 要点 | 说明 |
|------|------|
| 地区类别 | 喀什/和田为五类地区，人工机械系数上浮 |
| 运距 | 材料运输距离远，运费占比高 |
| 气候 | 冬季施工措施费不可漏项 |
| 地材 | 砂石料就近采购，注意料场储量 |
| 人工 | 少数民族地区用工需考虑技能培训 |

## 三、进度计划与资源调配

### 3.1 工期计算

```
总工期 = 施工准备（7-15天）+ 主体施工 + 装饰装修 + 竣工验收（15-30天）
        - 冬歇期（北疆约120天，南疆约60天，扣除或搭接）
```

### 3.2 劳动力计划（参考）

| 工种 | 典型配置（人/万㎡） | 备注 |
|------|-------------------|------|
| 钢筋工 | 15-25 | 按钢筋量调整 |
| 木工 | 20-30 | 按模板量调整 |
| 混凝土工 | 8-12 | — |
| 架子工 | 5-8 | — |
| 安装工 | 10-15 | 水电暖通 |

## 四、安全与质量保证

| 要素 | 新疆特殊要求 |
|------|-------------|
| 地震 | 抗震设防烈度7-8度，施工措施 |
| 风沙 | 南疆春季防风沙措施 |
| 高温 | 吐鲁番/南疆夏季高温施工措施 |
| 严寒 | 北疆冬季施工保温措施 |
| 维稳 | 按当地维稳要求配置 |

---

## 五、投标常见废标项（🔴红线）

1. 投标函未按要求签章
2. 投标保证金未到账/金额不足
3. 工期超过招标文件要求
4. 主要项目清单报价低于成本
5. 项目经理资格不满足要求
6. 业绩不符合招标条件
7. 联合体协议缺失或无效
