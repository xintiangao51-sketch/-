# M3 已结项目扫描·PowerShell 指南

## 目的

扫描 `D:\造价工作台`（70K 文件），找出**已结项目**（不在建的），统计"可提炼"清单。

## 在 PowerShell 中执行

```powershell
# 1. 列出 D:\造价工作下所有项目目录
$root = "D:\造价工作台"
Get-ChildItem $root -Directory | Select-Object Name, LastWriteTime, @{N="FileCount";E={(Get-ChildItem $_.FullName -Recurse -File -ErrorAction SilentlyContinue).Count}}

# 2. 找 "已结" 关键词的目录
$keywords = @("已结","竣工","结算完成","完工","已结算")
foreach ($k in $keywords) {
  Get-ChildItem $root -Recurse -Directory -ErrorAction SilentlyContinue | 
    Where-Object { $_.Name -like "*$k*" } | 
    Select-Object FullName, LastWriteTime
}

# 3. 找 2022-2024 年的已结项目（最可能含核心资料）
Get-ChildItem $root -Recurse -Directory -ErrorAction SilentlyContinue | 
  Where-Object { $_.LastWriteTime -gt "2024-01-01" -and $_.LastWriteTime -lt "2025-12-31" } |
  Select-Object FullName, LastWriteTime | Sort-Object LastWriteTime -Descending

# 4. 找大型项目（>100MB 资料）
Get-ChildItem $root -Recurse -Directory -ErrorAction SilentlyContinue | 
  ForEach-Object {
    $size = (Get-ChildItem $_.FullName -Recurse -File -ErrorAction SilentlyContinue | 
             Measure-Object -Property Length -Sum).Sum / 1MB
    if ($size -gt 100) { [PSCustomObject]@{Name=$_.Name; SizeMB=[math]::Round($size,1)} }
  } | Sort-Object SizeMB -Descending
```
