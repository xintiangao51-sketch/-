## ⚡ 一句话结论

本目录存放**造价业务相关的非 MD 原始资料**（PDF/Excel/Word/图纸/扫描件），与 `D:\知识库\`（提炼后的 MD 卡片）形成**"原料-产品"双层结构**。

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| **只放不提炼的资料** | 已经在 D:\知识库 提炼为 MD 卡片的 PDF 不重复放 |
| **不做 AI 操作** | 本目录是**人类查阅档案**，不是 AI 工作对象 |
| **不需脱敏** | 经确认，全部资料**保持原样**存档 |
| **可移动不修改** | 文件名/路径保持原样，便于法规/合同溯源 |

## 四、维护规范

### 4.1 命名规范

```
<项目名>/<文件类型>/<年份>_<文号>_<内容>.扩展名
例：哈密国源综合服务中心/合同/2024_GF-2024-0201_施工总承包合同.pdf
```

### 4.2 元数据（可选）

每个 PDF/Excel/Word **可配**同名 `.meta.md`，含：

```yaml
---
来源: 哈密国源业主-2024-01
项目: 哈密国源综合服务中心
专业: 房建-装饰
日期: 2024-01-15
作者: 昕田
类型: 合同
关联KB: KC-370 / KB-133
---

## 关键信息
- 合同金额: 7481 万元
- 工期: 24 个月
```

| 操作 | 规则 |

| 添加 | 老高/人工均可 |
| 删除 | ⚠ 必须人工确认，老高不可独立删 |
| 移动 | ⚠ 必须人工确认 |
| 修改 | ❌ 不允许修改原始文件，修改请用副本 |

## 六、操作执行（你 Windows 端跑）

```powershell

```

